## 二、Phase 0：基础与环境

先完成概率与数理统计的数学理论学习，再学习 Python 量化栈并在实践中巩固统计应用。

**对应知识库**：`knowledge/phase0_基础与环境/`（概率论、数理统计、python_量化栈）

### 2.1 知识清单

| 类别 | 内容 | 说明 |
|------|------|------|
| **概率论** | 概率空间、条件概率、贝叶斯、常见分布、大数定律与中心极限定理 | 复习为主，可配合教材做题 |
| **数理统计** | 估计（点估计、区间估计）、假设检验（t/F/χ²）、相关系数 | 与后续回归、回测检验直接相关 |
| **Python 量化栈** | NumPy, Pandas, Matplotlib/Seaborn | 数据处理与可视化 |

### 2.2 技能目标

- 能独立推导/默写：正态总体下均值与方差的检验、线性相关系数的检验。
- 能用 Python 完成：数据读取 → 清洗 → 简单统计 → 绘图。
- 本地搭建：Python 3.9+、Jupyter、常用库。

### 2.3 推荐资源

- **教材**：《概率论与数理统计》浙大版，或《Introduction to Probability》Bertsekas。
- **网课**：Coursera「Statistics with Python」或国内慕课「概率论与数理统计」。
- **代码**：Pandas 官方 10 Minutes to Pandas；NumPy 官方 Quickstart。更多见 `knowledge/速查与汇总/推荐书籍与资源.md`。

### 2.4 学习顺序（示例）

| 步骤 | 主题 | 具体任务 |
|------|------|----------|
| 1 | 概率论复习 | 概率与分布，做课后题 |
| 2 | 数理统计（上） | 估计（点估计、区间估计） |
| 3 | 数理统计（下） | 假设检验（t/F/χ²）、相关系数 |
| 4 | Python 环境 + NumPy/Pandas | 安装 Jupyter，过完入门教程 |
| 5 | 统计 + Python | Pandas 描述统计、SciPy 检验 |
| 6 | 综合小项目 + 阶段自测 | 数据加载→清洗→统计→可视化；默写 t 检验、Pearson 检验 |

### 2.5 产出物

- 概率与统计公式/推导速查笔记（可对应 `knowledge/速查与汇总/概率与统计速查.md` 风格）。
- 一个「数据加载–清洗–统计–可视化」的 Jupyter Notebook。

## 三、Phase 1：数学与统计深化

**对应知识库**：`knowledge/phase1_数学与统计深化/`（线性代数应用、时间序列、回归、最优化基础、机器学习入门）

### 3.1 知识清单

| 类别 | 内容 | 说明 |
|------|------|------|
| **线性代数（复习+应用）** | 特征值、特征向量、PCA、矩阵求逆与条件数 | 因子降维、多重共线性 |
| **时间序列** | 平稳性、ACF/PACF、AR/MA/ARMA、单位根检验、协整 | 价格/收益建模、配对交易基础 |
| **回归** | 多元线性回归、OLS、R²/调整 R²、F 检验、多重共线性、异方差 | 因子收益估计、因子中性化 |
| **最优化基础** | 梯度、凸函数、无约束/约束优化、拉格朗日 | 组合优化、风险预算的前置 |
| **机器学习入门** | 线性模型、树模型、过拟合与正则化、交叉验证 | 因子/收益预测、防止过拟合意识 |

### 3.2 技能目标

- 能对时间序列做平稳性检验（ADF）并选择 AR/MA 阶数。
- 能手推 OLS 正规方程，能解释 R²、t/F 检验、异方差的后果及处理思路。
- 能用 sklearn 做回归、随机森林；能写出 train/validation/test 与时间序列 CV。
- 能实现或调用 PCA，理解与因子降维的关系。

### 3.3 推荐资源

- 时间序列：Shumway《Time Series Analysis and Its Applications》前几章。
- 回归：《Introduction to Linear Regression Analysis》前几章；Wooldridge 计量前几章。
- 最优化：Boyd《Convex Optimization》前 3 章（选读）。
- 机器学习：Andrew Ng（Coursera）；《统计学习方法》李航。详见 `knowledge/速查与汇总/推荐书籍与资源.md`。

### 3.4 学习顺序（示例）

| 步骤 | 主题 | 具体任务 |
|------|------|----------|
| 1 | 线性代数应用 | 特征值、PCA；对因子矩阵做 PCA |
| 2 | 时间序列 | 平稳性、AR/MA/ARMA；statsmodels ADF、ARMA 拟合 |
| 3 | 回归 | OLS、假设、检验；statsmodels 多元回归 |
| 4 | 回归 + 金融 | 因子暴露→收益横截面回归 |
| 5 | 最优化基础 | 梯度、凸集/凸函数、拉格朗日；scipy/cvxpy 简单组合权重 |
| 6 | 机器学习入门 | 线性/岭/Lasso、树模型、交叉验证 |
| 7 | 阶段项目 | 收益预测小项目：特征→回归/树→时间序列 CV |

### 3.5 产出物

- 时间序列分析 Notebook（ADF + ARMA + 预测）。
- 横截面回归 Notebook（因子暴露→收益）。
- 「回归 + 时间序列 + 优化」要点笔记（可对应 `knowledge/速查与汇总/回归与时间序列要点.md`）。

## 四、Phase 2：金融与因子

**对应知识库**：`knowledge/phase2_金融与因子/`（核心 5 本 + 扩展 4 本）

### 4.1 知识清单

| 类别 | 内容 | 说明 |
|------|------|------|
| **金融市场基础** | 股票、期货、ETF、做多/做空、涨跌停、T+1、保证金与杠杆 | 避免策略假设脱离规则 |
| **多因子模型** | CAPM、Fama-French 三/五因子、因子构建、因子中性化、IC/IR | 策略研究核心框架 |
| **价量与微观结构** | 量价关系、订单簿、买卖价差、冲击成本、换手率、执行模型 | 价量策略与执行 |
| **回测框架** | 收益率计算、滑点与手续费、前视偏差、幸存者偏差、样本内/外 | 避免无效策略 |
| **风险与组合** | 波动率、VaR/CVaR、夏普/卡玛、最大回撤、均值-方差、风险平价、Black-Litterman | 策略评估与仓位 |
| **扩展（可选）** | 期权与波动率基础、固定收益基础、行为金融要点、资产配置 | 见 knowledge/phase2 下对应文档 |

### 4.2 技能目标

- 能说出 CAPM 与 Fama-French 的假设与因子含义；能计算单因子 IC、IR 和分组回测收益。
- 能实现：因子计算→去极值/标准化→中性化→分组回测。
- 能写出规范回测流程（区间、成本、再平衡），并能指出常见回测陷阱。
- 能计算夏普、最大回撤、卡玛比率，并能用 scipy/cvxpy 做均值-方差或风险平价组合。

### 4.3 推荐资源

- 《因子投资：方法与实践》；《主动投资组合管理》前几章。
- Coursera「Financial Engineering and Risk Management」部分章节；国内「量化投资」慕课。
- 数据与回测：Tushare/AKShare；聚宽/优矿/掘金。详见 `knowledge/速查与汇总/推荐书籍与资源.md`。

### 4.4 学习顺序（示例）

| 步骤 | 主题 | 具体任务 |
|------|------|----------|
| 1 | 金融市场与规则 | A 股/期货规则；交易成本、涨跌停速查表 |
| 2 | 多因子框架 | 《因子投资》前 4 章；3–5 个因子计算与单因子回测 |
| 3 | 因子中性化与组合 | 行业/市值中性化；多因子打分/回归；分层回测 |
| 4 | 价量与微观结构 | 量价因子、订单簿概念、冲击成本近似 |
| 5 | 回测规范 | 选股→调仓→收益→成本完整回测；回测检查清单 |
| 6 | 风险与组合 | 夏普/回撤；均值-方差组合（做空/不做空） |
| 7 | 阶段项目 | 多因子选股项目：因子池→中性化→组合→回测→风险指标 |

### 4.5 产出物

- 多因子研究 Notebook（IC/IR、分组收益、中性化）。
- 可复用的回测模板或框架说明。
- 「多因子 + 回测 + 风险」要点笔记（可对应 `knowledge/速查与汇总/多因子与回测要点.md`）。

## 五、Phase 3：策略与实证

**对应知识库**：`knowledge/phase3_策略与实证/`（策略类型、过拟合控制、实盘与系统、论文与研报）

### 5.1 知识清单

| 类别 | 内容 | 说明 |
|------|------|------|
| **策略类型** | 多因子选股、动量/反转、统计套利（协整配对）、简单择时 | 至少精通一类 |
| **过拟合控制** | 样本外、滚动/扩展窗口、时间序列 CV、正则化、因子与参数数量控制 | 策略稳健性核心 |
| **实盘与系统** | 信号生成、仓位管理、风控（止损/敞口）、实盘与回测差异 | 回测到实盘衔接 |
| **论文与研报** | 经典因子/策略论文与研报的读法与总结 | 表述与逻辑 |

### 5.2 技能目标

- 能独立完成「从想法到回测」的完整 pipeline，并写出清晰文档。
- 能说明如何避免过拟合（样本外、滚动、因子数量、正则等）。
- 能描述策略的逻辑、假设、风险、适用市场与周期。
- 至少参加 1 次量化/时间序列竞赛或 1 段小资金/模拟盘实盘。

### 5.3 推荐资源

- 竞赛：Kaggle 时间序列/金融；国内聚宽、天勤、券商量化赛。
- 论文：Fama-French、Jegadeesh & Titman 动量、协整配对综述。见 `knowledge/phase3_策略与实证/论文与研报.md`。
- 实盘：聚宽/掘金模拟盘，或小资金 + 简单风控。

### 5.4 学习顺序（示例）

| 步骤 | 主题 | 具体任务 |
|------|------|----------|
| 1 | 策略设计 | 选定主攻方向；写策略文档（逻辑、假设、风险、参数） |
| 2 | 实现与回测 | 完整实现；参数敏感性；样本外/滚动回测 |
| 3 | 过拟合与稳健性 | 过拟合控制检查清单；稳健性测试 |
| 4 | 竞赛或实盘 | 报名竞赛或开模拟盘；记录进展 |
| 5 | 论文/研报 | 精读 2–3 篇；写 1–2 页总结 |
| 6 | 阶段项目 | 可展示策略：代码 + README + 回测 + 风险指标；GitHub |

### 5.5 产出物

- 一个 GitHub 策略项目（代码规范、README、可复现）。
- 竞赛排名/实盘或模拟盘记录。
- 「策略逻辑 + 过拟合控制 + 实盘注意点」总结笔记（可对应 `knowledge/速查与汇总/策略与实证要点.md`）。
