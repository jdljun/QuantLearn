## 前言：为什么量化要学数理统计

在量化研究中，我们面对的是**有限的历史数据**（样本），而真正关心的是**未来的收益分布、因子与收益的关系、策略是否显著有效**（总体或未知参数）。我们无法观测「总体」本身，只能通过**样本**去**推断**总体。数理统计研究的正是：如何从样本构造**估计量**、如何评价估计的优劣（无偏性、有效性、一致性）、如何给出**置信区间**以体现不确定性、如何通过**假设检验**判断「均值是否为零」「相关系数是否显著」等。回测中的 t 检验、IC 的显著性、回归系数的 p 值、VaR 的估计，都建立在数理统计的框架上。本章按教材标准展开：**总体与样本 → 统计量与点估计 → 区间估计 → 假设检验的思想与两类错误 → 单总体与两总体检验 → 相关系数及其检验**，并配有**定义、定理的推导与数值例题**。

# 第一章 总体、样本与统计量

**动机**：我们只能观测到有限样本（如 252 个交易日收益），而要推断的是总体（如「该标的日收益的分布」）的均值、方差等。必须先明确总体、样本与简单随机样本的定义，才能讨论统计量与估计。

## 1.1 总体与样本

### 1.1.1 总体

**定义 1.1（总体）**  
在数理统计中，把研究对象的**全体**称为**总体**。总体可以是有限个个体（如某校全体学生），也可以是无限或抽象的（如「某股票未来每日收益的全体」）。  
为便于数学处理，常把总体与**随机变量 X** 及其**分布 F** 或**参数 θ** 等同：总体即「服从某分布的随机变量」，其分布描述了「从总体中随机抽取一个个体时，其取值的规律」。

**例 1.1**  
若关心「某股票日收益率」，则总体可抽象为随机变量 X，其分布 F 描述了日收益的概率规律（如近似正态、厚尾等）；**总体均值 μ=E(X)**、**总体方差 σ²=Var(X)** 是未知常数，是我们要推断的对象。

### 1.1.2 样本与简单随机样本

**定义 1.2（样本）**  
从总体中按一定规则**抽取**的 n 个个体 X₁, X₂, …, Xₙ，称为总体的一个**样本**，n 称为**样本容量**。  
若抽取满足：(1) **随机性**——每个个体被抽到的机会相同或符合某种随机机制；(2) **独立性**——各次抽取互不影响，则称 X₁,…,Xₙ 为**简单随机样本**。在数学上，即 X₁,…,Xₙ **相互独立**，且与总体 **同分布**（即每个 Xᵢ 与 X 同分布）。

**定义 1.3（样本值）**  
一次具体抽样得到的 n 个观测数值，记为小写 x₁, x₂, …, xₙ，称为**样本值**（或样本观测值）。  
注意：**统计量**是**样本**（随机变量）的函数，因此也是随机变量；把样本值代入统计量得到的数值，称为该统计量的**观测值**（或估计值）。

**例 1.2**  
从「日收益率总体」中随机抽取 252 个交易日，得到样本 X₁,…,X₂₅₂；一次抽样得到 252 个具体数字 x₁,…,x₂₅₂，即一年的日收益数据。样本均值 X̄=(1/252)Σ Xᵢ 是统计量；代入观测值得到 x̄=(1/252)Σ xᵢ，即该年的**历史平均收益**（估计值）。

## 1.2 统计量

**动机**：总体参数（如 μ、σ²）未知，我们只能用样本构造「不含未知参数」的函数——统计量——来估计或检验；统计量的分布（抽样分布）是区间估计与假设检验的基础。

**定义 1.4（统计量）**  
设 X₁,…,Xₙ 为样本。若函数 T(X₁,…,Xₙ) **不依赖任何未知参数**，则称 T 为**统计量**。  
统计量是随机变量的函数，因此也是随机变量，有其分布（称为**抽样分布**）。

### 1.2.1 常用统计量

**样本均值**  
**X̄ = (1/n) Σᵢ₌₁ⁿ Xᵢ**  
用于估计总体期望 μ=E(X)。  
性质：若 X₁,…,Xₙ 为简单随机样本，则 E(X̄)=μ，Var(X̄)=σ²/n（其中 σ²=Var(X₁)）。

**样本方差与样本标准差**  
**S² = [1/(n−1)] Σᵢ₌₁ⁿ (Xᵢ − X̄)²**  
**S = √S²**  
注意分母是 **n−1** 而非 n。理由见下。

**命题 1.1（S² 的无偏性及分母为 n−1 的由来）**  
设 X₁,…,Xₙ 为简单随机样本，总体均值 μ、方差 σ² 存在。则 **E(S²)=σ²**，即 S² 是 σ² 的**无偏估计**；若定义 σ̂²_n = (1/n)Σᵢ(Xᵢ−X̄)²（分母为 n），则 E(σ̂²_n)=(n−1)σ²/n < σ²，即**有偏**，因此**分母取 n−1 才能保证无偏**。

**证明**：记 μ=E(X₁)。先证恒等式  
**Σᵢ(Xᵢ−X̄)² = Σᵢ(Xᵢ−μ)² − n(X̄−μ)²**。  
展开左边：Σᵢ(Xᵢ−X̄)² = Σᵢ[(Xᵢ−μ)−(X̄−μ)]² = Σᵢ(Xᵢ−μ)² − 2(X̄−μ)Σᵢ(Xᵢ−μ) + n(X̄−μ)²。  
其中 Σᵢ(Xᵢ−μ)=n(X̄−μ)，故中间项为 −2n(X̄−μ)²，即得 Σᵢ(Xᵢ−X̄)² = Σᵢ(Xᵢ−μ)² − n(X̄−μ)²。  
对两边取期望：E[Σᵢ(Xᵢ−μ)²]=Σᵢ E(Xᵢ−μ)²=nσ²；E[n(X̄−μ)²]=n·Var(X̄)=n·(σ²/n)=σ²。  
故 E[Σᵢ(Xᵢ−X̄)²]=nσ²−σ²=**(n−1)σ²**。因此 E(S²)=E[(1/(n−1))Σᵢ(Xᵢ−X̄)²]=σ²。  
若用分母 n，则 E[(1/n)Σᵢ(Xᵢ−X̄)²]=(n−1)σ²/n，当 n 有限时小于 σ²（**低估方差**），且仅当 n→∞ 时趋于 σ²。□  

**注意**：自由度 n−1 可理解为：n 个偏差 (Xᵢ−X̄) 受约束 Σᵢ(Xᵢ−X̄)=0，故「自由」的只有 n−1 个；用 n−1 作分母既使估计无偏，也与 t 分布、χ² 分布的自由度一致。**易错点**：若误用分母 n，则 E[(1/n)Σᵢ(Xᵢ−X̄)²]=(n−1)σ²/n，会**低估**总体方差。  

**样本 k 阶原点矩**  
**(1/n) Σᵢ Xᵢᵏ**  
**样本 k 阶中心矩**  
**(1/n) Σᵢ (Xᵢ−X̄)ᵏ**  
一阶中心矩恒为 0；二阶中心矩（分母用 n）与 S² 差一因子 (n−1)/n。

**顺序统计量**  
将样本按从小到大排列：X₍₁₎ ≤ X₍₂₎ ≤ … ≤ X₍ₙ₎，称 X₍ₖ₎ 为**第 k 个顺序统计量**。  
**样本中位数**：n 为奇数时取 X₍_{(n+1)/2}₎，n 为偶数时取 (X₍_{n/2}₎+X₍_{n/2+1}₎)/2。  
**样本 p 分位数**：可用顺序统计量的线性插值定义（略）。

在量化中：**日收益的样本均值**即历史平均收益，**样本标准差 S** 即历史波动率；**因子暴露与下期收益的相关系数**（IC）也是基于样本的统计量。

## 本章小结（第一章）

**核心概念**：总体（常等同为随机变量 X 及其分布）；样本与样本容量；简单随机样本（独立同分布）；样本值；统计量（不含未知参数）与抽样分布；样本均值、样本方差 S²、顺序统计量。

**主要公式与结论**：  
1. X̄ = (1/n)Σᵢ Xᵢ；E(X̄)=μ，Var(X̄)=σ²/n。  
2. S² = [1/(n−1)]Σᵢ(Xᵢ−X̄)²；**E(S²)=σ²**（无偏），分母 n−1 的推导见命题 1.1。  
3. 恒等式：Σᵢ(Xᵢ−X̄)² = Σᵢ(Xᵢ−μ)² − n(X̄−μ)²。

# 第二章 点估计

**动机**：总体参数 θ（如 μ、σ²）未知，我们用样本构造估计量 θ̂；需要明确估计量与估计值的区别，以及如何评价估计量（无偏性、有效性、一致性）。

## 2.1 估计量与估计值

**定义 2.1（估计量与估计值）**  
设总体分布含有未知参数 θ。用样本 X₁,…,Xₙ 构造的、用于估计 θ 的统计量 θ̂=θ̂(X₁,…,Xₙ)，称为 θ 的**估计量**。把具体样本值 x₁,…,xₙ 代入 θ̂ 得到的数值，称为 θ 的**估计值**（仍常记作 θ̂）。  
估计量是随机变量，有分布；估计值是数值。

---

## 2.2 估计量的评价标准

**定义 2.2（无偏性）**  
若 **E(θ̂)=θ**，则称 θ̂ 为 θ 的**无偏估计**。  
无偏的含义：在重复抽样下，估计量的「平均」等于真值；单次估计可能偏高或偏低，但系统上不偏。

**定义 2.3（有效性）**  
设 θ̂₁、θ̂₂ 均为 θ 的无偏估计。若对任意 θ 都有 **Var(θ̂₁) ≤ Var(θ̂₂)**，且至少对某 θ 严格不等，则称 θ̂₁ 比 θ̂₂ **更有效**。  
在正态总体下，可以证明：**X̄ 是 μ 的最小方差无偏估计**（在无偏估计类中方差最小）。

**定义 2.4（一致性）**  
若当 n→∞ 时，θ̂ **依概率收敛**于 θ，即对任意 ε>0，lim P(|θ̂−θ|>ε)=0，则称 θ̂ 为 θ 的**一致估计**。  
大样本下，一致估计会越来越接近真值。**X̄ 是 μ 的一致估计**（由大数定律）；**S² 是 σ² 的一致估计**。

---

## 2.3 矩估计与极大似然估计（概念）

**矩估计**：用**样本矩**等于**总体矩**建立方程，解出参数。  
例如：总体均值 μ、方差 σ² 未知，令  
- 一阶样本原点矩 = 总体一阶矩：X̄ = μ；  
- 二阶样本中心矩 = 总体二阶中心矩：(1/n)Σ(Xᵢ−X̄)² = σ²（此处用 n 作分母得到的矩估计，对 σ² 有偏但一致）。  
解出 μ̂=X̄，σ̂²=(1/n)Σ(Xᵢ−X̄)²。  
若用无偏的 S² 估计 σ²，则与「二阶中心矩」的矩估计差一因子 (n−1)/n。

**极大似然估计（MLE）**：在已观测到样本值 x₁,…,xₙ 的条件下，选择使**似然函数 L(θ)**（离散型为联合概率，连续型为联合密度）达到最大的 θ 作为估计。  
许多常用估计（如正态总体的 μ、σ² 的 MLE）与矩估计或与无偏估计有简单关系；正态时 μ 的 MLE 为 X̄，σ² 的 MLE 为 (1/n)Σ(Xᵢ−X̄)²（有偏）。

量化中更常用的是**直接使用样本均值与 S²** 作为 μ 与 σ² 的估计，其理论依据即矩估计与无偏性、一致性。

## 本章小结（第二章）

**核心概念**：估计量与估计值（估计量是随机变量，估计值是数值）；无偏性、有效性、一致性；矩估计与极大似然估计（MLE）。

**主要公式与结论**：  
1. 无偏性：E(θ̂)=θ；有效性：在无偏估计中 Var(θ̂₁)≤Var(θ̂₂)；一致性：n→∞ 时 θ̂→^P θ。  
2. 矩估计：令样本矩=总体矩，解出参数；正态总体 μ 的 MLE 为 X̄，σ² 的 MLE 为 (1/n)Σ(Xᵢ−X̄)²（有偏）。  
3. 常用：用 X̄ 估计 μ，用 S² 估计 σ²（无偏且一致）。

# 第三章 区间估计（置信区间）

**动机**：点估计只给出一个数值，无法体现「估计有多不确定」；置信区间给出一个区间，使得在重复抽样下约有 (1−α) 比例的区间包含真实参数，从而量化不确定性。

## 3.1 置信区间的概念

**定义 3.1（置信区间）**  
设 θ 为总体参数，X₁,…,Xₙ 为样本。若由样本构造的两个统计量 θ̂_L、θ̂_U（θ̂_L < θ̂_U），使得对给定的 1−α（0<α<1），有  
**P(θ̂_L < θ < θ̂_U) = 1−α**，  
则称随机区间 (θ̂_L, θ̂_U) 为 θ 的**置信水平为 1−α 的置信区间**，1−α 称为**置信水平**（或置信度），α 常取 0.05 或 0.01。

**解释**：  
- 若重复抽样多次，每次得到一对 (θ̂_L, θ̂_U)；则**大约 (1−α) 比例的区间会包含真实 θ**。  
- 对**单次**抽样得到的区间，我们**不说**「θ 有 1−α 的概率落在此区间内」（因为 θ 是常数，要么在要么不在），而说「我们有 1−α 的**信心**认为 θ 落在此区间内」，即该区间是「众多可能区间中」有 1−α 比例会盖住 θ 的那一类。

---

## 3.2 正态总体、方差已知时 μ 的置信区间

**设定**：X₁,…,Xₙ 为来自 **N(μ, σ²)** 的简单随机样本，**σ² 已知**。

由正态分布的可加性，**X̄ = (1/n)Σ Xᵢ ∼ N(μ, σ²/n)**。故  
**(X̄−μ)/(σ/√n) ∼ N(0, 1)**。  
取标准正态的 (1−α/2) 分位数 z_{α/2}（即 P(Z ≤ z_{α/2})=1−α/2，如 α=0.05 时 z_{0.025}≈1.96），则  
**P(−z_{α/2} ≤ (X̄−μ)/(σ/√n) ≤ z_{α/2}) = 1−α**。  
等价于 **P(X̄ − z_{α/2}·σ/√n ≤ μ ≤ X̄ + z_{α/2}·σ/√n) = 1−α**。

**结论**：**μ 的置信水平 1−α 的置信区间为**  
**[X̄ − z_{α/2}·σ/√n,  X̄ + z_{α/2}·σ/√n]**。  
区间长度与 1/√n 成正比：样本量越大，区间越短。

**例 3.1**  
设日收益率服从 N(μ, 0.0001)，即 σ²=0.0001（σ=0.01，即 1%）已知。抽取 n=252 个交易日，得样本均值 x̄=0.0005（即 0.05%）。求 μ 的置信水平 95%（α=0.05）的置信区间。

**解**：1−α/2=0.975，z_{0.025}≈1.96。区间为  
\[
\bar{x} \pm z_{\alpha/2}\,\frac{\sigma}{\sqrt{n}} = 0.0005 \pm 1.96\times\frac{\sqrt{0.0001}}{\sqrt{252}} = 0.0005 \pm 1.96\times\frac{0.01}{15.87} \approx 0.0005 \pm 0.00123.
\]  
代入数值：下限 0.0005−0.00123=−0.00073，上限 0.0005+0.00123=0.00173。故 μ 的 95% 置信区间约为 **[−0.00073, 0.00173]**（即约 [−0.073%, 0.173%]）。  
**解释**：我们有 95% 的信心认为真实日均收益 μ 落在该区间内；区间包含 0，与后续「检验 μ=0 时可能不拒绝」一致。

**代码（Python）**：

```python
import numpy as np
from scipy import stats
xbar, sigma_sq, n = 0.0005, 0.0001, 252
z = stats.norm.ppf(0.975)  # 1.96
half = z * np.sqrt(sigma_sq / n)
ci = (xbar - half, xbar + half)
print(f"μ 的 95% 置信区间: [{ci[0]:.6f}, {ci[1]:.6f}]")
# 输出约 [-0.00073, 0.00173]
```

## 3.3 正态总体、方差未知时 μ 的置信区间（t 区间）

**设定**：X₁,…,Xₙ 来自 N(μ, σ²)，**σ² 未知**。用**样本方差 S²** 替代 σ²。

**定理 3.1**  
在以上设定下，**t = (X̄−μ)/(S/√n) ∼ t(n−1)**（自由度为 n−1 的 t 分布）。  
（证明需用到：(1) X̄ 与 S² 独立（正态总体时成立）；(2) (n−1)S²/σ² ∼ χ²(n−1)；则 (X̄−μ)/(σ/√n) 与 √[(n−1)S²/σ²] 独立，其比即 t(n−1)。）

**t 分布**：密度关于 0 对称，尾比标准正态**更厚**；自由度越大，越接近 N(0,1)。  
取 t(n−1) 的 (1−α/2) 分位数 t_{α/2}(n−1)，则  
**P(X̄ − t_{α/2}(n−1)·S/√n ≤ μ ≤ X̄ + t_{α/2}(n−1)·S/√n) = 1−α**。

**结论**：**μ 的置信水平 1−α 的置信区间为**  
**[X̄ − t_{α/2}(n−1)·S/√n,  X̄ + t_{α/2}(n−1)·S/√n]**。  
实践中 σ² 通常未知，故**此公式最常用**。n 较大（如 n>30）时，t_{α/2}(n−1) 与 z_{α/2} 接近，可用 z_{α/2} 近似。

**例 3.2**  
续例 3.1，若 σ² 未知，由 252 日数据算得 x̄=0.0005，s=0.01。则 t_{0.025}(251)≈1.969，μ 的 95% 置信区间为  
0.0005 ± 1.969×0.01/√252 ≈ 0.0005 ± 0.00124，  
即约 [−0.00074, 0.00174]，与例 3.1 接近（因 n 大，t≈z）。

**代码（Python）**：

```python
from scipy import stats
import numpy as np
xbar, s, n = 0.0005, 0.01, 252
t_half = stats.t.ppf(0.975, n - 1)
margin = t_half * s / np.sqrt(n)
ci_t = (xbar - margin, xbar + margin)
print(f"μ 的 95% t 置信区间: [{ci_t[0]:.6f}, {ci_t[1]:.6f}]")
```

## 3.4 方差 σ² 的置信区间（χ² 分布）

**设定**：X₁,…,Xₙ 来自 N(μ, σ²)。有 **(n−1)S²/σ² ∼ χ²(n−1)**。  
取 χ²(n−1) 的 α/2 与 1−α/2 分位数 χ²_{1−α/2}(n−1)、χ²_{α/2}(n−1)，则  
**P(χ²_{1−α/2}(n−1) ≤ (n−1)S²/σ² ≤ χ²_{α/2}(n−1)) = 1−α**，  
解出 σ² 得**置信区间**：**[ (n−1)S²/χ²_{α/2}(n−1),  (n−1)S²/χ²_{1−α/2}(n−1) ]**。  
对 σ 的区间可对两端开方。量化中较少直接求方差区间，但 χ² 会出现在**方差检验**中。

## 本章小结（第三章）

**核心概念**：置信区间、置信水平 1−α；正确理解「约 (1−α) 比例区间包含真值」与「单次区间不说 θ 落在区间内的概率」。

**主要公式与结论**：  
1. 定义：P(θ̂_L<θ<θ̂_U)=1−α。  
2. 正态总体、σ² 已知：μ 的 1−α 置信区间为 [X̄−z_{α/2}·σ/√n, X̄+z_{α/2}·σ/√n]。  
3. 正态总体、σ² 未知：μ 的 1−α 置信区间为 [X̄−t_{α/2}(n−1)·S/√n, X̄+t_{α/2}(n−1)·S/√n]；σ² 的区间用 (n−1)S² 与 χ² 分位数。

---

# 第四章 假设检验的基本思想

**动机**：我们常需判断「策略收益均值是否不为 0」「因子 IC 是否显著」等；假设检验在「原假设 H₀」与「备择假设 H₁」之间做决策，并控制犯第一类错误的概率（α），用 p 值报告证据强度。

## 4.1 原假设与备择假设

**假设检验**的任务是：根据样本，在「**原假设 H₀**」与「**备择假设 H₁**」之间做出决策（拒绝 H₀ 或不拒绝 H₀）。

- **原假设 H₀**：通常取「保守」「维持现状」「无效应」的那一侧，如 μ=0、ρ=0。  
- **备择假设 H₁**：希望证实的或要检测出的，如 μ≠0、μ>0、ρ≠0。  
- **检验**：构造**检验统计量** T，根据 T 的**抽样分布**（在 H₀ 成立时）确定**拒绝域**（使犯第一类错误的概率等于 α 的区域）；若 T 的观测值落在拒绝域，则**拒绝 H₀**，否则**不拒绝 H₀**。

**注意（不拒绝 H₀ 与接受 H₀ 的辨析）**  
**不拒绝 H₀ ≠ 接受 H₀**。  
- **不拒绝 H₀**：表示在给定显著性水平 α 下，当前样本提供的证据**不足以**拒绝 H₀，即「没有显著证据反对 H₀」。  
- **接受 H₀**：逻辑上表示「认为 H₀ 为真」，这需要证明，而假设检验**只能控制犯第一类错误的概率**，无法证明 H₀ 成立。  
因此规范表述是「不拒绝 H₀」或「在 α 水平下不能拒绝 H₀」；说「接受 H₀」容易误导为「已证明 H₀ 为真」。在量化中：若策略收益的 t 检验不拒绝 μ=0，应理解为「当前样本下无法认定策略显著有效」，而非「策略无效已证实」。

---

## 4.2 显著性水平与两类错误

**定义 4.1（第一类错误与显著性水平）**  
当 **H₀ 为真**时，若检验**拒绝 H₀**，则犯了**第一类错误**（弃真）。  
我们控制「犯第一类错误的最大概率」为 **α**（如 0.05），即 **P(拒绝 H₀ | H₀ 为真) ≤ α**。α 称为**显著性水平**；拒绝域就是按「P(拒绝 H₀|H₀)=α」选出的区域。

**定义 4.2（第二类错误与功效）**  
当 **H₀ 为假**（H₁ 为真）时，若检验**未拒绝 H₀**，则犯了**第二类错误**（取伪）。记 **β = P(未拒绝 H₀ | H₁ 为真)**。**功效**定义为 **1−β**，即正确拒绝 H₀ 的概率。  
实践中：先**固定 α**（如 0.05），再在选择检验与拒绝域时尽量**提高功效**（降低 β）。

**例 4.1**  
H₀：策略平均收益 μ=0；H₁：μ≠0。  
- 第一类错误：策略其实无效（μ=0），但检验误判为「显著不为 0」，从而采用该策略（假阳性）。  
- 第二类错误：策略其实有效（μ≠0），但检验未拒绝 μ=0，从而放弃该策略（假阴性）。  
我们通常更关心「不要轻易把无效策略当成有效」，故控制 α；在此基础上希望「有效策略能被检出」，即希望功效 1−β 大。

---

## 4.3 p 值

**定义 4.3（p 值）**  
在 H₀ 成立的假设下，**p 值**定义为：得到「与当前样本一样极端或更极端」的检验统计量取值的**概率**。  
- 若 **p < α**，则拒绝 H₀（因为当前样本在 H₀ 下属于「小概率」事件）。  
- 若 **p ≥ α**，则不拒绝 H₀。  
**p 值越小**，表示数据越不利于 H₀。  
报告 p 值比只报告「是否拒绝」更丰富：读者可按自己的 α 做判断。  
量化中：策略收益的 t 检验、因子 IC 的显著性，常用 **p 值**报告。

## 本章小结（第四章）

**核心概念**：原假设 H₀ 与备择假设 H₁；检验统计量、拒绝域；第一类错误（弃真）与显著性水平 α；第二类错误（取伪）与功效 1−β；p 值；不拒绝 H₀ ≠ 接受 H₀。

**主要公式与结论**：  
1. P(拒绝 H₀ | H₀ 为真) ≤ α；P(未拒绝 H₀ | H₁ 为真)=β，功效=1−β。  
2. p 值：H₀ 下得到与当前一样极端或更极端的概率；p<α 则拒绝 H₀。  
3. 量化应用：t 检验用于策略收益、因子收益显著性；p 值<0.05 时拒绝「收益为 0」或「IC 为 0」。

---

# 第五章 单正态总体的检验

以下设 **X₁,…,Xₙ 为来自 N(μ, σ²) 的简单随机样本**。

## 5.1 均值 μ 的检验（方差 σ² 已知）

- **H₀: μ=μ₀**；**H₁: μ≠μ₀**（双侧）。  
在 H₀ 下，**(X̄−μ₀)/(σ/√n) ∼ N(0,1)**。取显著性水平 α，拒绝域为 **|Z| > z_{α/2}**，其中 **Z = (X̄−μ₀)/(σ/√n)**。  
- **单侧**：H₁: μ>μ₀，拒绝域 Z > z_α；H₁: μ<μ₀，拒绝域 Z < −z_α。

**例 5.1**  
检验「日收益均值是否为 0」，σ²=0.0001 已知，n=252，得 x̄=0.0005。  
Z = 0.0005/(0.01/√252) ≈ 0.795。|Z|<1.96，故不拒绝 H₀: μ=0（α=0.05）。

---

## 5.2 均值 μ 的检验（方差 σ² 未知）：t 检验

**最常用**：σ² 未知时用 **S** 替代 σ。

**检验统计量**：**t = (X̄−μ₀)/(S/√n)**。在 H₀: μ=μ₀ 下，**t ∼ t(n−1)**。  
**拒绝域（双侧，显著性水平 α）**：**|t| > t_{α/2}(n−1)**。  
**单侧**：H₁: μ>μ₀，拒绝域 t > t_α(n−1)；H₁: μ<μ₀，拒绝域 t < −t_α(n−1)。

**例 5.2（t 检验：日收益均值是否为 0）**  
题干：252 个交易日收益，样本均值 x̄=0.0005（0.05%），样本标准差 s=0.01（1%）。在 α=0.05 下检验 H₀: μ=0，H₁: μ≠0。

**解**：  
- 检验统计量 t = (x̄−0)/(s/√n) = 0.0005/(0.01/√252) = 0.0005/0.000629 ≈ **0.795**。  
- 拒绝域：|t| > t_{0.025}(251) ≈ 1.969。  
- 因 |0.795| < 1.969，故**不拒绝** H₀: μ=0。  
- 结论：在 5% 显著性水平下，无法认定日均收益显著不为 0。

**代码（Python）与结果说明**：

```python
from scipy import stats
import numpy as np
# 252 个日收益（示例用模拟数据）
rets = np.random.normal(0.0005, 0.01, 252)
t_stat, p_value = stats.ttest_1samp(rets, 0)
print(f"t = {t_stat:.4f}, p = {p_value:.4f}")
# 若 p_value < 0.05 则拒绝 H₀；否则不拒绝。本例约 p≈0.43，不拒绝。
```

**注意**：不拒绝 μ=0 并不表示「已证明 μ=0」，只表示当前样本下证据不足；若需「策略有效」的结论，应依赖拒绝 H₀（p<α）。  
量化应用：策略回测后常用 t 检验判断超额收益是否显著；若 p-value<0.05，拒绝零假设，认为策略收益显著。

---

## 5.3 方差 σ² 的检验：χ² 检验

- **H₀: σ²=σ₀²**。  
在 H₀ 下，**χ² = (n−1)S²/σ₀² ∼ χ²(n−1)**。  
**双侧拒绝域**（显著性 α）：**χ² < χ²_{1−α/2}(n−1)** 或 **χ² > χ²_{α/2}(n−1)**。  
用于检验「波动率是否等于某给定值」或「是否发生显著变化」。

## 本章小结（第五章）

**核心概念**：单正态总体；Z 检验（σ² 已知）与 t 检验（σ² 未知）；χ² 检验（方差）。

**主要公式与结论**：  
1. σ² 已知：Z=(X̄−μ₀)/(σ/√n)∼N(0,1)，拒绝域 |Z|>z_{α/2}。  
2. σ² 未知：t=(X̄−μ₀)/(S/√n)∼t(n−1)，拒绝域 |t|>t_{α/2}(n−1)。  
3. σ² 检验：χ²=(n−1)S²/σ₀²∼χ²(n−1)，双侧拒绝域。  
4. 量化：策略/因子收益显著性用 t 检验，报告 p 值。

---

# 第六章 两正态总体的检验（概要）

- **两总体均值差**（方差已知）：(X̄−Ȳ) 服从正态，标准化后为 N(0,1)，可建拒绝域。  
- **两总体均值差**（方差未知、假定相等）：用**两样本 t 检验**，合并方差 S_p²，统计量 **t = (X̄−Ȳ)/(S_p√(1/n₁+1/n₂)) ∼ t(n₁+n₂−2)**。  
- **两总体方差比**：**F = S₁²/S₂²**，在 H₀: σ₁²=σ₂² 下 **F ∼ F(n₁−1, n₂−1)**。用于检验「两段样本波动率是否相同」（如不同 regime）。

## 本章小结（第六章）

**核心概念**：两正态总体；两样本 t 检验（合并方差 S_p）；F 检验（方差比）。

**主要公式与结论**：  
1. 均值差（方差未知、相等）：t=(X̄−Ȳ)/(S_p√(1/n₁+1/n₂))∼t(n₁+n₂−2)。  
2. 方差比：F=S₁²/S₂²，H₀: σ₁²=σ₂² 下 F∼F(n₁−1, n₂−1)。  
3. 量化：比较两段时期波动、两组策略收益均值。

# 第七章 相关系数及其检验

## 7.1 Pearson 相关系数

**总体相关系数**：**ρ = Cov(X,Y)/(σ_X σ_Y)**，−1≤ρ≤1。  
**样本 Pearson 相关系数**：  
**r = Σᵢ(Xᵢ−X̄)(Yᵢ−Ȳ) / √[Σᵢ(Xᵢ−X̄)² · Σᵢ(Yᵢ−Ȳ)²]**。  
性质：−1≤r≤1；r=±1 当且仅当样本点落在一条直线上；**r 只度量线性相关**。  
在量化中：**因子暴露与下期收益的 r** 即 **IC**；多因子之间的 r 用于考察**多重共线性**。

---

## 7.2 ρ=0 的 t 检验

**H₀: ρ=0**（无线性相关）。在 (X,Y) 为**二维正态**且 **ρ=0** 的条件下，可以证明统计量  
**t = r√(n−2) / √(1−r²)**  
服从 **t(n−2)**。  
**拒绝域（双侧，显著性 α）**：**|t| > t_{α/2}(n−2)**。  
用于判断「因子与收益的 IC 是否显著不为 0」：若 p 值<α，则拒绝 ρ=0，认为存在显著线性相关。

**例 7.1（相关系数显著性：因子 IC）**  
题干：n=60 期，因子与下期收益的样本 Pearson 相关系数 r=0.15。在 α=0.05 下检验 H₀: ρ=0，H₁: ρ≠0。

**解**：  
- 检验统计量 t = r√(n−2)/√(1−r²) = 0.15×√58/√(1−0.15²) = 0.15×7.62/√0.9775 ≈ **1.16**。  
- 拒绝域：|t| > t_{0.025}(58) ≈ 2.00。  
- |1.16| < 2.00，故**不拒绝** H₀: ρ=0，即当前样本下不能认为 IC 显著不为 0。  
- 若 r=0.35，则 t = 0.35×√58/√(1−0.1225) ≈ 2.87 > 2.00，**拒绝** ρ=0，认为 IC 显著。

**代码（Python）**：

```python
from scipy import stats
# 假设 x 为因子暴露序列，y 为下期收益序列，各 60 个
r, p = stats.pearsonr(x, y)
print(f"r = {r:.4f}, p = {p:.4f}")
# p < 0.05 则拒绝 ρ=0，认为 IC 显著
```

---

## 7.3 其他相关系数（简述）

- **Spearman 秩相关**：对 X、Y 分别取**秩**后，对秩求 Pearson 相关；只依赖**单调关系**，对异常值更稳健。  
- **Kendall τ**：基于「一致对」与「不一致对」；也用于排序与稳健相关。  
量化中：因子 IC 常用 **Pearson**；分布偏态或异常值多时可用 **Spearman**。

## 本章小结（第七章）

**核心概念**：总体相关系数 ρ 与样本 Pearson 相关系数 r；ρ=0 的 t 检验；Spearman、Kendall 秩相关。

**主要公式与结论**：  
1. r = Σᵢ(Xᵢ−X̄)(Yᵢ−Ȳ) / √[Σᵢ(Xᵢ−X̄)²·Σᵢ(Yᵢ−Ȳ)²]，−1≤r≤1。  
2. H₀: ρ=0 下，t = r√(n−2)/√(1−r²) ∼ t(n−2)；拒绝域 |t|>t_{α/2}(n−2)。  
3. 量化：因子 IC 即 r；p<α 时认为 IC 显著。

# 第八章 与量化研究的衔接

- **点估计**：历史均值、S² 作为期望、方差的估计；因子收益、IC 的估计。  
- **置信区间**：平均收益、夏普比率等的区间，体现估计不确定性。  
- **t 检验**：策略收益、因子收益、IC 是否显著不为 0；回归系数的 t 检验。  
- **F 检验**：多元回归「整体是否显著」、方差齐性。  
- **χ² 检验**：方差是否等于某值、列联表、分布拟合。  
- **相关系数及检验**：因子有效性（IC）、因子间相关性、组合与基准相关性。

**学习建议**：能默写「正态总体、方差未知时 μ 的 t 统计量」和「Pearson 的 r 及 ρ=0 的 t 检验公式」；能说明**为什么 S² 分母是 n−1**、**p 值与显著性水平的含义**、**第一类错误与第二类错误的区别**；能对给定样本值手算或用电算完成一次 t 检验与相关系数检验。
