## 前言：为什么量化要学概率论

在量化研究中，我们关心的是**未来的收益与风险**，而未来本质上是不确定的：明天的收益率、下个月的波动率、某因子在未来是否仍然有效，都无法在当下被唯一确定。概率论提供了一套严格的数学语言，把这种**不确定性**表述为「随机现象」，用**概率**度量各种结果出现的可能性，用**随机变量与分布**描述收益、风险等数量，用**大数定律与中心极限定理**解释「为什么用历史数据可以估计期望与方差」「为什么很多统计量近似正态」。后续的数理统计（估计、检验）、时间序列、回归与机器学习，都建立在概率论的基础上。因此，本章不是罗列公式，而是从**随机现象**出发，逐步建立**概率空间、条件概率、随机变量与分布、极限定理**的完整逻辑链，并配有**定义、性质推导与例题**，达到教材级的可自学程度。

# 第一章 随机现象与概率

## 1.1 随机现象与随机试验

**动机**：量化中我们关心的是未来收益、波动率、因子是否有效等，这些在当下都无法被唯一确定。要严格研究这类「不确定」现象，必须先明确什么叫随机现象、什么叫一次试验，才能进一步定义样本空间、事件与概率。

### 1.1.1 确定性现象与随机现象

自然界与人类活动中，有一类现象在**条件完全确定**时，**结果唯一确定**。例如：在真空中从同一高度释放的物体，落地时间由物理定律唯一决定；给定边长，正方形的面积唯一确定。这类现象称为**确定性现象**。

与之相对，还有一类现象：**在相同条件下重复进行，每次的结果可能不同，并且在单次试验之前无法预言会出现哪一个结果**。例如：掷一枚骰子，每次掷出的点数可能是 1 到 6 中的任意一个，且掷之前无法确定；某只股票明天的收益率可能为正也可能为负，事先无法精确预知。这类现象称为**随机现象**。

**注意**：「无法预知」不意味着没有规律。在**大量重复**之下，随机现象往往会呈现出稳定的统计规律（例如骰子每面出现的比例趋近 1/6），这正是概率论与数理统计研究的对象。

### 1.1.2 随机试验

对随机现象的一次**观察、测量或操作**，若满足以下三条，就称为一次**随机试验**（简称试验）：

1. **可重复性**：在相同条件下可以重复进行；
2. **结果的多样性**：每次试验恰好出现多种可能结果中的一个；
3. **结果的不可预言性**：试验之前不能确定本次会出现哪一个结果。

**例 1.1**  
- 掷一枚均匀骰子一次：可重复、结果有 6 种、每次结果不可预言 → 是随机试验。  
- 观察某股票明日收盘价相对今日的涨跌幅：可重复（每天都可观察）、结果有无穷多种（连续）、不可预言 → 是随机试验。  
- 「掷骰子直到出现 6 为止」所需次数：可重复、结果为 1,2,3,…、不可预言 → 是随机试验。

## 1.2 样本空间与事件

**动机**：讨论概率之前，必须明确「所有可能结果」的集合（样本空间）以及「我们关心的事件」如何用集合表示；这样概率才能定义为「事件（集合）到 [0,1] 的函数」。

### 1.2.1 样本空间

**定义 1.1（样本空间）**  
随机试验的**所有可能结果**组成的集合，称为该试验的**样本空间**，记作 **Ω**。样本空间中的每个元素 ω 称为一个**样本点**（或基本结果）。

- 掷一枚骰子：Ω = {1, 2, 3, 4, 5, 6}，共 6 个样本点。  
- 掷两枚骰子（区分顺序）：Ω = {(i,j) : i,j = 1,…,6}，共 36 个样本点。  
- 某日某股票收益率（抽象为连续值）：Ω = R（实数集），样本点不可列。  
- 某日某股票「涨/跌/平」：Ω = {涨, 跌, 平}。

样本空间可以是**有限集、可列集或不可列集**（如区间、R），取决于具体问题的建模方式。

### 1.2.2 事件

**定义 1.2（事件）**  
样本空间 Ω 的**子集**称为**事件**。若某次试验的结果 ω ∈ A，则称**事件 A 发生**；否则称**事件 A 不发生**。

- **基本事件**：只含一个样本点的事件，如 {1}、{2}。  
- **复合事件**：含多个样本点的事件，如 A = {2,4,6} 表示「掷出偶数点」。  
- **必然事件**：Ω 本身，每次试验都发生。  
- **不可能事件**：空集 ∅，每次试验都不发生。

**例 1.2**  
掷一枚骰子，Ω = {1,2,3,4,5,6}。令  
- A = {2,4,6}：掷出偶数；  
- B = {1,2,3}：掷出不超过 3；  
- C = {6}：掷出 6。  
则 A∩B = {2}（偶数且不超过 3）；A∪B = {1,2,3,4,6}；C ⊂ A。

### 1.2.3 事件的关系与运算

事件是集合，因此可以按集合论进行运算；在概率论中常用如下术语与记号：

| 记号与名称 | 集合含义 | 概率含义 |
|------------|----------|----------|
| A ⊂ B | A 是 B 的子集 | A 发生则 B 必发生 |
| A = B | A 与 B 相等 | A、B 同时发生或同时不发生 |
| A∪B | 并集 | A 发生**或** B 发生（至少一个发生） |
| A∩B（或 AB） | 交集 | A 发生**且** B 发生 |
| A \ B | 差集 | A 发生但 B 不发生 |
| A^c（或 Ā） | 补集（相对 Ω） | A 不发生 |

**运算律**（与集合论一致）：  
- 交换律：A∪B = B∪A，A∩B = B∩A；  
- 结合律：(A∪B)∪C = A∪(B∪C)，(A∩B)∩C = A∩(B∩C)；  
- 分配律：A∩(B∪C) = (A∩B)∪(A∩C)，A∪(B∩C) = (A∪B)∩(A∪C)；  
- 对偶律（De Morgan）：(A∪B)^c = A^c∩B^c，(A∩B)^c = A^c∪B^c。

**定义 1.3（互斥与对立）**  
- 若 A∩B = ∅，称事件 A 与 B **互斥**（或互不相容），即二者不能同时发生。  
- 若 A∪B = Ω 且 A∩B = ∅，称 A 与 B **互为对立事件**，此时 B = A^c（A 不发生就是 B 发生）。

**注意（互斥与对立的辨析）**  
**对立必互斥**（因为 A∩A^c=∅），但**互斥不一定对立**。对立要求「A 与 B 至少发生一个」即 A∪B=Ω；互斥只要求「不同时发生」。例如掷骰子：A={1}、B={2} 互斥，但 A∪B≠Ω，故不对立；而 A={偶数} 与 A^c={奇数} 既互斥又对立。

**例 1.3**
掷一枚骰子，A = {偶数点} = {2,4,6}，则 A 的对立事件为 A^c = {1,3,5} = {奇数点}。  
「掷出 1」与「掷出 2」互斥，但不对立（因为还有 3,4,5,6）。

## 1.3 概率的公理化定义

**动机**：古典概型与几何概型只适用于「等可能」或「测度成比例」的场合；我们还需要对一般随机现象给出「概率」的统一定义，并保证由此推出的性质与直觉一致（如加法公式、条件概率等）。柯尔莫哥洛夫（Kolmogorov）把下面三条直观要求作为**公理**，由此可严格推出概率论的全部结论，且不依赖「等可能」的假定。

在公理化之前，先说明我们想要「概率」满足的直观性质：非负、全体样本空间概率为 1、互斥事件之并的概率等于各概率之和。

**定义 1.4（概率的公理化定义）**  
设 Ω 为样本空间，**F** 为 Ω 上满足一定封闭性的事件组成的集合（称为**事件域**或 σ-代数，此处不展开，在离散有限情形 F 可取为 Ω 的全部子集）。若实值函数 P: F → R 满足：

1. **非负性**：对任意 A ∈ F，有 P(A) ≥ 0；  
2. **规范性**：P(Ω) = 1；  
3. **可列可加性**：对任意两两互斥的事件列 A₁, A₂, …（Aᵢ∩Aⱼ = ∅，i≠j），有  
   **P(∪ₙ Aₙ) = Σₙ P(Aₙ)**  

则称 P 为（Ω, F）上的**概率**，三元组 (Ω, F, P) 称为**概率空间**。

**命题 1.1（由公理推出的基本性质）**  
设 P 为概率，A、B 为事件，则：

1. P(∅) = 0；  
2. 对任意 A，0 ≤ P(A) ≤ 1；  
3. P(A^c) = 1 − P(A)；  
4. 若 A ⊂ B，则 P(A) ≤ P(B)；  
5. **概率的加法公式**：P(A∪B) = P(A) + P(B) − P(A∩B)。

**证明（要点）**  
- (1) 由可列可加性，P(Ω) = P(Ω∪∅∪∅∪…) = P(Ω)+P(∅)+…，故 P(∅)=0。  
- (2) 非负性给出 P(A)≥0；又 A⊂Ω，故 P(A)≤P(Ω)=1。  
- (3) A 与 A^c 互斥且 A∪A^c=Ω，故 1=P(Ω)=P(A)+P(A^c)，即 P(A^c)=1−P(A)。  
- (4) B = A∪(B\A)，且 A 与 B\A 互斥，故 P(B)=P(A)+P(B\A)≥P(A)。  
- (5) A∪B = A∪(B\A)，其中 A 与 B\A 互斥；又 B = (A∩B)∪(B\A) 且 (A∩B) 与 (B\A) 互斥，故 P(B\A)=P(B)−P(A∩B)。于是 P(A∪B)=P(A)+P(B\A)=P(A)+P(B)−P(A∩B)。□  

**推论**：对两两互斥的有限个事件 A₁,…,Aₙ，有 P(A₁∪…∪Aₙ)=P(A₁)+…+P(Aₙ)。

## 1.4 古典概型与几何概型

当样本空间具有某种「对称性」时，可以**不依赖频率**而直接由对称性给出概率，这就是古典概型与几何概型。

### 1.4.1 古典概型

**定义 1.5（古典概型）**  
若随机试验满足：  
(1) 样本空间 Ω **有限**；  
(2) 每个**基本事件**（单点事件）**等可能**发生（对称性），  
则称该试验为**古典概型**。在古典概型中，对任意事件 A，定义  
**P(A) = |A| / |Ω|**  
其中 |A| 表示 A 中样本点的个数。

**例 1.4**  
掷一枚均匀骰子，Ω = {1,2,3,4,5,6}，|Ω|=6，每个基本事件概率 1/6。  
- A = {偶数点} = {2,4,6}，|A|=3，故 P(A)=3/6=1/2。  
- B = {点数大于 4} = {5,6}，P(B)=2/6=1/3。  
- A∩B = {6}，P(A∩B)=1/6；由加法公式 P(A∪B)=P(A)+P(B)−P(A∩B)=1/2+1/3−1/6=2/3。

**例 1.5（抽样问题）**  
从 52 张扑克中**随机**抽 5 张，求「恰好 2 张为 A」的概率。

**解**  
- 样本点总数：|Ω| = C(52,5) = 52!/(5!·47!) = 2 598 960。  
- 有利场合：先从 4 张 A 中选 2 张，再从其余 48 张中选 3 张，共 C(4,2)×C(48,3) = 6×17 296 = 103 776。  
- 故所求概率 = 103 776 / 2 598 960 ≈ 0.0399（约 4%）。

**代码验证（Python）**：`from math import comb; prob = comb(4,2)*comb(48,3)/comb(52,5); print(round(prob,4))` → 0.0399。

**注意**：「等可能」对应「随机抽」，即每张被抽到概率相同；若抽样方式不同（如不放回与放回），样本空间与概率会不同。

### 1.4.2 几何概型

当样本空间与「长度、面积、体积」等连续测度对应，且我们假设「落在某子区域内的概率与该子区域的测度成比例」时，就得到**几何概型**。

**定义 1.6（几何概型）**  
设样本空间 Ω 对应一有界区域（如区间、平面区域），其测度（长度、面积或体积）记为 L(Ω)，且假设「点落在区域 A 内的概率与 A 的测度成比例」。则对可测子集 A，定义  
**P(A) = L(A) / L(Ω)**。

**例 1.6**  
在区间 [0,1] 上随机取一点，求该点落在 [0.2, 0.5] 内的概率。  
- Ω = [0,1]，L(Ω)=1；A = [0.2, 0.5]，L(A)=0.3。  
- P(A)=0.3/1=0.3。

几何概型在量化中直接使用较少，但**连续型随机变量的概率**可以理解为某种几何概型的极限（用面积表示概率），因此概念上仍有用。

**核心概念**：随机现象与随机试验；样本空间 Ω 与样本点；事件（子集）、基本事件、必然事件、不可能事件；事件运算（并、交、差、补）及运算律；互斥（A∩B=∅）与对立（A∪B=Ω 且 A∩B=∅）；概率空间 (Ω, F, P)；古典概型与几何概型。

4. 古典概型：P(A)=|A|/|Ω|；几何概型：P(A)=L(A)/L(Ω)。

# 第二章 条件概率与独立性

## 2.1 条件概率

在实际问题中，我们常需在「已知某事件 B 已发生」的前提下，讨论另一事件 A 发生的概率。例如：已知某日市场上涨（B），某股票也上涨（A）的概率；已知因子暴露为正（B），下期收益为正（A）的概率。这类概率称为**条件概率**。

**定义 2.1（条件概率）**  
设 A、B 为两事件，且 P(B) > 0。在**事件 B 已发生**的条件下，事件 A 发生的**条件概率**定义为  
**P(A|B) = P(A∩B) / P(B)**。

**解释**：把样本空间「收缩」到 B；在 B 的范围内，A 发生等价于 A∩B 发生，因此 A 在 B 中的「比例」为 P(A∩B)/P(B)。

**例 2.1**  
掷一枚均匀骰子，已知「点数不超过 4」（B = {1,2,3,4}），求「点数为偶数」（A = {2,4}）的概率。  
- P(B)=4/6，P(A∩B)=P({2,4})=2/6。  
- P(A|B)=P(A∩B)/P(B)=(2/6)/(4/6)=1/2。  
（也可直接理解为：在 {1,2,3,4} 中偶数有 2 个，共 4 个，故 2/4=1/2。）

由定义立即可得**乘法公式**：  
**P(A∩B) = P(B)·P(A|B) = P(A)·P(B|A)**  
（要求出现分母的概概率大于 0。）  
乘法公式可推广到多个事件：  
P(A₁∩A₂∩A₃) = P(A₁)·P(A₂|A₁)·P(A₃|A₁∩A₂)，依此类推。

---

## 2.2 全概率公式

当事件 A 的发生可能与多种「原因」B₁,…,Bₙ 之一有关，且这些原因**两两互斥**且**并起来是全体**（即构成样本空间的一个**划分**）时，可以把 P(A) 按「先发生哪个 Bᵢ」拆开计算。

**定义 2.2（划分）**  
若事件 B₁, B₂, …, Bₙ 满足：(1) Bᵢ∩Bⱼ = ∅（i≠j）；(2) B₁∪B₂∪…∪Bₙ = Ω，则称 {B₁,…,Bₙ} 为样本空间 Ω 的一个**划分**（或称完备事件组）。

**定理 2.1（全概率公式）**  
设 {B₁,…,Bₙ} 为 Ω 的一个划分，且对任意 i 有 P(Bᵢ) > 0。则对任意事件 A，  
**P(A) = Σᵢ₌₁ⁿ P(Bᵢ)P(A|Bᵢ)**。

**证明**：A = A∩Ω = A∩(B₁∪…∪Bₙ) = (A∩B₁)∪…∪(A∩Bₙ)，且 (A∩Bᵢ) 两两互斥，故  
P(A)=Σᵢ P(A∩Bᵢ)=Σᵢ P(Bᵢ)P(A|Bᵢ)。□  

**例 2.2（全概率公式：生产线次品率）**  
某工厂有三条生产线，产量占比分别为 50%、30%、20%，次品率分别为 1%、2%、3%。任取一件产品，求其为次品的概率。

**解**  
- 划分：B₁ =「来自第 1 条线」，B₂ =「来自第 2 条线」，B₃ =「来自第 3 条线」；A =「次品」。  
- P(A) = P(B₁)P(A|B₁) + P(B₂)P(A|B₂) + P(B₃)P(A|B₃)  
  = 0.5×0.01 + 0.3×0.02 + 0.2×0.03 = 0.005 + 0.006 + 0.006 = **0.017**。  
- 即任取一件，次品概率为 1.7%。

**代码验证**：

```python
P_B = [0.5, 0.3, 0.2]
P_A_given_B = [0.01, 0.02, 0.03]
P_A = sum(pb * pa for pb, pa in zip(P_B, P_A_given_B))
print(f"P(次品) = {P_A}")  # 0.017
```

## 2.3 贝叶斯公式

全概率公式解决的是「由原因推结果」：已知各 Bᵢ 及 P(A|Bᵢ)，求 P(A)。**贝叶斯公式**解决的是「由结果推原因」：观察到 A 发生了，问「原因是 Bⱼ」的概率是多少。

**定理 2.2（贝叶斯公式）**  
设 {B₁,…,Bₙ} 为 Ω 的划分，P(Bᵢ)>0，P(A)>0。则  
**P(Bⱼ|A) = P(Bⱼ)P(A|Bⱼ) / Σᵢ P(Bᵢ)P(A|Bᵢ)**。

**证明**：由条件概率定义，P(Bⱼ|A)=P(Bⱼ∩A)/P(A)。分子用乘法公式 P(Bⱼ∩A)=P(Bⱼ)P(A|Bⱼ)；分母用全概率公式 P(A)=Σᵢ P(Bᵢ)P(A|Bᵢ)。□  

**术语**：  
- **先验概率** P(Bⱼ)：在未观测到 A 之前，对「原因是 Bⱼ」的判断。  
- **似然** P(A|Bⱼ)：在原因 Bⱼ 下，观察到 A 的概率。  
- **后验概率** P(Bⱼ|A)：观察到 A 之后，对「原因是 Bⱼ」的更新判断。

**例 2.3**  
接例 2.2，若任取一件发现是次品，求它来自第 2 条生产线的概率。  
- P(B₂|A)=P(B₂)P(A|B₂)/P(A)=0.3×0.02/0.017=0.006/0.017≈0.353。  
即观察到次品后，「来自第 2 条线」的概率由先验 30% 更新为约 35.3%。

在量化中，Bⱼ 可表示不同的「市场状态」或「因子有效/无效」，A 表示观测到的收益或信号；贝叶斯公式用于**根据新数据更新对状态的判断**（贝叶斯推断、动态配置等）。

---

## 2.4 事件的独立性

直观上，「独立」表示 A 是否发生与 B 是否发生**互不影响**。数学上用「条件概率等于无条件概率」或「交的概率等于概率之积」来刻画。

**定义 2.3（两个事件的独立性）**  
若 **P(A∩B) = P(A)P(B)**，则称事件 A 与 B **相互独立**。

**性质**：当 P(B)>0 时，A 与 B 独立 ⟺ P(A|B)=P(A)；当 P(A)>0 时，A 与 B 独立 ⟺ P(B|A)=P(B)。  
（证明：若独立，则 P(A|B)=P(A∩B)/P(B)=P(A)P(B)/P(B)=P(A)；反之由 P(A|B)=P(A) 得 P(A∩B)=P(A)P(B)。）

**定义 2.4（多个事件的独立性）**  
- **两两独立**：任意两个事件独立，即对 i≠j 有 P(Aᵢ∩Aⱼ)=P(Aᵢ)P(Aⱼ)。  
- **相互独立**：对任意 k 个不同下标 i₁,…,iₖ，有 P(Aᵢ₁∩…∩Aᵢₖ)=P(Aᵢ₁)…P(Aᵢₖ)。

**注意（两两独立与相互独立的辨析）**  
**两两独立不蕴含相互独立**。即「任意两个事件独立」不能推出「任意多个事件交的概率等于各自概率之积」。  
**经典反例**：掷两枚均匀硬币，令 A=「第一枚正面」，B=「第二枚正面」，C=「恰好一枚正面」。  
- P(A)=P(B)=1/2，P(C)=1/2（正反或反正）。  
- P(A∩B)=1/4=P(A)P(B)，P(A∩C)=1/4=P(A)P(C)，P(B∩C)=1/4=P(B)P(C)，故 A、B、C **两两独立**。  
- 但 A∩B∩C=∅（不可能「第一枚正面且第二枚正面且恰好一枚正面」），故 P(A∩B∩C)=0；而 P(A)P(B)P(C)=1/8，故 **不相互独立**。  
在量化建模中，若假设多期收益「相互独立」，则联合分布可写为各边缘之积；仅「两两不相关」或「两两独立」不足以得到和的方差等于方差之和等结论，需用「相互独立」。

在量化建模中，常**假设**各期收益相互独立或至少不相关，以便简化方差与协方差的计算；该假设需通过检验与稳健性分析来审视。

---

## 本章小结（第二章）

**核心概念**：条件概率 P(A|B)（在 B 发生下 A 发生的概率）；划分（完备事件组）；先验、似然、后验；事件的独立性（两两独立与相互独立）。

**主要公式与结论**：  
1. 条件概率：P(A|B)=P(A∩B)/P(B)，P(B)>0。  
2. 乘法公式：P(A∩B)=P(B)P(A|B)=P(A)P(B|A)。  
3. 全概率公式：P(A)=Σᵢ P(Bᵢ)P(A|Bᵢ)（{Bᵢ} 为划分）。  
4. 贝叶斯公式：P(Bⱼ|A)=P(Bⱼ)P(A|Bⱼ)/Σᵢ P(Bᵢ)P(A|Bᵢ)。  
5. 独立性：P(A∩B)=P(A)P(B)；两两独立 ⇏ 相互独立（见反例）。

# 第三章 随机变量与分布

**动机**：我们不仅关心「事件是否发生」，更关心「与结果相关的数值」（如收益、损失、次数）。随机变量把样本点映射为实数，从而可以用期望、方差、分位数等刻画收益与风险。

## 3.1 随机变量

**定义 3.1（随机变量）**  
设 (Ω, F, P) 为概率空间。若对每个 ω∈Ω，都有唯一的实数 X(ω) 与之对应，且对任意实数 x，**{ω : X(ω) ≤ x}** 都是事件（属于 F），则称 X 为**随机变量**。

通俗说：随机变量是「把试验结果数值化」的函数。这样，我们关心的不再是具体的 ω，而是 X 落在某个区间或集合的概率，例如 P(X ≤ x)、P(a < X ≤ b)。

- **离散型随机变量**：X 只取**有限个或可列个**值（如整数）。  
- **连续型随机变量**：X 的取值充满某个区间（或 R），且对任意单点 a 有 P(X=a)=0，用**密度函数**描述。

---

## 3.2 离散型随机变量

### 3.2.1 分布列

**定义 3.2（分布列）**  
设 X 为离散型随机变量，其全部可能取值为 x₁, x₂, …（可列）。称  
**pₖ = P(X = xₖ)**，k = 1, 2, …  
为 X 的**概率分布列**（或分布律）。满足：(1) pₖ ≥ 0；(2) Σₖ pₖ = 1。

**定义 3.3（分布函数）**  
对任意实数 x，**F(x) = P(X ≤ x) = Σ_{xₖ≤x} pₖ** 称为 X 的**分布函数**。F 单调不减、右连续，且 F(−∞)=0，F(+∞)=1。

### 3.2.2 期望与方差

**定义 3.4（数学期望）**  
若级数 Σₖ |xₖ| pₖ 收敛，则称 **E(X) = Σₖ xₖ pₖ** 为 X 的**数学期望**（或均值）。

**定义 3.5（方差与标准差）**  
**Var(X) = E[(X−E(X))²]** 称为 X 的**方差**；**σ(X) = √Var(X)** 称为**标准差**。

**命题 3.1（方差的计算公式）**  
若 E(X)、E(X²) 存在，则 **Var(X) = E(X²) − [E(X)]²**。

**证明**：记 μ = E(X)。由期望的线性性，
\[
\text{Var}(X) = E[(X-\mu)^2] = E[X^2 - 2\mu X + \mu^2] = E(X^2) - 2\mu E(X) + \mu^2 = E(X^2) - 2\mu^2 + \mu^2 = E(X^2) - [E(X)]^2.
\]
□  
在实际计算时，用 **E(X²)−[E(X)]²** 往往比直接算 E[(X−μ)²] 更简便（离散型即 Σ xₖ² pₖ − (Σ xₖ pₖ)²）。

**性质**：E(aX+b)=aE(X)+b；Var(aX+b)=a²Var(X)。

### 3.2.3 常用离散分布

**（1）伯努利分布**  
X 只取 0 和 1，P(X=1)=p，P(X=0)=1−p，0<p<1。记作 X ∼ Bernoulli(p)。  
- E(X)=p；Var(X)=p(1−p)。  
- 用于描述「单次试验是否成功」（如单日是否上涨）。

**（2）二项分布**  
n 次**独立**的伯努利试验中「成功」的次数 X，P(X=k)=C(n,k) p^k (1−p)^{n−k}，k=0,1,…,n。记作 X ∼ B(n, p)。  
- E(X)=np；Var(X)=np(1−p)。  
- 推导：X=X₁+…+Xₙ，Xᵢ 为第 i 次试验的示性（1 成功、0 失败），E(Xᵢ)=p，Var(Xᵢ)=p(1−p)，由独立性 E(X)=np，Var(X)=np(1−p)。

**（3）泊松分布**  
X 取非负整数，P(X=k)=λ^k e^{−λ}/k!，k=0,1,2,…，λ>0。记作 X ∼ P(λ)。  
- E(X)=λ；Var(X)=λ。  
- 用于描述**稀有事件**在固定时间或空间内的发生次数（如某时段订单笔数、违约次数）；当 n 大、p 小时，B(n,p) 可近似为 P(np)。

**例 3.1**  
掷 10 次均匀硬币，X 为正面次数，则 X ∼ B(10, 1/2)。  
- P(X=5)=C(10,5)(1/2)^{10}=252/1024≈0.246。  
- E(X)=5，Var(X)=10×0.25=2.5。

**例 3.2（二项分布：抽样次品数，含完整题干与代码）**  
某工厂产品次品率为 5%，随机抽样 252 件，求次品数 X 不超过 10 的概率。

**解**  
- 设 X 为 252 件中次品件数，则 X ∼ B(n=252, p=0.05)。  
- 所求为 P(X ≤ 10) = Σ_{k=0}^{10} C(252,k)(0.05)^k(0.95)^{252−k}。  
- 直接手算较繁，用分布函数：P(X ≤ 10) = F(10)，其中 F 为 B(252, 0.05) 的 CDF。

**数值代入**：n=252，p=0.05，k=0,…,10。  
E(X)=252×0.05=12.6，Var(X)=252×0.05×0.95=11.97；由正态近似（见第四章），(X−12.6)/√11.97 近似 N(0,1)，故 P(X≤10)≈Φ((10−12.6)/√11.97)=Φ(−0.75)≈0.23。精确值由代码计算。

**代码（Python）与结果说明**：

```python
from scipy.stats import binom
n, p = 252, 0.05
prob = binom.cdf(10, n, p)
print(f"P(X≤10) = {prob:.4f}")  # 约 0.924，即 92.4%
# 说明：精确二项 CDF 得约 92.4%，表示在 252 件中次品数不超过 10 的概率为 92.4%
```

**注意**：二项分布要求 n 次试验**独立**、每次成功概率均为 p；若抽样为不放回且总体有限，应为超几何分布；当总体很大时二项是良好近似。

## 3.3 连续型随机变量

### 3.3.1 密度函数与分布函数

**定义 3.6（概率密度函数）**  
若存在非负可积函数 f(x)，使得对任意实数 a<b，有  
**P(a < X ≤ b) = ∫ₐᵇ f(x) dx**，  
则称 f 为 X 的**概率密度函数**（pdf）。此时 **F(x)=P(X≤x)=∫₋∞ˣ f(t)dt** 为**分布函数**。

**性质**：∫₋∞^{+∞} f(x)dx=1；P(X=a)=0；P(a<X≤b)=F(b)−F(a)=∫ₐᵇ f(x)dx；在 f 连续点处 F'(x)=f(x)。

**定义 3.7（数学期望与方差，连续型）**  
若 ∫ |x|f(x)dx < ∞，则 **E(X)=∫ x f(x)dx**；**Var(X)=E(X²)−[E(X)]²**，其中 E(X²)=∫ x² f(x)dx。

### 3.3.2 常见连续分布

**（1）均匀分布 U(a,b)**  
f(x)=1/(b−a)，a<x<b；其余为 0。  
- E(X)=(a+b)/2；Var(X)=(b−a)²/12。  
- 例：在 [0,1] 上随机取点，坐标 X∼U(0,1)。

**（2）正态分布 N(μ, σ²)**  
**f(x) = (1/√(2πσ²)) exp(−(x−μ)²/(2σ²))**，x∈R，σ>0。  
- E(X)=μ；Var(X)=σ²。  
- **标准化**：若 X∼N(μ,σ²)，令 Z=(X−μ)/σ，则 **Z∼N(0,1)**（标准正态）。  
- **线性组合**：若 X₁,…,Xₙ 相互独立，Xᵢ∼N(μᵢ,σᵢ²)，则 a₁X₁+…+aₙXₙ∼N(Σaᵢμᵢ, Σaᵢ²σᵢ²)。  
- 标准正态的分布函数记作 Φ(x)；分位数 z_α 满足 P(Z>z_α)=α，如 z_{0.025}≈1.96。  
- 在量化中：收益常用正态近似；置信区间与假设检验大量使用正态与 t 分布。

**（3）对数正态分布**  
若 **ln(X)∼N(μ,σ²)**，则称 X 服从**对数正态**。  
- 密度：x>0 时 f(x)=(1/(xσ√(2π)))exp(−(ln x−μ)²/(2σ²))；x≤0 时为 0。  
- E(X)=exp(μ+σ²/2)；Var(X)=exp(2μ+σ²)(e^{σ²}−1)。  
- 用于描述**价格**（价格取对数后常近似正态）。

**（4）指数分布**  
f(x)=λe^{−λx}，x≥0；λ>0。  
- E(X)=1/λ；Var(X)=1/λ²。  
- 用于描述**等待时间**、寿命等；与泊松过程紧密相关。

### 3.3.3 分位数

**定义 3.8（p 分位数）**  
对 0<p<1，若实数 xₚ 满足 P(X≤xₚ)≥p 且 P(X≥xₚ)≥1−p，则称 xₚ 为 X 的**p 分位数**。  
当 F 严格单调递增时，**xₚ=F⁻¹(p)**。  
- **中位数**：p=0.5。  
- **四分位数**：0.25、0.5、0.75。  
- **尾部**：0.01、0.05、0.95、0.99 等，用于**VaR**等风险度量。

---

## 3.4 多维随机变量（简述）

设 (X,Y) 为二维随机变量。  
- **联合分布函数**：F(x,y)=P(X≤x, Y≤y)。  
- **边缘分布**：F_X(x)=P(X≤x)=F(x,+∞)；类似 F_Y(y)。  
- **独立性**：X 与 Y 独立 ⟺ 对任意 x,y 有 F(x,y)=F_X(x)F_Y(y)；离散型 ⟺ P(X=xᵢ,Y=yⱼ)=P(X=xᵢ)P(Y=yⱼ)；连续型 ⟺ 联合密度 f(x,y)=f_X(x)f_Y(y)。  
- **协方差**：Cov(X,Y)=E[(X−E(X))(Y−E(Y))]=E(XY)−E(X)E(Y)。  
- **相关系数**：ρ_{XY}=Cov(X,Y)/(σ_X σ_Y)，−1≤ρ≤1；独立 ⇒ Cov=0（反之不成立）。  
在量化中：多资产收益的协方差与相关系数用于组合方差与风险分解。

## 本章小结（第三章）

**核心概念**：随机变量（离散型/连续型）；分布列与密度函数；分布函数 F(x)=P(X≤x)；数学期望与方差；常用离散分布（伯努利、二项、泊松）与连续分布（均匀、正态、对数正态、指数）；标准化与分位数。

**主要公式与结论**：  
1. 离散型：E(X)=Σ xₖ pₖ，Var(X)=E(X²)−[E(X)]²。  
2. 连续型：P(a<X≤b)=∫ₐᵇ f(x)dx，E(X)=∫ x f(x)dx，Var(X)=E(X²)−[E(X)]²。  
3. 二项 B(n,p)：E(X)=np，Var(X)=np(1−p)。正态 N(μ,σ²)：标准化 Z=(X−μ)/σ∼N(0,1)。  
4. 分位数：xₚ 满足 P(X≤xₚ)≥p 且 P(X≥xₚ)≥1−p；F 严格单调时 xₚ=F⁻¹(p)。

# 第四章 大数定律与中心极限定理

## 4.1 大数定律

设 X₁, X₂, … 为**独立同分布**的随机变量序列，E(X₁)=μ。记**样本均值** **X̄ₙ = (X₁+…+Xₙ)/n**。

**定理 4.1（弱大数定律）**  
在 E(X₁) 存在的条件下，当 n→∞ 时，**X̄ₙ 依概率收敛于 μ**，即对任意 ε>0，  
**lim_{n→∞} P(|X̄ₙ−μ| > ε) = 0**。

**含义**：在大量重复下，**样本均值会稳定到总体期望附近**；样本量越大，偏离 μ 超过 ε 的概率越小。  
在量化中：用**历史平均收益**估计期望收益、用**多期 IC 的平均**估计因子预测能力，其理论依据即大数定律。

**定理 4.2（强大数定律）**  
在稍强条件（如 E|X₁|<∞）下，**X̄ₙ 几乎必然收敛于 μ**，即 P(lim_{n→∞} X̄ₙ=μ)=1。  
含义：长期来看，样本均值**几乎一定**等于 μ（与弱大数的「概率趋于 0」相比，结论更强）。

---

## 4.2 中心极限定理（CLT）

大数定律说明 X̄ₙ **收敛到谁**（μ）；中心极限定理说明 **收敛时波动有多大**——即 (X̄ₙ−μ) 的「缩放版本」的**分布**趋于标准正态。

**定理 4.3（林德伯格–列维中心极限定理）**  
设 X₁, X₂, … 独立同分布，E(X₁)=μ，Var(X₁)=σ²，0<σ²<∞。记 Sₙ=X₁+…+Xₙ，则**标准化和**  
**Zₙ = (Sₙ − nμ)/(σ√n) = (X̄ₙ−μ)/(σ/√n)**  
当 n→∞ 时，**依分布收敛到标准正态 N(0,1)**，即对任意实数 z，  
**lim_{n→∞} P(Zₙ ≤ z) = Φ(z)**。

**等价表述**：  
- **X̄ₙ** 的分布**近似** **N(μ, σ²/n)**：样本量越大，X̄ₙ 的方差 σ²/n 越小，越集中在 μ 附近。  
- **Sₙ** 的分布近似 **N(nμ, nσ²)**：独立同分布之和，大样本下近似正态。

**应用**：  
- 无论 X₁ 的分布形状如何（只要期望与方差有限），**大样本下样本均值近似正态**，因此可以用**正态分位数**做置信区间、用**t 或渐近正态**做假设检验。  
- 组合收益 = 多期收益之和，若各期独立同分布，则长期组合收益分布近似正态（在量化中常用）。  
- 蒙特卡洛估计的渐近正态性也来自 CLT。

**例 4.1（CLT 在年化收益与波动率中的应用）**  
设每日收益率 r_t 独立同分布，E(r_t)=0.0005（即 0.05%），Var(r_t)=0.0001（即日波动率 σ=1%）。考虑 252 个交易日（约一年）。  
- 累计收益 S = r₁+…+r₂₅₂，由 CLT 知 S 近似服从正态，且 E(S)=252×0.0005=0.126，Var(S)=252×0.0001=0.0252。  
- 代入：即 S 近似 ∼ N(0.126, 0.0252)。故**年化收益率**（简单加总）约 12.6%，**年化波动率**（标准差）为 √0.0252≈0.159，即约 15.9%。  
- 若关心「一年后收益为负」的概率：P(S<0) ≈ Φ((0−0.126)/√0.0252)=Φ(−0.79)≈0.21，即约 21%。  
**单位说明**：此处收益为小数形式（0.0005 表示 0.05%）；若用百分数表示，E(r_t)=0.05%，年化约 12.6%。

**代码验证（Python）**：

```python
import numpy as np
from scipy import stats
mu_d, var_d = 0.0005, 0.0001
n_days = 252
mu_S = n_days * mu_d
var_S = n_days * var_d
# P(S < 0)
z = (0 - mu_S) / np.sqrt(var_S)
p_neg = stats.norm.cdf(z)
print(f"年化期望={mu_S:.4f}, 年化方差={var_S:.4f}, 年化波动率={np.sqrt(var_S):.4f}")
print(f"P(年收益<0) ≈ Φ({z:.2f}) = {p_neg:.2%}")  # 约 21%
```

---

## 4.3 大数定律与中心极限定理的关系

- **大数定律**：关心 **X̄ₙ → μ**（极限是**常数**）。  
- **中心极限定理**：关心 **(X̄ₙ−μ)/(σ/√n)** 的**分布**趋于 N(0,1)（极限是**分布**）。  
二者结合：既知道「样本均值收敛到 μ」，又知道「收敛时以 1/√n 的速度波动，且波动形状为正态」。

## 本章小结（第四章）

**核心概念**：独立同分布序列；样本均值 X̄ₙ；依概率收敛与几乎必然收敛；依分布收敛；弱大数定律、强大数定律、中心极限定理（CLT）。

**主要公式与结论**：  
1. 弱大数定律：lim_{n→∞} P(|X̄ₙ−μ|>ε)=0（X̄ₙ 依概率收敛于 μ）。  
2. 中心极限定理：Zₙ=(X̄ₙ−μ)/(σ/√n) →^d N(0,1)；X̄ₙ 近似 N(μ, σ²/n)，Sₙ 近似 N(nμ, nσ²)。  
3. 量化应用：历史均值估计期望、t 检验/置信区间的渐近依据、组合收益的正态近似、蒙特卡洛估计。

# 第五章 与量化研究的衔接

- **收益与风险**：收益率序列视为随机变量；**期望**=收益中心，**方差/标准差**=波动风险，**分位数**=VaR/CVaR。  
- **假设检验**：均值是否为零、相关系数是否显著，都基于「在概率模型下，统计量的分布」；CLT 保证大样本下近似正态。  
- **贝叶斯**：先验（状态/信念）+ 似然（观测）→ 后验（更新）；用于状态推断、动态配置、因子择时。  
- **大数定律与 CLT**：回测样本外稳定性、t 统计量渐近正态、蒙特卡洛收敛，都建立在这些定理之上。

**学习建议**：能默写条件概率、贝叶斯公式、正态标准化、(X̄ₙ−μ)/(σ/√n)→N(0,1)；能在一维连续型下计算 E(X)、Var(X) 和分位数；能区分「大数定律说的是收敛到谁」「CLT 说的是收敛时的分布形状」。
