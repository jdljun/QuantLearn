# 第 1 章 一元线性回归

**动机**：因子与收益的关系常简化为线性（收益对单一因子的暴露）；一元回归是多元与 OLS 矩阵形式的基础，且斜率 b₁ = Cov(X,Y)/Var(X) 与 IC、因子暴露的边际效应一致。先在一元情形下把模型、最小二乘与几何意义讲清，再推广到多元。

---

## 1.1 模型与记号

### 1.1.1 总体模型

**定义 1.1.1（一元线性回归总体模型）**  
设 **Y** 为被解释变量（如收益率），**X** 为解释变量（如因子暴露）。**总体模型**为：
$$Y = \beta_0 + \beta_1 X + \varepsilon$$
其中 **β₀** 为截距，**β₁** 为斜率，**ε** 为随机误差项。常假定：
- \(E(\varepsilon)=0\)（误差均值为零）；
- \(\operatorname{Var}(\varepsilon)=\sigma^2\)（同方差）；
- \(\varepsilon\) 与 \(X\) 不相关（或更强：\(\varepsilon\) 与 \(X\) 独立）。

在上述假定下，\(E(Y|X)=\beta_0+\beta_1 X\)，即给定 X 时 Y 的条件期望为直线。

### 1.1.2 样本与拟合

**定义 1.1.2（样本与拟合）**  
设有 **n** 组观测 \((X_i, Y_i)\)，\(i=1,\ldots,n\)。**拟合直线**为 \(\hat{Y}_i = b_0 + b_1 X_i\)，其中 \(b_0,b_1\) 为待估参数。**残差**定义为：
$$e_i = Y_i - \hat{Y}_i = Y_i - b_0 - b_1 X_i.$$

---

## 1.2 最小二乘与正规方程

### 1.2.1 最小二乘准则

**动机**：需要一种准则从样本中估计 \(\beta_0,\beta_1\)。最小二乘法选择 \(b_0,b_1\) 使残差平方和最小，在 Gauss-Markov 假定下得到的估计量具有良好性质（无偏、方差最小等）。

**定义 1.2.1（普通最小二乘，OLS）**  
**OLS 估计量** \((b_0, b_1)\) 是使残差平方和
$$S(b_0,b_1) = \sum_{i=1}^n e_i^2 = \sum_{i=1}^n (Y_i - b_0 - b_1 X_i)^2$$
达到最小的解。

### 1.2.2 正规方程与解的推导

对 \(S\) 分别关于 \(b_0\) 和 \(b_1\) 求偏导并令其为零：

**关于 b₀：**
$$\frac{\partial S}{\partial b_0} = -2\sum_{i=1}^n (Y_i - b_0 - b_1 X_i) = 0.$$
即
$$n\bar{Y} - n b_0 - n b_1 \bar{X} = 0 \quad \Rightarrow \quad b_0 = \bar{Y} - b_1 \bar{X}.$$

**关于 b₁：**
$$\frac{\partial S}{\partial b_1} = -2\sum_{i=1}^n (Y_i - b_0 - b_1 X_i) X_i = 0.$$
将 \(b_0 = \bar{Y} - b_1 \bar{X}\) 代入得
$$\sum_i (Y_i - \bar{Y} + b_1 \bar{X} - b_1 X_i) X_i = 0.$$
整理：
$$\sum_i (Y_i - \bar{Y}) X_i = b_1 \sum_i (X_i - \bar{X}) X_i = b_1 \sum_i (X_i - \bar{X})^2.$$
因此
$$b_1 = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sum_{i=1}^n (X_i - \bar{X})^2} = \frac{\widehat{\operatorname{Cov}}(X,Y)}{\widehat{\operatorname{Var}}(X)},$$
其中 \(\widehat{\operatorname{Cov}}(X,Y) = \frac{1}{n}\sum_i (X_i-\bar{X})(Y_i-\bar{Y})\)，\(\widehat{\operatorname{Var}}(X) = \frac{1}{n}\sum_i (X_i-\bar{X})^2\) 为样本协方差与样本方差（有时分母用 \(n-1\)，此处与分子约去后等价于用 \(n\) 的写法）。

**定理 1.2.1（一元 OLS 解的显式）**  
在 \(\widehat{\operatorname{Var}}(X)>0\) 时，OLS 解唯一，且
$$b_1 = \frac{\widehat{\operatorname{Cov}}(X,Y)}{\widehat{\operatorname{Var}}(X)}, \qquad b_0 = \bar{Y} - b_1 \bar{X}.$$
上述两式称为**正规方程**的解。

**推导小结**：关键步骤是代入 \(b_0=\bar{Y}-b_1\bar{X}\) 后得到关于 \(b_1\) 的一元方程，分子为 X 与 Y 的样本协方差，分母为 X 的样本方差。

---

## 1.3 几何与统计意义

### 1.3.1 斜率的含义

**性质 1.3.1**  
\(b_1\) 表示：X 每增加 1 单位时，Y 的拟合值（在样本中即期望的估计）增加 \(b_1\) 单位。在总体中对应 \(\beta_1\)。

### 1.3.2 正交性

**定理 1.3.2（残差与解释变量的样本正交性）**  
OLS 拟合下，残差 \(e_i\) 与 \(X_i\) 的样本协方差为零，即
$$\sum_{i=1}^n e_i (X_i - \bar{X}) = 0, \qquad \sum_{i=1}^n e_i = 0.$$

**证明**：由正规方程 \(\partial S/\partial b_0=0\) 得 \(\sum_i e_i = 0\)；由 \(\partial S/\partial b_1=0\) 得 \(\sum_i e_i X_i = 0\)，故 \(\sum_i e_i (X_i - \bar{X}) = \sum_i e_i X_i - \bar{X}\sum_i e_i = 0\)。

**几何意义**：拟合值 \(\hat{Y}\) 是 Y 在由常数列 **1** 与 **X** 张成的平面上的**正交投影**；残差 **e** 与该平面正交，故与 1、X 的样本内积均为 0。

---

## 1.4 例题

**例题 1.4.1**  
某因子暴露 X 与收益率 Y 的 5 组样本如下：

| i | 1 | 2 | 3 | 4 | 5 |
|---|----|----|----|----|----|
| X_i | 1 | 2 | 3 | 4 | 5 |
| Y_i | 2.1 | 3.9 | 5.8 | 7.2 | 9.0 |

求 OLS 拟合的 \(b_0\)、\(b_1\) 及 \(R^2\)（决定系数，见第 3 章）。

**解**：  
\(\bar{X}=3\)，\(\bar{Y}=5.6\)。  
\(\sum_i (X_i-\bar{X})^2 = 10\)，\(\sum_i (X_i-\bar{X})(Y_i-\bar{Y}) = (1-3)(2.1-5.6)+\cdots = 11.6\)。  
故
$$b_1 = \frac{11.6}{10} = 1.16, \qquad b_0 = 5.6 - 1.16\times 3 = 2.12.$$
拟合方程 \(\hat{Y} = 2.12 + 1.16 X\)。  
\(TSS = \sum_i (Y_i-\bar{Y})^2\)，\(RSS = \sum_i e_i^2\)，\(R^2 = 1 - RSS/TSS\)。计算得 \(TSS=22.64\)，\(RSS\approx 0.304\)，故 \(R^2 \approx 0.987\)。

**代码与结果**：
```python
import numpy as np
X = np.array([1, 2, 3, 4, 5])
Y = np.array([2.1, 3.9, 5.8, 7.2, 9.0])
n = len(X)
xbar, ybar = X.mean(), Y.mean()
cov_xy = ((X - xbar) * (Y - ybar)).sum() / n
var_x = ((X - xbar)**2).sum() / n
b1 = cov_xy / var_x
b0 = ybar - b1 * xbar
print(f"b0 = {b0:.4f}, b1 = {b1:.4f}")  # b0 = 2.1200, b1 = 1.1600
Yhat = b0 + b1 * X
RSS = ((Y - Yhat)**2).sum()
TSS = ((Y - ybar)**2).sum()
R2 = 1 - RSS / TSS
print(f"R2 = {R2:.4f}")  # R2 ≈ 0.9866
```
输出：b0 = 2.1200, b1 = 1.1600，R² ≈ 0.9866，表示该直线解释了约 98.7% 的样本方差。

---

## 1.5 注意

**注意（方差为零）**  
当 \(\widehat{\operatorname{Var}}(X)=0\) 时所有 \(X_i\) 相同，正规方程分母为零，\(b_1\) 无定义；此时不应做一元回归，或需增加解释变量。

**注意（相关与因果）**  
\(b_1\) 只刻画 X 与 Y 的**线性相关**与斜率，不表示因果；因果推断需要额外假定（如外生性、工具变量等）。

---

## 本章小结（第 1 章）

**核心概念**：一元线性总体模型 \(Y=\beta_0+\beta_1 X+\varepsilon\)；样本拟合与残差；OLS 最小化残差平方和；正规方程；正交性。

**主要公式与结论**  
1. OLS 解：\(b_1 = \widehat{\operatorname{Cov}}(X,Y)/\widehat{\operatorname{Var}}(X)\)，\(b_0 = \bar{Y} - b_1 \bar{X}\)。  
2. 最小化目标：\(\sum_i e_i^2 = \sum_i (Y_i - b_0 - b_1 X_i)^2\)。  
3. 残差与 X 的样本协方差为 0；拟合值 \(\hat{Y}\) 为 Y 在 \((1,X)\) 张成平面上的正交投影。

# 第 2 章 多元线性回归与 OLS

**动机**：实际中常用多个因子（市值、估值、动量等）联合解释收益，需要多元线性模型与矩阵形式的 OLS，并得到 \(\hat{\beta}=(X^\top X)^{-1}X^\top Y\) 及拟合、残差的矩阵表示。

---

## 2.1 矩阵形式与假定

### 2.1.1 矩阵形式

**定义 2.1.1（多元线性回归模型）**  
**总体模型**为
$$\boldsymbol{Y} = \boldsymbol{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon},$$
其中 \(\boldsymbol{Y}\) 为 \(n\times 1\) 的被解释变量向量，\(\boldsymbol{X}\) 为 \(n\times (p+1)\) 的设计矩阵（第一列常为全 1，对应截距），\(\boldsymbol{\beta}\) 为 \((p+1)\times 1\) 的系数向量，\(\boldsymbol{\varepsilon}\) 为 \(n\times 1\) 的误差向量。第 \(i\) 行对应：
$$Y_i = \beta_0 + \beta_1 X_{i1} + \cdots + \beta_p X_{ip} + \varepsilon_i.$$

### 2.1.2 经典假定

**定义 2.1.2（经典线性回归假定）**  
1. **线性**：\(E(\boldsymbol{Y}|\boldsymbol{X}) = \boldsymbol{X}\boldsymbol{\beta}\)，即 \(E(\boldsymbol{\varepsilon}|\boldsymbol{X})=\boldsymbol{0}\)（严格外生或至少 \(\boldsymbol{\varepsilon}\) 与 \(\boldsymbol{X}\) 不相关）。  
2. **同方差与无自相关**：\(\operatorname{Var}(\boldsymbol{\varepsilon}|\boldsymbol{X}) = \sigma^2 \boldsymbol{I}_n\)。  
3. **秩条件**：\(\operatorname{rank}(\boldsymbol{X}) = p+1\)（无完全共线性，即 \(\boldsymbol{X}\) 列满秩）。

---

## 2.2 正规方程与 OLS 解

### 2.2.1 最小化与正规方程

**定义 2.2.1（多元 OLS）**  
OLS 估计量 \(\hat{\boldsymbol{\beta}}\) 最小化
$$\|\boldsymbol{Y} - \boldsymbol{X}\boldsymbol{\beta}\|^2 = (\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta})^\top (\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta}).$$

**推导**：记 \(f(\boldsymbol{\beta}) = (\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta})^\top (\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta})\)。对 \(\boldsymbol{\beta}\) 求导（向量求导）：
$$\frac{\partial f}{\partial \boldsymbol{\beta}} = -2\boldsymbol{X}^\top (\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta}).$$
令其为零得 **正规方程**：
$$\boldsymbol{X}^\top \boldsymbol{X}\boldsymbol{\beta} = \boldsymbol{X}^\top \boldsymbol{Y}.$$

**定理 2.2.2（OLS 估计量）**  
当 \(\boldsymbol{X}^\top \boldsymbol{X}\) 可逆时（即 \(\boldsymbol{X}\) 列满秩），
$$\hat{\boldsymbol{\beta}} = (\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top \boldsymbol{Y}.$$

**证明**：由正规方程直接得到 \(\hat{\boldsymbol{\beta}} = (\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top \boldsymbol{Y}\)。二阶导矩阵为 \(2\boldsymbol{X}^\top \boldsymbol{X}\)，在秩条件下半正定，故为全局最小。

### 2.2.2 拟合值、残差与平方和

**定义 2.2.3**  
- **帽子矩阵**：\(\boldsymbol{H} = \boldsymbol{X}(\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top\)。  
- **拟合值**：\(\hat{\boldsymbol{Y}} = \boldsymbol{X}\hat{\boldsymbol{\beta}} = \boldsymbol{H}\boldsymbol{Y}\)。  
- **残差向量**：\(\boldsymbol{e} = \boldsymbol{Y} - \hat{\boldsymbol{Y}} = (\boldsymbol{I}-\boldsymbol{H})\boldsymbol{Y}\)。

**性质 2.2.4**  
\(\boldsymbol{H}\) 为对称幂等阵（\(\boldsymbol{H}=\boldsymbol{H}^\top\)，\(\boldsymbol{H}^2=\boldsymbol{H}\)）；\(\boldsymbol{I}-\boldsymbol{H}\) 亦幂等；\(\boldsymbol{H}\boldsymbol{X}=\boldsymbol{X}\)，故 \(\boldsymbol{X}^\top \boldsymbol{e} = \boldsymbol{0}\)（残差与所有解释变量正交）。

**定义 2.2.5（平方和分解）**  
- **总平方和**：\(TSS = \sum_i (Y_i - \bar{Y})^2 = \|\boldsymbol{Y}-\bar{Y}\boldsymbol{1}\|^2\)。  
- **残差平方和**：\(RSS = \boldsymbol{e}^\top \boldsymbol{e} = \|\boldsymbol{Y}-\hat{\boldsymbol{Y}}\|^2\)。  
- **解释平方和**：\(ESS = \sum_i (\hat{Y}_i - \bar{Y})^2 = \|\hat{\boldsymbol{Y}}-\bar{Y}\boldsymbol{1}\|^2\)。  

**定理 2.2.6**  
\(TSS = ESS + RSS\)。

**证明**：记 \(\boldsymbol{Y}-\bar{Y}\boldsymbol{1} = (\hat{\boldsymbol{Y}}-\bar{Y}\boldsymbol{1}) + \boldsymbol{e}\)。两向量正交：\((\hat{\boldsymbol{Y}}-\bar{Y}\boldsymbol{1})^\top \boldsymbol{e} = (\boldsymbol{H}\boldsymbol{Y}-\bar{Y}\boldsymbol{1})^\top (\boldsymbol{I}-\boldsymbol{H})\boldsymbol{Y}\)。利用 \(\boldsymbol{H}\boldsymbol{1}=\boldsymbol{1}\)（若第一列为 1）可得 \((\boldsymbol{H}\boldsymbol{Y}-\bar{Y}\boldsymbol{1})^\top (\boldsymbol{I}-\boldsymbol{H})\boldsymbol{Y} = 0\)。故 \(\|(\boldsymbol{Y}-\bar{Y}\boldsymbol{1})\|^2 = \|\hat{\boldsymbol{Y}}-\bar{Y}\boldsymbol{1}\|^2 + \|\boldsymbol{e}\|^2\)，即 \(TSS=ESS+RSS\)。

---

## 2.3 高斯-马尔可夫定理

**定理 2.3.1（Gauss-Markov）**  
在经典假定（\(E(\boldsymbol{\varepsilon}|\boldsymbol{X})=\boldsymbol{0}\)，\(\operatorname{Var}(\boldsymbol{\varepsilon})=\sigma^2\boldsymbol{I}\)，\(\boldsymbol{X}\) 满秩）下，OLS 估计量 \(\hat{\boldsymbol{\beta}}\) 在「线性无偏」估计类中是 **BLUE**（Best Linear Unbiased Estimator），即方差最小。

**证明要点**：任意线性无偏估计可写为 \(\tilde{\boldsymbol{\beta}} = \boldsymbol{C}\boldsymbol{Y}\)，无偏要求 \(\boldsymbol{C}\boldsymbol{X}=\boldsymbol{I}\)。可证 \(\operatorname{Var}(\tilde{\boldsymbol{\beta}}) = \operatorname{Var}(\hat{\boldsymbol{\beta}}) + \sigma^2 \boldsymbol{D}\boldsymbol{D}^\top\)，其中 \(\boldsymbol{D}\) 与偏离 OLS 的矩阵有关，\(\boldsymbol{D}\boldsymbol{D}^\top\) 半正定，故 OLS 方差最小。

---

## 2.4 σ² 的估计

**定义 2.4.1（残差方差估计）**  
**回归标准误（残差方差）** 定义为
$$s^2 = \frac{RSS}{n-p-1} = \frac{\boldsymbol{e}^\top \boldsymbol{e}}{n-p-1}.$$
**回归标准误**为 \(\hat{\sigma} = \sqrt{s^2}\)。

**定理 2.4.2**  
在经典假定下，\(s^2\) 是 \(\sigma^2\) 的**无偏估计**，即 \(E(s^2|\boldsymbol{X})=\sigma^2\)。

**证明要点**：\(RSS = \boldsymbol{e}^\top \boldsymbol{e} = \boldsymbol{\varepsilon}^\top (\boldsymbol{I}-\boldsymbol{H})\boldsymbol{\varepsilon}\)（在真实 \(\boldsymbol{\beta}\) 下残差与拟合值的关系可类似写出）。利用 \(E(\boldsymbol{\varepsilon}^\top \boldsymbol{M}\boldsymbol{\varepsilon}) = \sigma^2 \operatorname{tr}(\boldsymbol{M})\)（对幂等 \(\boldsymbol{M}\)）得 \(E(RSS)=\sigma^2 \operatorname{tr}(\boldsymbol{I}-\boldsymbol{H}) = \sigma^2(n-p-1)\)，故 \(E(s^2)=\sigma^2\)。分母 \(n-p-1\) 称为**自由度**，因为 \(p+1\) 个系数估计消耗了 \(p+1\) 个自由度。

---

## 2.5 例题

**例题 2.5.1**  
设 \(n=10\)，\(p=2\)（即 2 个解释变量加常数），已知 \(RSS=24\)，求 \(s^2\) 与 \(\hat{\sigma}\)。

**解**：\(s^2 = RSS/(n-p-1) = 24/(10-3) = 24/7 \approx 3.429\)，\(\hat{\sigma} = \sqrt{24/7} \approx 1.852\)。

---

## 2.6 注意

**注意（满秩）**  
若 \(\boldsymbol{X}\) 列不满秩（存在完全共线性），\(\boldsymbol{X}^\top \boldsymbol{X}\) 不可逆，\(\hat{\boldsymbol{\beta}}\) 无唯一解；实际中需删除或合并共线变量，或使用正则化。

---

## 本章小结（第 2 章）

**核心概念**：多元线性模型 \(\boldsymbol{Y}=\boldsymbol{X}\boldsymbol{\beta}+\boldsymbol{\varepsilon}\)；正规方程 \(\boldsymbol{X}^\top \boldsymbol{X}\boldsymbol{\beta}=\boldsymbol{X}^\top \boldsymbol{Y}\)；OLS \(\hat{\boldsymbol{\beta}}=(\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top \boldsymbol{Y}\)；帽子矩阵 \(\boldsymbol{H}\)；\(TSS=ESS+RSS\)；Gauss-Markov（OLS 为 BLUE）；\(s^2=RSS/(n-p-1)\) 为 \(\sigma^2\) 的无偏估计。

**主要公式与结论**  
1. \(\hat{\boldsymbol{\beta}} = (\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top \boldsymbol{Y}\)；\(\hat{\boldsymbol{Y}}=\boldsymbol{H}\boldsymbol{Y}\)，\(\boldsymbol{e}=(\boldsymbol{I}-\boldsymbol{H})\boldsymbol{Y}\)。  
2. \(TSS=ESS+RSS\)；Gauss-Markov 下 OLS 为 BLUE。  
3. \(s^2 = RSS/(n-p-1)\)，\(E(s^2)=\sigma^2\)。

---

# 第 3 章 R²、调整 R² 与 F 检验

**动机**：需要度量回归「拟合好坏」与「整体显著性」：R² 表示解释方差比例，调整 R² 用于模型比较，F 检验用于判断所有解释变量（除常数外）是否联合显著。

---

## 3.1 决定系数 R²

### 3.1.1 定义

**定义 3.1.1（决定系数）**  
**R²**（决定系数）定义为
$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}.$$
表示由解释变量 **X** 所解释的 **Y** 的样本方差比例。易见 \(0 \le R^2 \le 1\)。

### 3.1.2 性质与注意

**性质**：R² 随解释变量个数增加而增加（即使新变量与 Y 无关），故不能单凭 R² 选模型。

**注意**：R² 高只说明样本内拟合好，不保证预测或因果；且 R² 对因变量尺度敏感（如 Y 取百分比或小数会改变 TSS）。

---

## 3.2 调整 R²

**定义 3.2.1（调整 R²）**  
**调整 R²** 为
$$\bar{R}^2 = 1 - \frac{RSS/(n-p-1)}{TSS/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-p-1}.$$
对增加无贡献的变量会使 \(\bar{R}^2\) 下降，常用于模型比较与变量选择（常与 AIC/BIC 配合）。

---

## 3.3 F 检验（整体显著性）

### 3.3.1 假设与统计量

**定义 3.3.1（整体显著性检验）**  
- **原假设** \(H_0\)：\(\beta_1 = \beta_2 = \cdots = \beta_p = 0\)（除常数外所有系数为 0）。  
- **备择** \(H_1\)：至少有一个 \(\beta_j \neq 0\)。

**定理 3.3.2**  
在 \(H_0\) 与经典假定下，统计量
$$F = \frac{ESS/p}{RSS/(n-p-1)} = \frac{ESS/p}{s^2}$$
服从 **F 分布** \(F(p,\, n-p-1)\)。

**证明要点**：在正态误差假定下，\(ESS/\sigma^2\) 与 \(RSS/\sigma^2\) 独立，分别服从自由度为 \(p\) 与 \(n-p-1\) 的 \(\chi^2\) 分布，故其比服从 F 分布。

### 3.3.2 使用方式

若 **F** 很大（或 p 值 \(<\alpha\)），拒绝 \(H_0\)，认为回归**整体显著**。单变量 **t 检验**回答「某个系数是否显著」；**F 检验**回答「所有解释变量联合是否显著」。

---

## 3.4 例题

**例题 3.4.1**  
回归得到 \(ESS=120\)，\(RSS=80\)，\(n=25\)，\(p=3\)。求 R²、\(\bar{R}^2\) 及整体显著性 F 统计量及对应 p 值（查 F 表或软件）。

**解**：\(TSS=ESS+RSS=200\)，\(R^2 = 120/200 = 0.6\)。  
\(\bar{R}^2 = 1 - (80/21)/(200/24) = 1 - (80/21)\times (24/200) \approx 0.543\)。  
\(F = (120/3)/(80/21) = 40/(80/21) = 10.5\)，自由度 \((3, 21)\)。查表或 `scipy.stats.f.cdf(10.5, 3, 21)` 得 p 值很小，拒绝 \(H_0\)，整体显著。

**代码**：
```python
from scipy import stats
ESS, RSS, n, p = 120, 80, 25, 3
F = (ESS / p) / (RSS / (n - p - 1))
pval = 1 - stats.f.cdf(F, p, n - p - 1)
print(f"F = {F:.4f}, p-value = {pval:.6f}")  # p-value 很小
```

---

## 本章小结（第 3 章）

**核心概念**：R²、调整 R²、F 检验（整体显著性）。

**主要公式与结论**  
1. \(R^2 = ESS/TSS = 1 - RSS/TSS\)。  
2. \(\bar{R}^2 = 1 - (1-R^2)(n-1)/(n-p-1)\)。  
3. \(F = (ESS/p)/(RSS/(n-p-1))\) 在 \(H_0:\beta_1=\cdots=\beta_p=0\) 下服从 \(F(p, n-p-1)\)。

# 第 4 章 单个系数的推断：t 检验与置信区间

**动机**：除整体显著性外，常需判断**单个因子**的系数是否显著（如 \(\beta_j=0\)），并给出系数的置信区间。这依赖 \(\hat{\beta}_j\) 的分布与标准误。

---

## 4.1 β̂ 的分布与标准误

**定理 4.1.1**  
在经典假定且误差服从正态分布时，\(\hat{\boldsymbol{\beta}} \sim N(\boldsymbol{\beta},\, \sigma^2 (\boldsymbol{X}^\top \boldsymbol{X})^{-1})\)。故第 \(j\) 个系数 \(\hat{\beta}_j \sim N(\beta_j,\, \sigma^2 [(\boldsymbol{X}^\top \boldsymbol{X})^{-1}]_{jj})\)。

**定义 4.1.2（系数标准误）**  
用 \(s^2\) 替代 \(\sigma^2\) 后，**\(\hat{\beta}_j\) 的标准误**为
$$SE(\hat{\beta}_j) = \sqrt{s^2 \cdot [(\boldsymbol{X}^\top \boldsymbol{X})^{-1}]_{jj}}.$$

**定理 4.1.3**  
在 \(H_0: \beta_j = 0\) 下，**t 统计量**
$$t_j = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)}$$
服从 **t 分布** \(t(n-p-1)\)。

---

## 4.2 单系数检验与置信区间

**检验** \(H_0: \beta_j=0\)：拒绝域为 \(|t_j| > t_{\alpha/2}(n-p-1)\)；或直接看 **p 值**，若 p 值 \(<\alpha\) 则拒绝。

**\(\beta_j\) 的 \(1-\alpha\) 置信区间**：
$$\hat{\beta}_j \pm t_{\alpha/2}(n-p-1) \cdot SE(\hat{\beta}_j).$$

**量化应用**：因子收益是否显著、暴露对收益的边际效应，都看 t 与置信区间；若 p 值 \(<0.05\)，常认为该因子在 5% 水平下显著。

---

## 4.3 例题

**例题 4.3.1**  
某回归得到 \(\hat{\beta}_1 = 0.8\)，\(SE(\hat{\beta}_1)=0.32\)，自由度 \(df=20\)。求 \(H_0:\beta_1=0\) 的 t 统计量、双侧 p 值及 95% 置信区间。

**解**：\(t_1 = 0.8/0.32 = 2.5\)。双侧 p 值 \(= 2\times P(T_{20}>2.5)\)。查表或 `2*(1 - stats.t.cdf(2.5, 20))` 得 p 值 \(\approx 0.021\)，拒绝 \(H_0\)。  
\(t_{0.025}(20) \approx 2.086\)，95% 置信区间为 \(0.8 \pm 2.086\times 0.32 \approx [0.13,\, 1.47]\)。

**代码**：
```python
from scipy import stats
b1, se, df = 0.8, 0.32, 20
t = b1 / se
pval = 2 * (1 - stats.t.cdf(abs(t), df))
ci = stats.t.interval(0.95, df, loc=b1, scale=se)
print(f"t = {t}, p-value = {pval:.4f}, 95% CI = {ci}")
```

---

## 本章小结（第 4 章）

**核心概念**：\(\hat{\boldsymbol{\beta}}\) 的分布；\(SE(\hat{\beta}_j)\)；t 统计量；单系数 t 检验与置信区间。

**主要公式与结论**  
1. \(\hat{\boldsymbol{\beta}} \sim N(\boldsymbol{\beta},\, \sigma^2 (\boldsymbol{X}^\top \boldsymbol{X})^{-1})\)；\(SE(\hat{\beta}_j) = \sqrt{s^2 [(\boldsymbol{X}^\top \boldsymbol{X})^{-1}]_{jj}}\)。  
2. \(t_j = \hat{\beta}_j / SE(\hat{\beta}_j)\) 在 \(H_0:\beta_j=0\) 下服从 \(t(n-p-1)\)。  
3. \(\beta_j\) 的 \(1-\alpha\) 置信区间：\(\hat{\beta}_j \pm t_{\alpha/2} \cdot SE(\hat{\beta}_j)\)。

# 第 5 章 多重共线性

**动机**：解释变量之间高度相关时，\(\boldsymbol{X}^\top \boldsymbol{X}\) 接近奇异，\(\hat{\boldsymbol{\beta}}\) 方差变大、对数据敏感，t 统计量偏小，系数难以解释；需要识别与应对多重共线性。

---

## 5.1 含义与后果

**定义 5.1.1（多重共线性）**  
**多重共线性**指解释变量之间高度（或近似）线性相关，即 \(\boldsymbol{X}\) 的列近似线性相关。

**后果**：  
- \((\boldsymbol{X}^\top \boldsymbol{X})\) 接近奇异，**条件数**大，\(\hat{\boldsymbol{\beta}}\) 对数据微小变化非常敏感。  
- 系数估计方差大，**t 统计量偏小**，容易「不显著」。  
- 单个系数难以解释（符号可能违背直觉），但**联合预测 \(\hat{Y}\) 可能仍可用**。

---

## 5.2 识别：VIF 与条件数

**定义 5.2.1（方差膨胀因子）**  
第 \(j\) 个解释变量的 **VIF** 为
$$VIF_j = \frac{1}{1 - R_j^2},$$
其中 \(R_j^2\) 为将 \(X_j\) 对其余解释变量回归得到的 R²。  
**经验法则**：VIF > 5 或 10 常视为存在严重共线性。条件数 \(\kappa(\boldsymbol{X}^\top \boldsymbol{X})\) 很大也可作为参考。

---

## 5.3 处理思路

- **增加样本**（若可能）；**删除**或**合并**高度相关变量。  
- **正则化**：岭回归、Lasso，牺牲无偏换稳定性。  
- **主成分/因子**：用 PCA 或因子模型降维后再回归。  
- **经济约束**：用先验或约束减少自由参数。  

**量化应用**：多因子暴露常相关，行业/风格中性化、正交化或 Ridge 是常见做法。

---

## 5.4 注意

**注意**：多重共线性不破坏无偏性，主要影响**方差**与**推断**（t 变小）；若只关心预测 \(\hat{Y}\)，有时可接受一定共线性。

---

## 本章小结（第 5 章）

**核心概念**：多重共线性；VIF；条件数；处理（删/并变量、正则化、PCA）。

**主要公式与结论**  
1. \(VIF_j = 1/(1-R_j^2)\)；VIF > 5 或 10 需警惕。  
2. 后果：\(\hat{\boldsymbol{\beta}}\) 方差大、t 偏小；联合预测仍可能可用。

---

# 第 6 章 异方差

**动机**：经典假定要求 \(\operatorname{Var}(\varepsilon_i|X)=\sigma^2\) 常数。截面或时间序列中常出现**异方差**（方差随 i 或随 \(X_i\) 变化），导致标准误与 t/F 推断错误，需要检验与稳健标准误。

---

## 6.1 含义与后果

**定义 6.1.1**  
- **同方差**：\(\operatorname{Var}(\varepsilon_i|\boldsymbol{X})=\sigma^2\) 对任意 \(i\)。  
- **异方差**：\(\operatorname{Var}(\varepsilon_i|\boldsymbol{X})\) 随 \(i\)（或随 \(X_i\)）变化，如 \(\operatorname{Var}(\varepsilon_i)=\sigma_i^2\)。

**后果**：OLS 仍**无偏、一致**，但**标准误不再正确**；t 与 F 的分布前提被破坏，**推断可能错误**（显著性被高估或低估）。若异方差形式已知可用 **GLS** 得 BLUE；形式未知时用**稳健标准误**（如 White、HC）。

---

## 6.2 检验与稳健标准误

**检验**：**White 检验**（残差平方对解释变量及交叉项回归）；**Breusch-Pagan**（残差平方对 X 回归）；或残差/\(|e|\) 对 \(\hat{Y}\) 或 X 的散点图（喇叭形等）。

**稳健标准误**：**White 稳健方差**在异方差存在时仍给出**一致**的标准误估计，t 与置信区间渐近正确。软件中常用 `cov_type='HC0'` 或 `'HC1'`（如 statsmodels）；报告时若怀疑异方差应报稳健标准误。

**量化应用**：截面回归中不同股票残差方差不同；可用稳健标准误，或对收益/残差标准化（如除以滚动波动率）后再回归，或采用 GARCH 等建模波动率。

---

## 本章小结（第 6 章）

**核心概念**：同方差 vs 异方差；OLS 无偏一致但标准误错误；稳健标准误（White、HC1）；GLS（形式已知时）。

**主要公式与结论**  
1. 异方差下 OLS 仍无偏一致；标准误与 t/F 推断需用稳健形式。  
2. 实践：使用 `cov_type='HC1'` 等得到渐近正确的推断。

---

# 第 7 章 横截面回归与因子收益

**动机**：量化中常用**横截面回归**估计每期因子收益，并用时间序列平均或面板回归得到平均因子收益与标准误；因子中性化可借助回归残差或约束优化。

---

## 7.1 横截面回归

**定义 7.1.1**  
**横截面回归**：每期 \(t\)，以**个股收益**为 \(Y\)，以**因子暴露**（如市值、估值、动量）为 \(X\)，做 OLS；得到的 \(\hat{\boldsymbol{\beta}}_t\) 即该期**因子收益**（因子回报）。

**时间序列平均**：对多期回归的 \(\hat{\boldsymbol{\beta}}_t\) 取平均得到「平均因子收益」；也可用**面板回归**（时间+截面）一次性估计并得到标准误。

---

## 7.2 因子中性化

若希望组合相对某因子暴露为 0，可对「收益」对「因子暴露」回归，用**残差**作为「中性化后的收益」；或直接解约束优化（组合权重与因子暴露正交）。

---

## 本章小结（第 7 章）

**核心概念**：横截面回归；因子收益；因子中性化（回归残差或约束优化）。

---

# 第 8 章 Python 实现要点（statsmodels）

- `import statsmodels.api as sm`；给 X 加常数列：`X = sm.add_constant(X)`。  
- `model = sm.OLS(y, X).fit()`；`model.summary()` 含 R²、\(\bar{R}^2\)、F、t、p 值、置信区间。  
- 稳健标准误：`model = sm.OLS(y, X).fit(cov_type='HC1')`。  
- 预测：`model.predict(X_new)`；残差：`model.resid`。

**本章小结（第 8 章）**  
横截面回归：每期 Y=个股收益，X=因子暴露，\(\hat{\boldsymbol{\beta}}\) 即因子收益；实现用 `sm.OLS(y, sm.add_constant(X)).fit()`，`cov_type='HC1'` 得稳健标准误。建议：能手推一元 OLS 的 \(b_0,b_1\)；能写出多元 \(\hat{\boldsymbol{\beta}}=(\boldsymbol{X}^\top \boldsymbol{X})^{-1}\boldsymbol{X}^\top \boldsymbol{Y}\) 并说明 R²、F、t 的含义；能解释多重共线性与异方差对推断的影响及应对思路。
