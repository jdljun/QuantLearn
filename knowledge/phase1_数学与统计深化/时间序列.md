# 第 1 章 时间序列的基本概念

**动机**：价格与收益率按时间排列，构成时间序列；平稳性决定能否用时间平均估计总体矩、能否做 ARMA 建模。本节建立基本记号与平稳性、白噪声的定义。

---

## 1.1 定义与记号

### 1.1.1 时间序列

**定义 1.1.1（时间序列）**  
**时间序列**是按时间顺序排列的一组观测 \(\{X_t\}\)，其中 \(t\) 为时间下标（离散时 \(t=0,1,2,\ldots\)）。  
量化中常见：日收益率 \(r_t\)、对数价格 \(p_t=\ln P_t\)、成交量、因子值等。

### 1.1.2 滞后与差分

**定义 1.1.2（滞后与差分）**  
- **k 阶滞后**：\(X_{t-k}\) 为 \(X_t\) 的滞后 \(k\) 期。  
- **一阶差分**：\(\nabla X_t = X_t - X_{t-1}\)。  
- **d 阶差分**：\(\nabla^d X_t\) 表示对 \(X_t\) 做 \(d\) 次一阶差分。  
对数价格的一阶差分近似收益率：\(\nabla p_t = p_t - p_{t-1} \approx r_t\)。

---

## 1.2 平稳性

### 1.2.1 弱平稳

**定义 1.2.1（弱平稳 / 二阶平稳）**  
时间序列 \(\{X_t\}\) 称为**弱平稳**，若：  
1. \(E(X_t) = \mu\) 与 \(t\) 无关；  
2. \(\operatorname{Var}(X_t) = \sigma^2\) 与 \(t\) 无关；  
3. \(\operatorname{Cov}(X_t, X_{t-k}) = \gamma_k\) 只与滞后 \(k\) 有关，与 \(t\) 无关。

**定义 1.2.2（自协方差与自相关）**  
- **自协方差**：\(\gamma_k = \operatorname{Cov}(X_t, X_{t-k})\)，\(\gamma_0 = \operatorname{Var}(X_t)\)。  
- **自相关**：\(\rho_k = \gamma_k / \gamma_0\)，满足 \(-1 \le \rho_k \le 1\)。

### 1.2.2 严平稳与弱平稳的关系

**注意（严平稳 vs 弱平稳）**  
**严平稳**要求任意有限维分布不随时间平移而改变，实践中难验证；**弱平稳**只涉及一、二阶矩，便于用样本 ACF 等推断。严平稳且二阶矩存在 ⇒ 弱平稳；反之不成立。  
**平稳的意义**：可用「时间平均」估计总体矩（大数定律）；模型参数不随时间漂移，可估计与预测。

---

## 1.3 白噪声

**定义 1.3.1（白噪声）**  
序列 \(\{\varepsilon_t\}\) 称为**白噪声**，若：  
- \(E(\varepsilon_t) = 0\)；  
- \(\operatorname{Var}(\varepsilon_t) = \sigma^2\)（常数）；  
- \(\operatorname{Cov}(\varepsilon_t, \varepsilon_s) = 0\)（\(t \neq s\)）。  
若还服从正态分布，则称为**高斯白噪声**。白噪声是「无记忆」的基准；许多模型假设扰动项为白噪声。

---

## 1.4 例题

**例题 1.4.1**  
设 \(\{X_t\}\) 为白噪声，\(\operatorname{Var}(X_t)=\sigma^2\)。求 \(\nabla X_t = X_t - X_{t-1}\) 的均值、方差及 \(\operatorname{Cov}(\nabla X_t, \nabla X_{t-1})\)。

**解**：\(E(\nabla X_t)=0\)。\(\operatorname{Var}(\nabla X_t) = \operatorname{Var}(X_t) + \operatorname{Var}(X_{t-1}) = 2\sigma^2\)（因 \(X_t\) 与 \(X_{t-1}\) 不相关）。  
\(\operatorname{Cov}(\nabla X_t, \nabla X_{t-1}) = \operatorname{Cov}(X_t - X_{t-1}, X_{t-1} - X_{t-2}) = -\operatorname{Var}(X_{t-1}) = -\sigma^2\)。

**代码**：
```python
import numpy as np
np.random.seed(42)
n = 1000
eps = np.random.randn(n) * 2  # 白噪声, Var=4
diff_eps = np.diff(eps)
print("E(diff) ≈ 0:", np.mean(diff_eps))
print("Var(diff) ≈ 2*4 = 8:", np.var(diff_eps))
# 滞后 1 自协方差
cov_lag1 = np.cov(diff_eps[1:], diff_eps[:-1])[0, 1]
print("Cov(diff_t, diff_{t-1}) ≈ -4:", cov_lag1)
```

---

## 本章小结（第 1 章）

**核心概念**：时间序列、滞后、差分 \(\nabla X_t\)；弱平稳（均值、方差、自协方差与 \(t\) 无关）；自协方差 \(\gamma_k\)、自相关 \(\rho_k\)；白噪声。

**主要公式与结论**  
1. 弱平稳：\(E(X_t)=\mu\)，\(\operatorname{Var}(X_t)=\sigma^2\)，\(\operatorname{Cov}(X_t, X_{t-k})=\gamma_k\)；\(\rho_k = \gamma_k/\gamma_0\)。  
2. 白噪声：\(E(\varepsilon_t)=0\)，\(\operatorname{Var}(\varepsilon_t)=\sigma^2\)，\(\operatorname{Cov}(\varepsilon_t,\varepsilon_s)=0\)（\(t\neq s\)）。  
3. 平稳的意义：可用时间平均估计总体矩；模型参数不随时间漂移。

# 第 2 章 自相关与偏自相关

**动机**：需要从样本中估计序列的「记忆」结构，以识别模型类型与阶数。自相关函数（ACF）与偏自相关函数（PACF）是主要工具；AR(p) 与 MA(q) 在 ACF/PACF 上呈现「截尾」或「拖尾」的典型模式。

---

## 2.1 自相关函数 ACF

### 2.1.1 样本自相关

**定义 2.1.1（样本自协方差与样本自相关）**  
设观测为 \(X_1,\ldots,X_n\)。**样本自协方差**（滞后 \(k\)）为
$$\hat{\gamma}_k = \frac{1}{n} \sum_{t=k+1}^{n} (X_t - \bar{X})(X_{t-k} - \bar{X}),$$
其中 \(\bar{X}\) 为样本均值。**样本自相关**为
$$\hat{\rho}_k = \frac{\hat{\gamma}_k}{\hat{\gamma}_0}.$$

**ACF 图**：横轴为滞后 \(k\)，纵轴为 \(\hat{\rho}_k\)。用途：看序列是否有短期/长期记忆；辅助判断 AR、MA 的阶数。

---

## 2.2 偏自相关函数 PACF

### 2.2.1 定义

**定义 2.2.1（偏自相关）**  
**偏自相关** \(\phi_{kk}\) 表示在给定 \(X_{t-1},\ldots,X_{t-k+1}\) 的条件下，\(X_t\) 与 \(X_{t-k}\) 的「净」相关（去掉中间滞后的影响）。可由 Yule-Walker 方程递推得到；样本形式为将 \(X_t\) 对 \(X_{t-1},\ldots,X_{t-k}\) 回归时，最后一阶系数估计 \(\hat{\phi}_{kk}\)。

**PACF 图**：横轴 \(k\)，纵轴 \(\hat{\phi}_{kk}\)。

### 2.2.2 截尾与拖尾

**定义 2.2.2（截尾与拖尾）**  
- **截尾**：某阶之后理论值严格为 0；样本中表现为明显落入置信带内。  
- **拖尾**：逐渐衰减但不严格为 0（如指数或正弦式衰减）。

**定理 2.2.3（AR/MA 的 ACF 与 PACF 特征）**  
- **AR(p)**：ACF 拖尾，PACF 在 \(p\) 阶截尾。  
- **MA(q)**：ACF 在 \(q\) 阶截尾，PACF 拖尾。  
- **ARMA(p,q)**：ACF、PACF 均拖尾。

据此可初步识别模型类型与阶数。

---

## 2.3 例题

**例题 2.3.1**  
对某收益率序列（\(n=252\)）计算前 5 阶样本 ACF 得到 \(\hat{\rho}_1=0.35\)，\(\hat{\rho}_2=0.18\)，\(\hat{\rho}_3=0.08\)，\(\hat{\rho}_4=0.02\)，\(\hat{\rho}_5=-0.01\)。试判断更符合 AR 还是 MA，并粗略估计阶数。

**解**：ACF 从 0.35 逐渐衰减至接近 0，呈**拖尾**。若 PACF 在 1 阶后截尾，则更符合 **AR(1)**；若 PACF 也拖尾，则可能是 ARMA。需要同时看 PACF 图才能定阶；本例仅凭 ACF 拖尾可初步猜测为 AR 或 ARMA，阶数需结合 PACF 或信息准则。

**代码**：
```python
from statsmodels.graphics.tsaplots import plot_acf, plot_pacf
import numpy as np
# 假设已有序列 x
# plot_acf(x, lags=20); plot_pacf(x, lags=20)
# 或数值：from statsmodels.tsa.stattools import acf, pacf
# acf(x, nlags=5), pacf(x, nlags=5)
```

## 本章小结（第 2 章）

**核心概念**：样本 ACF \(\hat{\rho}_k\)、样本 PACF \(\hat{\phi}_{kk}\)；截尾与拖尾；AR(p) 的 PACF 在 \(p\) 阶截尾、ACF 拖尾；MA(q) 的 ACF 在 \(q\) 阶截尾、PACF 拖尾；ARMA 两者都拖尾。

**主要公式与结论**  
1. \(\hat{\rho}_k = \hat{\gamma}_k / \hat{\gamma}_0\)；\(\hat{\gamma}_k\) 为样本自协方差。  
2. AR(p)：PACF 在 \(p\) 截尾；MA(q)：ACF 在 \(q\) 截尾；ARMA 均拖尾。

# 第 3 章 AR、MA 与 ARMA 模型

**动机**：用有限参数模型刻画平稳序列的依赖结构；AR 用过去观测的线性组合，MA 用过去扰动的线性组合，ARMA 二者结合。平稳性与可逆性条件决定模型是否可用。

---

## 3.1 自回归 AR(p)

### 3.1.1 定义与滞后算子

**定义 3.1.1（AR(p) 模型）**  
**AR(p)** 模型为
$$X_t = c + \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t,$$
其中 \(\{\varepsilon_t\}\) 为白噪声。**滞后算子** \(L\) 定义为 \(L X_t = X_{t-1}\)，则 \(L^k X_t = X_{t-k}\)。**滞后多项式**为 \(\phi(L) = 1 - \phi_1 L - \cdots - \phi_p L^p\)，模型可写为 \(\phi(L) X_t = c + \varepsilon_t\)。

### 3.1.2 平稳条件

**定理 3.1.2（AR(p) 平稳条件）**  
AR(p) 存在平稳解当且仅当 \(\phi(L)=0\) 的根**全在单位圆外**（复平面）。  
**AR(1)**：\(X_t = \phi X_{t-1} + \varepsilon_t\) 平稳 ⟺ \(|\phi| < 1\)。

**均值**：若 \(c=0\) 且平稳，\(E(X_t)=0\)；若 \(c \neq 0\)，在 \(\phi_1+\cdots+\phi_p \neq 1\) 时 \(E(X_t) = c/(1-\phi_1-\cdots-\phi_p)\)。

---

## 3.2 移动平均 MA(q)

**定义 3.2.1（MA(q) 模型）**  
**MA(q)** 模型为
$$X_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \cdots + \theta_q \varepsilon_{t-q},$$
其中 \(\{\varepsilon_t\}\) 为白噪声。MA 模型总是**平稳**的（白噪声的有限线性组合）。**可逆条件**：\(\theta(L)=1+\theta_1 L+\cdots+\theta_q L^q=0\) 的根在单位圆外，此时可用 AR(\(\infty\)) 表示。

---

## 3.3 ARMA(p,q)

**定义 3.3.1（ARMA(p,q)）**  
**ARMA(p,q)** 为 \(\phi(L) X_t = c + \theta(L) \varepsilon_t\)，结合 AR 与 MA。平稳性由 \(\phi(L)\) 决定，可逆性由 \(\theta(L)\) 决定。实际中常用低阶 ARMA(1,1)、AR(2) 等；阶数可由 ACF/PACF 或信息准则（AIC/BIC）选择。

---

## 3.4 估计与预测

**估计**：通常用最大似然或条件最小二乘；软件（如 statsmodels）给出系数估计、标准误与 AIC/BIC。  
**预测**：最小均方误差意义下的点预测为 \(E(X_{t+h} | X_t, X_{t-1}, \ldots)\)；AR 模型可递推得到 \(h\) 步预测；随 \(h\) 增大，预测趋于无条件均值。

---

## 3.5 例题

**例题 3.5.1**  
对 AR(1) 模型 \(X_t = 0.7 X_{t-1} + \varepsilon_t\)，\(\varepsilon_t \sim \text{i.i.d.}(0, 1)\)。求 \(E(X_t)\)、\(\operatorname{Var}(X_t)\) 及 \(\rho_1\)。

**解**：平稳时 \(E(X_t)=0\)。由 \(X_t = 0.7 X_{t-1} + \varepsilon_t\) 两边取方差得 \(\gamma_0 = 0.49 \gamma_0 + 1\)，故 \(\gamma_0 = 1/(1-0.49) = 1/0.51 \approx 1.96\)。  
\(\gamma_1 = \operatorname{Cov}(X_t, X_{t-1}) = \operatorname{Cov}(0.7 X_{t-1}+\varepsilon_t, X_{t-1}) = 0.7 \gamma_0\)，故 \(\rho_1 = \gamma_1/\gamma_0 = 0.7\)。

**代码**：
```python
import numpy as np
from statsmodels.tsa.arima.model import ARIMA
np.random.seed(42)
n = 500
eps = np.random.randn(n)
x = np.zeros(n)
x[0] = eps[0]
for t in range(1, n):
    x[t] = 0.7 * x[t-1] + eps[t]
# 估计
fit = ARIMA(x, order=(1, 0, 0)).fit()
print(fit.summary())  # 查看 phi 估计、AIC 等
print("理论 Var ≈ 1.96, 样本 Var =", np.var(x))
```

## 本章小结（第 3 章）

**核心概念**：AR(p)、MA(q)、ARMA(p,q)；滞后算子与平稳/可逆条件；AR(1) 平稳 ⟺ \(|\phi|<1\)；估计与预测。

**主要公式与结论**  
1. AR(p)：\(\phi(L)X_t = c + \varepsilon_t\)；平稳 ⟺ \(\phi(L)=0\) 的根在单位圆外。  
2. MA(q)：总是平稳；可逆条件为 \(\theta(L)=0\) 的根在单位圆外。  
3. ARMA 阶数由 ACF/PACF 或 AIC/BIC 选择。

# 第 4 章 单位根检验

**动机**：若 AR 多项式有**单位根**（即 \(\phi(1)=0\)），序列**非平稳**，时间平均不再估计总体均值，ARMA 理论不适用。许多**价格**序列接近随机游走；**收益率**往往更接近平稳。需要检验序列是否含单位根（如 ADF）。

---

## 4.1 单位根与非平稳

**定义 4.1.1（单位根）**  
若 AR 多项式 \(\phi(L)\) 有**单位根**（即 \(\phi(1)=0\)），则序列**非平稳**。  
**AR(1)** 中 \(\phi=1\) 即**随机游走**：\(X_t = X_{t-1} + \varepsilon_t\)，此时 \(\operatorname{Var}(X_t)\) 随 \(t\) 增长，样本均值不再估计总体均值。

---

## 4.2 ADF 检验

**定义 4.2.1（ADF 检验）**  
**原假设** \(H_0\)：序列有单位根（非平稳）。**备择** \(H_1\)：序列平稳（无单位根）。  
做法：对模型
$$\nabla X_t = \alpha + \beta X_{t-1} + \sum_{j=1}^{k} \gamma_j \nabla X_{t-j} + \text{误差}$$
做回归（可带常数、趋势）；检验 **\(\beta = 0\)**（即存在单位根）。ADF 统计量有非标准分布，需查专用临界值或由软件给出 p 值。  
**若 p 值小**（如 <0.05），拒绝 \(H_0\)，认为序列**平稳**；若 p 值大，不拒绝单位根，认为**非平稳**。

---

## 4.3 其他检验（简述）

**KPSS**：原假设为平稳，与 ADF 互补。**PP（Phillips-Perron）**：与 ADF 类似，对异方差更稳健。实践中常同时看 ADF 与 KPSS，两者一致时结论更可靠。

---

## 4.4 例题

**例题 4.4.1**  
对某日收益率序列（252 个观测）做 ADF 检验，得到 ADF 统计量 \(-3.2\)，5% 临界值约 \(-2.86\)，p 值约 0.02。请解释结论。

**解**：ADF 统计量 \(-3.2 < -2.86\)（更负），落在拒绝域；p 值 0.02 < 0.05，故拒绝「有单位根」，认为该收益率序列**平稳**，可进行 ARMA 建模或回归。

**代码**：
```python

# print('ADF stat:', result[0], 'p-value:', result[1], 'lags:', result[2])

```

## 本章小结（第 4 章）

**核心概念**：单位根；随机游走；ADF 检验（\(H_0\)：有单位根）；p 值小则拒绝 \(H_0\)，认为平稳。

**主要公式与结论**  
1. 单位根：\(\phi(1)=0\) 时非平稳；AR(1) 中 \(\phi=1\) 为随机游走。  
2. ADF：对 \(\nabla X_t = \alpha + \beta X_{t-1} + \cdots\) 检验 \(\beta=0\)；p 值 < 0.05 则拒绝 \(H_0\)，认为平稳。

# 第 5 章 协整

**动机**：两个**非平稳**序列的**线性组合**可能**平稳**，称为协整；表示两序列长期存在「均衡关系」，短期可偏离但会向均衡回归，是**配对交易**、统计套利的基础。

---

## 5.1 概念

**定义 5.1.1（协整）**  
若两个（或一组）**非平稳**序列的**某种线性组合**是**平稳**的，则称它们**协整**。  
例如：价格 \(P_t^A\)、\(P_t^B\) 都是 I(1)（一阶单整，即一阶差分后平稳），但 \(P_t^A - \beta P_t^B\) 平稳，则 \((1, -\beta)\) 为**协整向量**。经济含义：两序列长期存在均衡关系，短期偏离会回归。

---

## 5.2 检验与估计

**Engle-Granger 两步法**：  
1. 对两序列做回归（如 \(P_t^A = \alpha + \beta P_t^B + u_t\)）；  
2. 对残差 \(\hat{u}_t\) 做 ADF 检验；若残差平稳，则认为两序列协整。  
**Johansen 检验**：多变量情形下找协整秩与协整向量，更严谨。  
**估计**：协整系数 \(\beta\) 可用 OLS 估计；协整时 OLS 是**超一致**的（收敛速度比通常的 \(\sqrt{n}\) 更快）。

---

## 5.3 与配对交易的关系

若价差（或对数价差）平稳，可设定「价差偏离均值过远时建仓、回归时平仓」的规则；需结合交易成本、样本外检验与风控。

---

## 5.4 例题

**例题 5.4.1**

**解**：p 值 0.03 < 0.05，拒绝「无协整」的假设，认为两序列**存在协整关系**，可考虑配对交易或统计套利。  
代码示例：`from statsmodels.tsa.stattools import coint; coint(log_A, log_B)` 返回统计量、p 值等。

## 本章小结（第 5 章）

**核心概念**：协整；Engle-Granger 两步法（回归后对残差做 ADF）；协整时 OLS 超一致；配对交易。

**主要公式与结论**  
1. 协整：两非平稳序列的某线性组合平稳。  
2. Engle-Granger：先 OLS 回归，再对残差做 ADF；残差平稳 ⇒ 协整。  
3. 量化中：配对交易、统计套利的基础。

# 第 6 章 Python 实现要点（statsmodels）

- **ACF/PACF 图**：`from statsmodels.graphics.tsaplots import plot_acf, plot_pacf`；`plot_acf(x, lags=...)`。  
- **ADF**：`from statsmodels.tsa.stattools import adfuller`；`adfuller(x)` 返回统计量、p 值、临界值等。  
- **ARMA 拟合**：`from statsmodels.tsa.arima.model import ARIMA`；`ARIMA(x, order=(p,0,q)).fit()`；`.forecast(steps=h)` 得预测。  
- **协整**：`from statsmodels.tsa.stattools import coint`；`coint(y1, y2)` 得 Engle-Granger 的 p 值等。

---

## 与量化研究的衔接

- **收益建模**：日收益常用 ARMA 或 GARCH；平稳性由 ADF 确认。  
- **预测**：短期收益/波动率预测可基于 ARMA/ARMA-GARCH。  
- **配对/统计套利**：协整检验筛选标的对；价差序列的均值回归策略。  
- **因子与宏观序列**：部分宏观或情绪指标为时间序列，可用 ARIMA 做预测或作为回归中的解释变量。

**本章小结（第 6 章）**  
ACF/PACF：`plot_acf`, `plot_pacf`；ADF：`adfuller(x)`；ARMA：`ARIMA(x, order=(p,0,q)).fit()`，`.forecast(steps=h)`；协整：`coint(y1, y2)`。建议：能解释平稳与单位根的含义；能根据 ACF/PACF 大致判断 AR/MA 阶数；会用 ADF 判断序列是否平稳、用协整检验判断两序列是否协整；会用 statsmodels 完成 ADF、ARMA 拟合与简单预测。
