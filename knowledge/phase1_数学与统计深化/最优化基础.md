# 第 1 章 无约束优化：梯度与一阶条件

**动机**：组合权重、因子权重、损失函数最小化都需求「使目标函数最小」的点；梯度给出上升最快方向，一阶条件 \(\nabla f(\boldsymbol{x}^*)=\boldsymbol{0}\) 是局部极小的必要条件，梯度下降是迭代求解的基本方式。

---

## 1.1 梯度

### 1.1.1 定义

**定义 1.1.1（梯度）**  
设 \(f(\boldsymbol{x})\) 为 \(\mathbb{R}^n\) 上的标量函数，\(\boldsymbol{x}\) 可微。**梯度** \(\nabla f(\boldsymbol{x})\) 为 \(n\) 维向量，第 \(i\) 分量为 \(\frac{\partial f}{\partial x_i}\)：
$$\nabla f(\boldsymbol{x}) = \left( \frac{\partial f}{\partial x_1}, \ldots, \frac{\partial f}{\partial x_n} \right)^\top.$$
梯度方向是函数**增加最快**的方向；**\(-\nabla f\)** 是**下降最快**的方向。

### 1.1.2 一阶必要条件

**定理 1.1.2（一阶必要条件）**  
若 \(\boldsymbol{x}^*\) 为 \(f\) 的**局部极小点**且 \(f\) 在该点可微，则
$$\nabla f(\boldsymbol{x}^*) = \boldsymbol{0}.$$
满足 \(\nabla f(\boldsymbol{x})=\boldsymbol{0}\) 的点称为**驻点**（或临界点）。

**证明**：若 \(\nabla f(\boldsymbol{x}^*) \neq \boldsymbol{0}\)，取方向 \(\boldsymbol{d} = -\nabla f(\boldsymbol{x}^*)\)，则沿 \(\boldsymbol{d}\) 方向 \(f\) 在 \(\boldsymbol{x}^*\) 处方向导数为负，即存在 \(t>0\) 小使得 \(f(\boldsymbol{x}^* + t\boldsymbol{d}) < f(\boldsymbol{x}^*)\)，与 \(\boldsymbol{x}^*\) 为局部极小矛盾。故 \(\nabla f(\boldsymbol{x}^*)=\boldsymbol{0}\)。

**注意**：驻点不一定是极小（可能是极大或鞍点），需二阶条件（Hessian 正定）区分。

---

## 1.2 梯度下降（概念）

**定义 1.2.1（梯度下降）**  
**梯度下降**迭代格式为
$$\boldsymbol{x}^{k+1} = \boldsymbol{x}^k - \alpha_k \nabla f(\boldsymbol{x}^k),$$
其中 \(\alpha_k\) 为步长（学习率）。在适当条件下，序列收敛到驻点；若 \(f\) 凸，则收敛到全局最小。量化中：回归、神经网络等的拟合常用梯度下降或其变体（SGD、Adam 等）。

---

## 1.3 常见函数的梯度

**定理 1.3.1（常用梯度公式）**  
- 线性：\(f(\boldsymbol{x})=\boldsymbol{c}^\top \boldsymbol{x}\)，\(\nabla f = \boldsymbol{c}\)。  
- 二次型：\(f(\boldsymbol{x})=\boldsymbol{x}^\top \boldsymbol{A}\boldsymbol{x}\)（\(\boldsymbol{A}\) 对称），\(\nabla f = 2\boldsymbol{A}\boldsymbol{x}\)。  
- 二次型+线性：\(f(\boldsymbol{x})=\frac{1}{2}\boldsymbol{x}^\top \boldsymbol{A}\boldsymbol{x} - \boldsymbol{b}^\top \boldsymbol{x}\)，\(\nabla f = \boldsymbol{A}\boldsymbol{x} - \boldsymbol{b}\)。  
- 组合方差：\(f(\boldsymbol{w})=\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\)，\(\nabla f = 2\boldsymbol{\Sigma}\boldsymbol{w}\)。

**推导（二次型）**：\(f = \sum_{i,j} A_{ij} x_i x_j\)，\(\frac{\partial f}{\partial x_k} = 2\sum_j A_{kj} x_j = 2(\boldsymbol{A}\boldsymbol{x})_k\)，故 \(\nabla f = 2\boldsymbol{A}\boldsymbol{x}\)。

---

## 1.4 例题

**例题 1.4.1**  
设 \(f(x_1, x_2) = x_1^2 + 2x_2^2 - x_1 x_2 + x_1 - 3x_2\)。求驻点并判断是否为极小。

**解**：\(\frac{\partial f}{\partial x_1} = 2x_1 - x_2 + 1 = 0\)，\(\frac{\partial f}{\partial x_2} = 4x_2 - x_1 - 3 = 0\)。解得 \(x_1 = 1\)，\(x_2 = 1\)。Hessian \(\boldsymbol{H} = \begin{pmatrix} 2 & -1 \\ -1 & 4 \end{pmatrix}\)，顺序主子式 2>0、8-1=7>0，故正定，驻点为严格局部极小。

**代码**：
```python
import numpy as np
from scipy.optimize import minimize
def f(x):
    return x[0]**2 + 2*x[1]**2 - x[0]*x[1] + x[0] - 3*x[1]
res = minimize(f, [0, 0], method='BFGS')
print("驻点:", res.x)   # 应接近 [1, 1]
print("f(驻点):", res.fun)
```

---

## 本章小结（第 1 章）

**核心概念**：梯度 \(\nabla f\)（增加最快方向）；一阶必要条件：局部极小 ⇒ \(\nabla f(\boldsymbol{x}^*)=\boldsymbol{0}\)（驻点）；梯度下降 \(x^{k+1}=x^k - \alpha_k \nabla f(x^k)\)。

**主要公式与结论**  
1. \(\nabla (\boldsymbol{x}^\top \boldsymbol{A}\boldsymbol{x}) = 2\boldsymbol{A}\boldsymbol{x}\)（\(\boldsymbol{A}\) 对称）；\(\nabla (\frac{1}{2}\boldsymbol{x}^\top \boldsymbol{A}\boldsymbol{x} - \boldsymbol{b}^\top \boldsymbol{x}) = \boldsymbol{A}\boldsymbol{x} - \boldsymbol{b}\)。  
2. 组合方差 \(\nabla (\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}) = 2\boldsymbol{\Sigma}\boldsymbol{w}\)。

# 第 2 章 凸集与凸函数

**动机**：凸优化中「局部极小即全局极小」，且很多算法有收敛保证。组合方差、线性约束下的二次目标都是凸问题；需掌握凸集、凸函数的定义与判别（尤其二阶条件）。

---

## 2.1 凸集

**定义 2.1.1（凸集）**  
集合 \(C \subseteq \mathbb{R}^n\) 称为**凸集**，若对任意 \(\boldsymbol{x}, \boldsymbol{y} \in C\) 与 \(\lambda \in [0, 1]\)，有
$$\lambda \boldsymbol{x} + (1-\lambda)\boldsymbol{y} \in C.$$
即连接两点线段仍在 \(C\) 内。例：超平面、半空间、多面体、椭球、范数球都是凸集；凸集的交仍为凸集。

---

## 2.2 凸函数

**定义 2.2.1（凸函数）**  
设 \(f\) 的定义域为凸集 \(C\)。\(f\) 为**凸函数**，若对任意 \(\boldsymbol{x}, \boldsymbol{y} \in C\) 与 \(\lambda \in [0, 1]\)，有
$$f(\lambda \boldsymbol{x} + (1-\lambda)\boldsymbol{y}) \le \lambda f(\boldsymbol{x}) + (1-\lambda) f(\boldsymbol{y}).$$
若不等号在 \(\boldsymbol{x}\neq \boldsymbol{y}\) 且 \(\lambda \in (0,1)\) 时严格成立，则称**严格凸**。**凹函数**：\(-f\) 为凸；最小化凸函数等价于最大化凹函数。

---

## 2.3 凸性与二阶条件

**定理 2.3.1（可微凸函数的二阶刻画）**  
若 \(f\) 二阶连续可微，则：  
- \(f\) 凸 ⟺ **Hessian 矩阵 \(\nabla^2 f(\boldsymbol{x})\) 半正定**（对定义域内任意 \(\boldsymbol{x}\)）。  
- 若 \(\nabla^2 f(\boldsymbol{x})\) 正定，则 \(f\) 严格凸。  
一元情形：\(f\) 凸 ⟺ \(f'' \ge 0\)。

**定理 2.3.2（凸函数的全局极小）**  
**凸函数的局部极小即全局极小**；若 \(f\) 严格凸且存在极小点，则唯一。

**证明要点**：若 \(\boldsymbol{x}^*\) 为局部极小，\(\boldsymbol{y}\) 为任意点，则对充分小的 \(\lambda \in (0,1)\)，\(f(\boldsymbol{x}^* + \lambda(\boldsymbol{y}-\boldsymbol{x}^*)) \ge f(\boldsymbol{x}^*)\)；由凸性 \(f(\boldsymbol{x}^* + \lambda(\boldsymbol{y}-\boldsymbol{x}^*)) \le \lambda f(\boldsymbol{y}) + (1-\lambda)f(\boldsymbol{x}^*)\)，联立得 \(f(\boldsymbol{y}) \ge f(\boldsymbol{x}^*)\)，故 \(\boldsymbol{x}^*\) 为全局极小。

组合方差 \(\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\)（\(\boldsymbol{\Sigma}\) 正定）是 \(\boldsymbol{w}\) 的凸函数；负收益 \(-\boldsymbol{\mu}^\top \boldsymbol{w}\) 是线性（凸且凹）；因此「在给定收益下最小化方差」是凸规划。

---

## 2.4 常见凸函数

线性、二次型 \(\boldsymbol{x}^\top \boldsymbol{A}\boldsymbol{x}\)（\(\boldsymbol{A}\) 半正定）、范数、负熵、log-sum-exp；凸函数的非负组合、仿射复合仍为凸。

---

## 本章小结（第 2 章）

**核心概念**：凸集（\(\lambda \boldsymbol{x}+(1-\lambda)\boldsymbol{y} \in C\)）；凸函数（\(f(\lambda \boldsymbol{x}+(1-\lambda)\boldsymbol{y}) \le \lambda f(\boldsymbol{x})+(1-\lambda)f(\boldsymbol{y})\)）；可微时 \(f\) 凸 ⟺ \(\nabla^2 f\) 半正定；凸则局部极小=全局极小。

**主要公式与结论**  
1. \(\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\)（\(\boldsymbol{\Sigma}\) 正定）为凸。  
2. 凸函数的局部极小即全局极小；严格凸时若存在极小则唯一。

# 第 3 章 无约束二次规划

**动机**：许多问题可化为二次目标、线性约束（或无约束）；OLS 正规方程即无约束二次型最小化；组合最小方差在无约束时退化为 \(\boldsymbol{w}=\boldsymbol{0}\)，故实际需加约束（下一章）。

---

## 3.1 形式与一阶条件

**定义 3.1.1（无约束二次规划）**  
考虑
$$\min_{\boldsymbol{x}} \quad \frac{1}{2} \boldsymbol{x}^\top \boldsymbol{Q}\boldsymbol{x} - \boldsymbol{c}^\top \boldsymbol{x},$$
其中 \(\boldsymbol{Q}\) 为对称矩阵。一阶条件：\(\nabla f = \boldsymbol{Q}\boldsymbol{x} - \boldsymbol{c} = \boldsymbol{0}\)，即
$$\boldsymbol{Q}\boldsymbol{x} = \boldsymbol{c}.$$

**定理 3.1.2**  
若 \(\boldsymbol{Q}\) **正定**，则 \(\boldsymbol{x}^* = \boldsymbol{Q}^{-1}\boldsymbol{c}\) 为**唯一全局最小**。若 \(\boldsymbol{Q}\) 半正定且 \(\boldsymbol{c}\) 在 \(\boldsymbol{Q}\) 的列空间中，则解集为非空凸集；若 \(\boldsymbol{Q}\) 不定，则可能无下界或为鞍点。

**证明**：正定时 Hessian 为 \(\boldsymbol{Q}\) 正定，故驻点即全局最小；解 \(\boldsymbol{Q}\boldsymbol{x}=\boldsymbol{c}\) 唯一。

**与 OLS 的联系**：回归最小化 \(\|\boldsymbol{Y}-\boldsymbol{X}\boldsymbol{\beta}\|^2 = \boldsymbol{\beta}^\top \boldsymbol{X}^\top \boldsymbol{X}\boldsymbol{\beta} - 2\boldsymbol{Y}^\top \boldsymbol{X}\boldsymbol{\beta} + \text{常数}\)，即 \(\boldsymbol{Q}=\boldsymbol{X}^\top \boldsymbol{X}\)，\(\boldsymbol{c}=\boldsymbol{X}^\top \boldsymbol{Y}\)，正规方程 \((\boldsymbol{X}^\top \boldsymbol{X})\boldsymbol{\beta} = \boldsymbol{X}^\top \boldsymbol{Y}\) 即此类。

---

## 3.2 均值-方差（无约束形式）

设收益向量 \(\boldsymbol{\mu}\)，协方差 \(\boldsymbol{\Sigma}\)；组合权重 \(\boldsymbol{w}\)，收益 \(\boldsymbol{\mu}^\top \boldsymbol{w}\)，方差 \(\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\)。**最小化方差** \(\min \boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\) 在无约束时解为 \(\boldsymbol{w}=\boldsymbol{0}\)（无意义）；故需加约束（见第 4 章）。

---

## 本章小结（第 3 章）

**核心概念**：无约束二次规划 \(\min \frac{1}{2}\boldsymbol{x}^\top \boldsymbol{Q}\boldsymbol{x} - \boldsymbol{c}^\top \boldsymbol{x}\)；一阶条件 \(\boldsymbol{Q}\boldsymbol{x}=\boldsymbol{c}\)；\(\boldsymbol{Q}\) 正定时 \(\boldsymbol{x}^*=\boldsymbol{Q}^{-1}\boldsymbol{c}\) 唯一全局最小；OLS 正规方程即此类。

---

# 第 4 章 等式约束与拉格朗日

**动机**：组合优化常带「满仓」「目标收益」等等式约束；拉格朗日乘子法给出最优性条件与求解思路；均值-方差两约束下的解析形式是典型应用。

---

## 4.1 拉格朗日函数与一阶条件

**定义 4.1.1（等式约束问题）**  
考虑
$$\min f(\boldsymbol{x}) \quad \text{s.t.} \quad h_j(\boldsymbol{x}) = 0,\; j=1,\ldots,m.$$

**定义 4.1.2（拉格朗日函数）**  
**拉格朗日函数**为
$$L(\boldsymbol{x}, \boldsymbol{\lambda}) = f(\boldsymbol{x}) + \sum_{j=1}^{m} \lambda_j h_j(\boldsymbol{x}),$$
其中 \(\lambda_j\) 为拉格朗日乘子。

**定理 4.1.3（一阶必要条件）**  
在约束规格（如约束梯度线性无关）下，若 \(\boldsymbol{x}^*\) 为局部最优，则存在 \(\boldsymbol{\lambda}^*\) 使得  
- \(\nabla_{\boldsymbol{x}} L(\boldsymbol{x}^*, \boldsymbol{\lambda}^*) = \boldsymbol{0}\)（即 \(\nabla f(\boldsymbol{x}^*) + \sum_j \lambda_j^* \nabla h_j(\boldsymbol{x}^*) = \boldsymbol{0}\)）；  
- \(h_j(\boldsymbol{x}^*) = 0\) 对所有 \(j\)。  
解上述方程组得候选点，再结合二阶条件或问题的凸性判断是否为最小。

---

## 4.2 均值-方差：目标收益与满仓约束

**问题**：
$$\min \frac{1}{2} \boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w} \quad \text{s.t.} \quad \boldsymbol{\mu}^\top \boldsymbol{w} = r_{target},\; \boldsymbol{1}^\top \boldsymbol{w} = 1.$$

**拉格朗日**：\(L = \frac{1}{2}\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w} + \lambda_1(\boldsymbol{\mu}^\top \boldsymbol{w} - r_{target}) + \lambda_2(\boldsymbol{1}^\top \boldsymbol{w} - 1)\)。

**一阶条件**：\(\nabla_{\boldsymbol{w}} L = \boldsymbol{\Sigma}\boldsymbol{w} + \lambda_1 \boldsymbol{\mu} + \lambda_2 \boldsymbol{1} = \boldsymbol{0}\)，故
$$\boldsymbol{w} = -\boldsymbol{\Sigma}^{-1}(\lambda_1 \boldsymbol{\mu} + \lambda_2 \boldsymbol{1}).$$
代入两个等式 \(\boldsymbol{\mu}^\top \boldsymbol{w} = r_{target}\)、\(\boldsymbol{1}^\top \boldsymbol{w} = 1\) 得到关于 \(\lambda_1, \lambda_2\) 的线性方程组，解出后再得 \(\boldsymbol{w}^*\)。

**解析形式**：记 \(\boldsymbol{M} = [\boldsymbol{\mu}, \boldsymbol{1}]\)，\(\boldsymbol{b} = (r_{target}, 1)^\top\)，可写成 \(\boldsymbol{w}^* = \boldsymbol{\Sigma}^{-1}\boldsymbol{M}(\boldsymbol{M}^\top \boldsymbol{\Sigma}^{-1}\boldsymbol{M})^{-1}\boldsymbol{b}\)（两约束情形）。

---

## 4.3 不等式约束与 KKT（简要）

**问题**：\(\min f(\boldsymbol{x})\) s.t. \(g_i(\boldsymbol{x}) \le 0\)。  
**KKT 条件**：在凸规划与约束规格下，\(\boldsymbol{x}^*\) 最优 ⟺ 存在 \(\lambda_i \ge 0\) 使得：  
- \(\nabla f(\boldsymbol{x}^*) + \sum_i \lambda_i \nabla g_i(\boldsymbol{x}^*) = \boldsymbol{0}\)；  
- \(g_i(\boldsymbol{x}^*) \le 0\)；\(\lambda_i g_i(\boldsymbol{x}^*) = 0\)（互补松弛）；\(\lambda_i \ge 0\)。  
做空限制 \(\boldsymbol{w} \ge \boldsymbol{0}\)、风险约束、杠杆约束等都会引入不等式，用 cvxpy、scipy.optimize.minimize 等数值求解。

---

## 4.4 例题

**例题 4.4.1**  
\(\min x_1^2 + x_2^2\) s.t. \(x_1 + x_2 = 1\)。用拉格朗日法求候选点并验证为最小。

**解**：\(L = x_1^2 + x_2^2 + \lambda(x_1+x_2-1)\)。\(\partial L/\partial x_1 = 2x_1 + \lambda = 0\)，\(\partial L/\partial x_2 = 2x_2 + \lambda = 0\)，故 \(x_1=x_2\)；代入 \(x_1+x_2=1\) 得 \(x_1=x_2=1/2\)。目标为凸、约束为仿射，故 \((1/2, 1/2)\) 为全局最小，最小值为 \(1/4+1/4=1/2\)。

**代码**：
```python
import numpy as np
from scipy.optimize import minimize
def f(x):
    return x[0]**2 + x[1]**2
cons = {'type': 'eq', 'fun': lambda x: x[0] + x[1] - 1}
res = minimize(f, [0, 0], method='SLSQP', constraints=cons)
print("最优解:", res.x)   # [0.5, 0.5]
print("最小值:", res.fun) # 0.5
```

---

## 本章小结（第 4 章）

**核心概念**：拉格朗日 \(L = f + \sum \lambda_j h_j\)；一阶必要条件 \(\nabla_{\boldsymbol{x}} L = \boldsymbol{0}\) 且 \(h_j=0\)；均值-方差两约束下的解 \(\boldsymbol{w}^* = -\boldsymbol{\Sigma}^{-1}(\lambda_1 \boldsymbol{\mu} + \lambda_2 \boldsymbol{1})\) 代入约束解 \(\lambda\)；KKT 为不等式约束时的最优性条件。

**主要公式与结论**  
1. 等式约束：\(\nabla f + \sum \lambda_j \nabla h_j = \boldsymbol{0}\)，\(h_j=0\)。  
2. 均值-方差：\(\min \frac{1}{2}\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\boldsymbol{\mu}^\top \boldsymbol{w}=r\)，\(\boldsymbol{1}^\top \boldsymbol{w}=1\)；\(\boldsymbol{w}^* = \boldsymbol{\Sigma}^{-1}\boldsymbol{M}(\boldsymbol{M}^\top \boldsymbol{\Sigma}^{-1}\boldsymbol{M})^{-1}\boldsymbol{b}\)。

# 第 5 章 数值求解（Python）

**动机**：实际组合与约束多样，常需数值求解；scipy.optimize 适用于一般无约束/约束问题，cvxpy 适用于凸优化与二次规划。

---

## 5.1 scipy.optimize

- **minimize(f, x0, method='BFGS')**：无约束，可提供 jac 加速收敛。  
- **minimize(f, x0, constraints=..., method='SLSQP')**：含等式/不等式约束。  
- **minimize_scalar**：一维优化。

---

## 5.2 cvxpy（凸优化）

定义变量、目标、约束，调用 `problem.solve()`；适合二次规划、线性规划、锥规划。  
例：\(\min \boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\boldsymbol{1}^\top \boldsymbol{w}=1\)，\(\boldsymbol{\mu}^\top \boldsymbol{w} \ge r\)，\(\boldsymbol{w} \ge \boldsymbol{0}\)：
```python
import cvxpy as cp
w = cp.Variable(n)
prob = cp.Problem(cp.Minimize(cp.quad_form(w, Sigma)),
                  [cp.sum(w) == 1, mu.T @ w >= r, w >= 0])
prob.solve()
w_opt = w.value
```

---

## 5.3 组合权重示例

目标：\(\min \boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\boldsymbol{1}^\top \boldsymbol{w}=1\)，可选 \(\boldsymbol{\mu}^\top \boldsymbol{w} \ge r\) 或 \(\boldsymbol{w} \ge \boldsymbol{0}\)。用 cvxpy 或 scipy 写出约束，得到 \(\boldsymbol{w}^*\)，再算组合收益、波动、夏普等。

---

## 本章小结（第 5 章）

**核心概念**：scipy.optimize.minimize（BFGS、SLSQP）；cvxpy（凸/二次规划）；组合权重 min \(\boldsymbol{w}^\top \boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\boldsymbol{1}^\top \boldsymbol{w}=1\)、\(\boldsymbol{\mu}^\top \boldsymbol{w} \ge r\)、\(\boldsymbol{w} \ge \boldsymbol{0}\) 等。  
建议：能写出二次型与线性函数的梯度；能说明凸函数「局部极小=全局极小」；能写出带等式约束的拉格朗日并写出均值-方差两约束下的组合权重思路；会用 scipy 或 cvxpy 求简单组合权重。

# 第 6 章 与量化研究的衔接

- **组合优化**：均值-方差、最大夏普、风险平价、行业/因子中性约束，都归结为（含等式/不等式的）凸或二次规划。  
- **因子权重**：多因子打分或回归系数可作为目标收益的代理，再在约束下优化组合权重。  
- **机器学习**：损失最小化（Ridge、Lasso、神经网络）多为无约束或简单约束；梯度下降、SGD 是基础工具。  
- **风险预算**：各资产/因子对组合风险的边际贡献相等或成比例，可写成约束或目标，用数值优化求解。
