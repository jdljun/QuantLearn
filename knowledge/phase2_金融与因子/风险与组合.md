## 第 1 章 波动率

**动机**：收益的不确定性（波动率）是风险的核心度量；组合方差、夏普比率、VaR 与组合优化都依赖波动率与协方差的定义与估计。本节建立波动率的定义、样本估计、年化及在组合中的用法。

### 1.1 定义与估计

**定义 1.1.1（波动率）**  
设收益率 \(R\) 为随机变量，**波动率** \(\sigma\) 为其**标准差**：
\[
\sigma = \sqrt{\mathrm{Var}(R)} = \sqrt{E[(R - E[R])^2]}.
\]

**定义 1.1.2（样本波动率）**  
给定收益率样本 \(r_1, r_2, \ldots, r_n\)，样本均值 \(\bar{r} = \frac{1}{n}\sum_{t=1}^n r_t\)，**样本波动率**（无偏估计）为
\[
\hat{\sigma} = \sqrt{\frac{1}{n-1}\sum_{t=1}^n (r_t - \bar{r})^2}.
\]

**推导（为何用 \(n-1\)）**  
记 \(S^2 = \frac{1}{n-1}\sum_{t=1}^n (r_t - \bar{r})^2\)。因为 \(\sum_t (r_t - \bar{r}) = 0\)，\(n\) 个残差 \(r_t - \bar{r}\) 只有 \(n-1\) 个自由度；且可证 \(E[S^2] = \sigma^2\)（即 \(S^2\) 是 \(\sigma^2\) 的无偏估计），而 \(\frac{1}{n}\sum_t (r_t-\bar{r})^2\) 的期望为 \(\frac{n-1}{n}\sigma^2\)，偏小。故常用 \(n-1\) 做分母。

**性质 1.1.1（年化波动率）**  
若 \(r_t\) 为**日度**收益率且假设各日独立、同分布，则 **\(T\) 期（如 252 个交易日）年化波动率**为
\[
\sigma_{\text{annual}} = \sigma_{\text{daily}} \cdot \sqrt{252}.
\]
推导：\(\mathrm{Var}(r_1 + \cdots + r_T) = T \cdot \sigma_{\text{daily}}^2\)，故年化标准差为 \(\sigma_{\text{daily}}\sqrt{T}\)，取 \(T=252\)。

**注意**：若用简单收益 \(R_t\)，多期收益为连乘，严格年化需用 \((1+R_1)\cdots(1+R_T)-1\) 再算标准差；在收益较小时与「日波动率×√252」近似。对数收益多期可加，年化即 \(\sigma_{\text{daily}}\sqrt{252}\)。

### 1.2 在组合中的用法

**定义 1.2.1（组合收益与权重）**  
设 \(n\) 个资产，权重向量 \(\boldsymbol{w} = (w_1,\ldots,w_n)^\top\)，满足 \(\boldsymbol{1}^\top\boldsymbol{w} = 1\)（可含 \(\boldsymbol{w} \ge \boldsymbol{0}\)）。收益率向量 \(\boldsymbol{r} = (R_1,\ldots,R_n)^\top\)，则**组合收益**为 \(R_p = \boldsymbol{w}^\top\boldsymbol{r} = \sum_{i=1}^n w_i R_i\)。

**定理 1.2.1（组合方差）**  
设 \(\boldsymbol{\Sigma}\) 为收益率的 \(n\times n\) 协方差矩阵，则组合收益的方差为
\[
\sigma_p^2 = \mathrm{Var}(R_p) = \boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w},
\]
**组合波动率**为 \(\sigma_p = \sqrt{\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}}\)。

**证明**：\(R_p = \boldsymbol{w}^\top\boldsymbol{r}\)，故
\[
\mathrm{Var}(R_p) = \mathrm{Var}(\boldsymbol{w}^\top\boldsymbol{r}) = \boldsymbol{w}^\top \mathrm{Cov}(\boldsymbol{r})\boldsymbol{w} = \boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}.
\]

**定义 1.2.2（边际风险贡献）**  
第 \(i\) 资产对组合方差的**边际贡献**（权重增加一单位时方差的变化）与 \((\boldsymbol{\Sigma}\boldsymbol{w})_i\) 相关；**风险平价**追求各资产边际贡献相等（或与权重成比例）。

**注意**：**特质波动**为回归残差的标准差（收益 = 因子部分 + 特质部分），用于控制「非因子风险」；**系统波动**为因子暴露与因子波动共同决定的部分。

### 1.3 已实现波动率与隐含波动率（简述）

- **已实现波动率**：用高频收益的平方和（或加权）估计当日方差，比日收益方差更细。  
- **隐含波动率**：从期权价格反推的市场对未来波动率的预期；用于期权定价与风险意识。量化策略评估中常用**历史日度或滚动波动率**即可。

---

### 本章小结（第 1 章）

1. **波动率**：收益的标准差；样本估计 \(\hat{\sigma} = \sqrt{\frac{1}{n-1}\sum_t (r_t-\bar{r})^2}\)；年化 \(\sigma_{\text{annual}} = \sigma_{\text{daily}}\sqrt{252}\)。  
2. **组合波动率**：\(\sigma_p = \sqrt{\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}}\)；边际贡献与 \(\boldsymbol{\Sigma}\boldsymbol{w}\) 相关。

## 第 2 章 最大回撤（Maximum Drawdown）

**动机**：波动率衡量的是收益的离散程度，不直接衡量「最坏一段从高到低的损失」；最大回撤（MDD）刻画资金曲线从历史高点到之后低点的最大跌幅，是风控与策略评估的必报指标。

### 2.1 定义

**定义 2.1.1（回撤）**  
设 \(NAV_t\) 为 \(t\) 时刻净值，\(NAV_{\max,t} = \max_{s\le t} NAV_s\) 为 \(t\) 及之前净值的最高点。**\(t\) 时刻回撤**（比例）为
\[
\text{Drawdown}_t = \frac{NAV_t - NAV_{\max,t}}{NAV_{\max,t}}.
\]
通常 \(NAV_t \le NAV_{\max,t}\)，故 \(\text{Drawdown}_t \le 0\)。

**定义 2.1.2（最大回撤 MDD）**  
**最大回撤**为回撤序列在样本期内的**最小值**（因回撤为负，最小即跌幅最大）：
\[
\text{MDD} = \min_{t} \text{Drawdown}_t = \min_t \frac{NAV_t - NAV_{\max,t}}{NAV_{\max,t}}.
\]
有时也写为 \(\text{MDD} = \max_t |\text{Drawdown}_t|\)（最大跌幅的绝对值）。

### 2.2 性质与用途

- MDD 衡量「**最坏的一段从高到低**」的损失；**不区分**路径（先跌后涨与先涨后跌可能 MDD 相同）。  
- **回撤持续时间**：从高点到再次创新高所需时间；可补充报告。  
- **量化应用**：实盘与风控中设定「最大可接受回撤」，超过则减仓或止损；回测中 MDD 是必报指标；与夏普、卡玛一起使用。

### 2.3 计算实现与例题

**算法要点**：  
1. 维护历史最高净值：\(NAV_{\max,t} = \max(NAV_{\max,t-1}, NAV_t)\)。  
2. \(DD_t = (NAV_t - NAV_{\max,t}) / NAV_{\max,t}\)；若 \(NAV_{\max,t}=0\) 需单独处理。  
3. \(\text{MDD} = \min_t DD_t\)（或 \(\max_t |DD_t|\)）。

**例题 2.3.1**  
某策略日净值序列（5 日）为：1.00, 1.02, 0.98, 0.97, 1.01。求各日回撤与最大回撤。

**解**：  
- \(NAV_{\max} = 1.00, 1.02, 1.02, 1.02, 1.02\)。  
- \(DD_1=0,\ DD_2=0,\ DD_3=(0.98-1.02)/1.02 \approx -3.92\%,\ DD_4=(0.97-1.02)/1.02 \approx -4.90\%,\ DD_5=(1.01-1.02)/1.02 \approx -0.98\%\)。  
- \(\text{MDD} = \min(DD_t) = DD_4 \approx -4.90\%\)，即最大回撤约 4.90%。

**代码示例**（Python）：

```python
import numpy as np

def max_drawdown(nav: np.ndarray) -> float:
    """nav: 净值序列（从早到晚）"""
    nav_max = np.maximum.accumulate(nav)
    dd = (nav - nav_max) / nav_max
    return np.min(dd)  # 负值，表示最大跌幅

nav = np.array([1.00, 1.02, 0.98, 0.97, 1.01])
print(max_drawdown(nav))  # 约 -0.0490，即 -4.90%
```

**注意**：若净值序列很长，需注意数值稳定性（如净值始终为正）；\(NAV_{\max,t}\) 用累积最大值即可，无需循环内再求 \(\max_{s\le t}\)。

---

### 本章小结（第 2 章）

1. **回撤**：\(DD_t = (NAV_t - NAV_{\max,t})/NAV_{\max,t}\)；**MDD** = \(\min_t DD_t\)。  
2. **用途**：风控阈值、回测必报指标；可与卡玛比率（年化收益/|MDD|）结合使用。

## 第 3 章 VaR 与 CVaR（Expected Shortfall）

**动机**：波动率与 MDD 不直接回答「在给定概率下，损失会多大」；VaR 与 CVaR 从分布尾部刻画风险，常用于风控限额与压力测试。

### 3.1 VaR（Value at Risk）

**定义 3.1.1（VaR）**  
在给定**置信水平 \(1-\alpha\)**（如 95%，即 \(\alpha=0.05\)）下，**未来一段时期内**组合**损失**不超过某数值 \(V\) 的概率为 \(1-\alpha\)；该数值 \(V\) 即 **VaR**（通常取正数表示损失规模）。

等价表述：**损失超过 VaR 的概率为 \(\alpha\)**；即 VaR 是损失的 **\(\alpha\) 分位数**。若将「损失」定义为 \(-R\)（\(R\) 为收益），则 VaR 对应收益分布的 **\(\alpha\) 分位数**的相反数。

**例题 3.1.1**  
1 日 95% VaR = 2% 表示：有 5% 的概率，单日损失超过 2%（或单日收益低于 −2%）。

**计算方式**  
- **历史法**：收益序列的 **\(\alpha\) 分位数**（取负号若定义为损失）。  
- **参数法**：若假设收益 \(R \sim N(\mu,\sigma^2)\)，损失 \(L = -R\)，则 \(L\) 的 \(\alpha\) 分位数 \(q_\alpha(L)\) 满足 \(P(L \le q_\alpha) = \alpha\)。标准正态 \(\alpha\) 分位数记为 \(z_\alpha\)（\(P(Z \le z_\alpha)=\alpha\)），则
\[
\text{VaR}_\alpha = \mu - z_\alpha \sigma \quad \text{（若 VaR 定义为收益的下分位数，即 } P(R \le -\text{VaR}) = \alpha\text{）}.
\]
常见：\(\alpha=0.05\) 时 \(z_{0.05}\approx -1.645\)，故 \(\text{VaR} \approx \mu + 1.645\sigma\)（收益为正态时，5% 左尾对应 \(\mu - 1.645\sigma\)，取负得损失）。  
- **蒙特卡洛**：模拟收益分布再取 \(\alpha\) 分位数。

### 3.2 CVaR / Expected Shortfall (ES)

**定义 3.2.1（CVaR / ES）**  
**CVaR**（Conditional VaR）或 **Expected Shortfall (ES)**：在**损失超过 VaR** 的条件下，损失的**期望值**；即**尾部平均损失**。
\[
\text{CVaR}_\alpha = E[\text{损失} \mid \text{损失} \ge \text{VaR}_\alpha].
\]

**性质**：CVaR ≥ VaR；对厚尾分布，CVaR 更能反映极端损失；CVaR 满足**次可加性**（组合 CVaR ≤ 各资产 CVaR 之和），适合做风险配置。

**计算**：历史法下，取**低于 VaR 对应的收益**的那部分样本（即损失超过 VaR 的部分），求其平均再取负号；或通过 Rockafellar-Uryasev 公式与优化器求解。

### 3.3 在量化中的用法

- **风控**：限制组合的 1 日或 10 日 VaR/CVaR 不超过预算。  
- **评估**：回测中报告「最大 1 日损失」「95% VaR」等，补充夏普与回撤；实盘可设 VaR 预警线。

**注意**：VaR 不满足次可加性，组合 VaR 可能大于各资产 VaR 之和；CVaR 更适合作风险约束或优化目标。

---

### 本章小结（第 3 章）

1. **VaR**：损失（或负收益）的 \(\alpha\) 分位数；历史法/参数法/蒙特卡洛。正态时 \(\text{VaR} \approx \mu + 1.645\sigma\)（5%）。  
2. **CVaR/ES**：尾部平均损失；CVaR ≥ VaR；满足次可加性。  
3. **应用**：风控限额、回测与实盘风险报告。

## 第 4 章 夏普比率与卡玛比率

**动机**：仅看收益或仅看风险都不足；夏普比率衡量「单位风险承担的额外回报」，卡玛比率衡量「单位最大回撤带来的收益」，是策略评估与比较的常用指标。

### 4.1 夏普比率（Sharpe Ratio）

**定义 4.1.1（夏普比率）**  
设 \(R_p\) 为组合收益，\(R_f\) 为无风险利率，\(\sigma_p\) 为组合波动率（标准差）。**夏普比率**为
\[
\text{SR} = \frac{R_p - R_f}{\sigma_p},
\]
即**超额收益（相对无风险利率）**除以**组合波动率**。

**年化**：若 \(R_p\)、\(\sigma_p\) 为年化收益与年化波动率，则 SR 已为年化夏普；若为日度，在收益近似不相关时有 \(\text{SR}_{\text{annual}} \approx \text{SR}_{\text{daily}} \times \sqrt{252}\)（因年化超额收益 ≈ 日度超额收益×252，年化波动率 ≈ 日度波动率×√252）。

**含义**：单位风险承担的额外回报；越高越好。通常 **SR > 1** 视为较好，**SR > 2** 视为优秀；需结合样本长度与多重检验谨慎解读。

### 4.2 信息比率（IR）

**定义 4.2.1（信息比率）**  
**IR** = \((R_p - R_b) / \sigma_{\text{active}}\)，即**主动收益**（相对基准 \(R_b\)）除以**主动风险**（跟踪误差，即 \(R_p - R_b\) 的标准差）。

**含义**：单位主动风险带来的超额收益；用于**相对收益策略**（如指数增强）。

### 4.3 卡玛比率（Calmar Ratio）

**定义 4.3.1（卡玛比率）**  
**Calmar** = 年化收益 / \(|MDD|\)（或 年化收益 / 最大回撤的绝对值）。

**含义**：单位最大回撤带来的收益；强调**回撤惩罚**，对回撤敏感的策略更适用。

### 4.4 其他指标（简述）

- **索提诺比率**：分母用**下行波动率**（只考虑负收益的波动），更关注下行风险。  
- **收益/回撤比**：类似 Calmar，可自定义回撤窗口（如 1 年滚动 MDD）。

**注意**：夏普与卡玛都依赖历史估计；样本短时估计误差大；实盘与回测的假设（成本、滑点、前视）一致时才有可比性。

---

### 本章小结（第 4 章）

1. **夏普**：\(\text{SR} = (R_p - R_f)/\sigma_p\)；年化 SR ≈ 日度 SR × √252。  
2. **IR**：\((R_p - R_b)/\sigma_{\text{active}}\)；用于相对收益策略。  
3. **卡玛**：年化收益 / |MDD|。

## 第 5 章 均值-方差组合优化

**动机**：在给定预期收益与协方差估计下，如何配置权重使风险最小或夏普最大？均值-方差框架（Markowitz）给出明确数学形式，并可转化为二次规划（QP）求解。

### 5.1 问题形式

**输入**：预期收益向量 \(\boldsymbol{\mu}\)、协方差矩阵 \(\boldsymbol{\Sigma}\)（或估计值 \(\hat{\boldsymbol{\mu}}\)、\(\hat{\boldsymbol{\Sigma}}\)）。  
**目标**：在约束下**最小化方差**或**最大化夏普**。  
**常见约束**：\(\boldsymbol{1}^\top\boldsymbol{w} = 1\)（满仓）、\(\boldsymbol{w} \ge \boldsymbol{0}\)（不做空）、\(\boldsymbol{\mu}^\top\boldsymbol{w} \ge r_{\min}\)（最低收益）、行业/因子中性等。

### 5.2 最小方差组合

**问题**：
\[
\min_{\boldsymbol{w}} \ \frac{1}{2}\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w} \quad \text{s.t.} \quad \boldsymbol{1}^\top\boldsymbol{w} = 1.
\]
可选 \(\boldsymbol{w} \ge \boldsymbol{0}\)。

**定理 5.2.1（最小方差权重，无不等式约束）**  
在仅约束 \(\boldsymbol{1}^\top\boldsymbol{w}=1\) 时，解析解为
\[
\boldsymbol{w} = \frac{\boldsymbol{\Sigma}^{-1}\boldsymbol{1}}{\boldsymbol{1}^\top\boldsymbol{\Sigma}^{-1}\boldsymbol{1}}.
\]

**证明要点**：拉格朗日 \(L = \frac{1}{2}\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w} - \lambda(\boldsymbol{1}^\top\boldsymbol{w}-1)\)，令 \(\nabla_{\boldsymbol{w}} L = \boldsymbol{\Sigma}\boldsymbol{w} - \lambda\boldsymbol{1} = \boldsymbol{0}\) 得 \(\boldsymbol{w} = \lambda\boldsymbol{\Sigma}^{-1}\boldsymbol{1}\)；代入 \(\boldsymbol{1}^\top\boldsymbol{w}=1\) 得 \(\lambda = 1/(\boldsymbol{1}^\top\boldsymbol{\Sigma}^{-1}\boldsymbol{1})\)，即上式。有不等式 \(\boldsymbol{w}\ge\boldsymbol{0}\) 时需用 QP 求解。

### 5.3 目标收益下的最小方差（Markowitz）

**问题**：
\[
\min_{\boldsymbol{w}} \ \frac{1}{2}\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w} \quad \text{s.t.} \quad \boldsymbol{1}^\top\boldsymbol{w} = 1,\ \boldsymbol{\mu}^\top\boldsymbol{w} = r_{\text{target}}.
\]

**结论**：有效前沿上的组合可由**两基金分离**表示：最小方差组合与另一「高收益」组合的线性组合；解析式可写为 \(\boldsymbol{w}\) 关于 \(r_{\text{target}}\) 的显式（见最优化基础）。实际中 \(\boldsymbol{\mu}\)、\(\boldsymbol{\Sigma}\) 用样本估计，**估计误差**会导致权重不稳定；**正则化**（如 \(\boldsymbol{\Sigma}+\lambda I\)）、**约束**（如 \(\boldsymbol{w}\ge\boldsymbol{0}\)）、**Black-Litterman** 等可改善。

### 5.4 最大夏普组合

**问题**：
\[
\max_{\boldsymbol{w}} \ \frac{\boldsymbol{\mu}^\top\boldsymbol{w} - r_f}{\sqrt{\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}}} \quad \text{s.t.} \quad \boldsymbol{1}^\top\boldsymbol{w} = 1 \ (\text{及可选 } \boldsymbol{w}\ge\boldsymbol{0}).
\]

**等价形式**：在**超额收益**空间上，令 \(\tilde{\boldsymbol{\mu}} = \boldsymbol{\mu} - r_f\boldsymbol{1}\)，可化为 \(\min \boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\tilde{\boldsymbol{\mu}}^\top\boldsymbol{w} = 1\)（或固定常数），仍为二次规划。

**定理 5.4.1（最大夏普权重，无不等式约束）**  
解析解为 \(\boldsymbol{w} \propto \boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu} - r_f\boldsymbol{1})\)，再归一化使 \(\boldsymbol{1}^\top\boldsymbol{w}=1\)，即
\[
\boldsymbol{w} = \frac{\boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu} - r_f\boldsymbol{1})}{\boldsymbol{1}^\top\boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu} - r_f\boldsymbol{1})}.
\]

**推导要点**：目标等价于 \(\max \tilde{\boldsymbol{\mu}}^\top\boldsymbol{w}\) s.t. \(\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}=1\) 与 \(\boldsymbol{1}^\top\boldsymbol{w}=1\)；由一阶条件可得上述比例关系。

### 5.5 风险平价（定义、公式与实现）

**动机**：均值-方差对 \(\boldsymbol{\mu}\) 估计敏感，且常使权重集中于少数资产；**风险平价**追求各资产对组合风险的**贡献**相等，分散化程度高，对收益估计依赖低。

**定义 5.5.1（边际风险贡献）**  
组合方差 \(\sigma_p^2 = \boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}\)。对 \(w_i\) 求偏导得 \(\partial \sigma_p^2 / \partial w_i = 2(\boldsymbol{\Sigma}\boldsymbol{w})_i\)。第 \(i\) 资产对组合方差的**边际贡献**为  
\[
\text{MRC}_i = w_i \cdot (\boldsymbol{\Sigma}\boldsymbol{w})_i = w_i \cdot \frac{\partial \sigma_p^2}{\partial w_i}.
\]  
**风险贡献**（RC）为 \(w_i \cdot (\boldsymbol{\Sigma}\boldsymbol{w})_i\)，满足 \(\sum_i \text{RC}_i = \sigma_p^2\)。

**定义 5.5.2（风险平价）**  
**风险平价**（Risk Parity）要求各资产的**风险贡献相等**：\(\text{RC}_i = \sigma_p^2 / n\)（\(n\) 为资产数），即  
\[
w_i (\boldsymbol{\Sigma}\boldsymbol{w})_i = w_j (\boldsymbol{\Sigma}\boldsymbol{w})_j \quad \forall i,j.
\]

**定理 5.5.1（简化情形：对角协方差）**  
若 \(\boldsymbol{\Sigma}\) 为**对角矩阵**（资产间不相关），则风险平价权重与**波动率倒数**成比例：  
\[
w_i \propto \frac{1}{\sigma_i}, \qquad \boldsymbol{w} = \frac{\boldsymbol{\sigma}^{-1}}{\boldsymbol{1}^\top\boldsymbol{\sigma}^{-1}},
\]  
其中 \(\boldsymbol{\sigma} = (\sigma_1,\ldots,\sigma_n)^\top\) 为各资产波动率。此时 \(\text{RC}_i = w_i^2 \sigma_i^2\)，令其相等得 \(w_i \propto 1/\sigma_i\)。

**一般情形（相关）**：风险平价权重**无闭式解**，需**迭代**或**凸优化**求解。一种等价形式为：求 \(\boldsymbol{w} \ge \boldsymbol{0}\)、\(\boldsymbol{1}^\top\boldsymbol{w}=1\)，使 \(\sum_i \ln w_i + \ln(\boldsymbol{\Sigma}\boldsymbol{w})_i\) 在约束下一致（或最小化 \(\sum_i \text{RC}_i \ln \text{RC}_i\) 等）；实践中常用**迭代**：给定初始 \(\boldsymbol{w}\)，计算 \((\boldsymbol{\Sigma}\boldsymbol{w})_i\)，令 \(w_i^{\text{new}} \propto w_i / (\boldsymbol{\Sigma}\boldsymbol{w})_i\) 再归一化，直至收敛。

**实现要点**：cvxpy 可写为凸问题（见文献）；或使用「等风险贡献」的现成求解器。**与目标波动率结合**：先求风险平价权重 \(\boldsymbol{w}\)，再乘以标量 \(\theta = \sigma_{\text{target}}/\hat{\sigma}_p\) 使组合波动率等于目标（见资产配置）。

### 5.6 例题与 Python 实现

**例题 5.6.1**  
3 个资产，预期年化收益 \(\boldsymbol{\mu} = (0.08, 0.10, 0.06)^\top\)，协方差矩阵（年化）
\[
\boldsymbol{\Sigma} = \begin{pmatrix} 0.04 & 0.02 & 0.01 \\ 0.02 & 0.09 & 0.02 \\ 0.01 & 0.02 & 0.0225 \end{pmatrix},
\]
无风险利率 \(r_f = 0.02\)。求（1）最小方差组合权重；（2）最大夏普组合权重（允许做空）。

**解**：  
（1）\(\boldsymbol{w}_{\text{mv}} = \boldsymbol{\Sigma}^{-1}\boldsymbol{1}/(\boldsymbol{1}^\top\boldsymbol{\Sigma}^{-1}\boldsymbol{1})\)，代入数值计算即可。  
（2）\(\boldsymbol{w} \propto \boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu} - r_f\boldsymbol{1})\)，再归一化。

**代码示例**（cvxpy，最小方差 + 满仓 + 非负）：

```python
import numpy as np
import cvxpy as cp

mu = np.array([0.08, 0.10, 0.06])
Sigma = np.array([
    [0.04, 0.02, 0.01],
    [0.02, 0.09, 0.02],
    [0.01, 0.02, 0.0225]
])
n = len(mu)
w = cp.Variable(n)
obj = cp.Minimize(cp.quad_form(w, Sigma))
con = [cp.sum(w) == 1, w >= 0]
prob = cp.Problem(obj, con)
prob.solve()
print("最小方差权重:", w.value)
# 最大夏普：等价于 min w'Sigma w s.t. (mu-rf)'w=1, sum(w)=1, w>=0
rf = 0.02
w2 = cp.Variable(n)
obj2 = cp.Minimize(cp.quad_form(w2, Sigma))
con2 = [cp.sum(w2) == 1, (mu - rf) @ w2 == 1, w2 >= 0]
prob2 = cp.Problem(obj2, con2)
prob2.solve()
print("最大夏普权重（归一化前和为1）:", w2.value)
# 若需 1'w=1 且为夏普最大，常用另一种参数化：见文档
```

**注意**：\(\boldsymbol{\mu}\)、\(\boldsymbol{\Sigma}\) 的估计误差会导致优化权重对输入敏感；实践中常用正则化、约束或滚动估计降低不稳定性。

### 5.7 贝叶斯观点与 Black-Litterman（简述）

**动机**：纯均值-方差使用**样本均值** \(\hat{\boldsymbol{\mu}}\)，估计误差大且权重敏感；**Black-Litterman** 将**市场均衡收益**（由基准权重反推）与**投资者观点**（如某资产超额收益的预期）通过**贝叶斯**结合，得到**后验预期收益**，再做均值-方差优化，使权重更稳定且可融入观点。

**定义 5.7.1（均衡收益）**  
设**市场组合**或基准权重为 \(\boldsymbol{w}_{\text{mkt}}\)，在均衡下可反推隐含预期收益：若 \(\boldsymbol{w}_{\text{mkt}} \propto \boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu}_{\text{equil}} - r_f\boldsymbol{1})\)，则 \(\boldsymbol{\mu}_{\text{equil}}\) 为**均衡预期收益**（CAPM 下与市场 beta 一致）。

**定义 5.7.2（观点）**  
投资者对部分资产或线性组合的收益提出**观点**，如「资产 1 超额收益为 \(q_1 \pm \omega_1\)」（\(\omega_1\) 为不确定性）。写成 \(\boldsymbol{P}\boldsymbol{\mu} = \boldsymbol{q} + \boldsymbol{\varepsilon}\)，\(\boldsymbol{\varepsilon} \sim N(\boldsymbol{0}, \boldsymbol{\Omega})\)。

**Black-Litterman 后验**（简述）：先验为 \(\boldsymbol{\mu} \sim N(\boldsymbol{\mu}_{\text{equil}}, \tau\boldsymbol{\Sigma})\)（或经缩放），似然为观点 \(\boldsymbol{P}\boldsymbol{\mu} = \boldsymbol{q} + \boldsymbol{\varepsilon}\)；**后验** \(\boldsymbol{\mu} \mid \text{views}\) 为高斯，均值为  
\[
\boldsymbol{\mu}_{\text{BL}} = \bigl[(\tau\boldsymbol{\Sigma})^{-1} + \boldsymbol{P}^\top\boldsymbol{\Omega}^{-1}\boldsymbol{P}\bigr]^{-1} \bigl[(\tau\boldsymbol{\Sigma})^{-1}\boldsymbol{\mu}_{\text{equil}} + \boldsymbol{P}^\top\boldsymbol{\Omega}^{-1}\boldsymbol{q}\bigr].
\]  
用 \(\boldsymbol{\mu}_{\text{BL}}\) 替代 \(\boldsymbol{\mu}\) 做均值-方差或最大夏普优化，得到**融入观点**的权重；\(\tau\) 与 \(\boldsymbol{\Omega}\) 控制对均衡与观点的信任度。

**性质**：无观点时 \(\boldsymbol{\mu}_{\text{BL}} = \boldsymbol{\mu}_{\text{equil}}\)，权重回归基准；观点越确定（\(\boldsymbol{\Omega}\) 小）则后验越偏向观点。详见《主动投资组合管理》或 Black-Litterman 原文献。

### 本章小结（第 5 章）

1. **最小方差**：\(\min \frac{1}{2}\boldsymbol{w}^\top\boldsymbol{\Sigma}\boldsymbol{w}\) s.t. \(\boldsymbol{1}^\top\boldsymbol{w}=1\)；\(\boldsymbol{w} = \boldsymbol{\Sigma}^{-1}\boldsymbol{1}/(\boldsymbol{1}^\top\boldsymbol{\Sigma}^{-1}\boldsymbol{1})\)。  
2. **最大夏普**：\(\boldsymbol{w} \propto \boldsymbol{\Sigma}^{-1}(\boldsymbol{\mu}-r_f\boldsymbol{1})\)，再归一化；有不等式时用 QP。  
3. **风险平价**：各资产风险贡献相等；对角 \(\boldsymbol{\Sigma}\) 时 \(w_i \propto 1/\sigma_i\)；一般情形迭代或凸优化。  
4. **Black-Litterman**：均衡收益 + 观点 → 贝叶斯后验 \(\boldsymbol{\mu}_{\text{BL}}\) → 均值-方差优化；降低对 \(\boldsymbol{\mu}\) 估计的敏感度并融入观点。  
5. **实现**：cvxpy 的 `cp.quad_form(w, Sigma)` + 约束；风险平价可用迭代或专用求解器。

## 第 6 章 与量化研究的衔接

- **策略评估**：回测必须报告**年化收益、波动率、夏普、MDD、卡玛**（及可选 VaR/CVaR、IR）；并说明样本区间与成本假设。  
- **组合构建**：多因子选股后，可用**均值-方差或风险平价**在标的池内分配权重；或先按因子打分选股，再等权/市值加权。  
- **风控**：实盘限制**单日 VaR、最大仓位、行业暴露**等；回测中可做**压力测试**（如波动率放大、收益下修）看 MDD 与 VaR 变化。  
- **估计风险**：\(\boldsymbol{\mu}\)、\(\boldsymbol{\Sigma}\) 的估计误差会导致优化权重不稳定；正则化、约束、滚动估计、贝叶斯收缩是常用手段。

**建议**：能计算并年化夏普、MDD、卡玛；能说明 VaR 与 CVaR 的含义与区别；能写出均值-方差问题的约束形式，并用 cvxpy 或 scipy 求最小方差或最大夏普权重（含 \(\boldsymbol{1}^\top\boldsymbol{w}=1\) 与可选 \(\boldsymbol{w}\ge\boldsymbol{0}\)）。
