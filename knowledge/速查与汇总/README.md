# 速查与汇总 使用说明

> 本目录为各阶段**产出物**与**速查笔记**的汇总，便于复习、考前/实盘前快速回顾和按主题查找公式与要点。

---

## 一、按学习阶段选文件

| 阶段 | 文件 | 用途 |
|------|------|------|
| **Phase 0** | [概率与统计速查.md](概率与统计速查.md) | 概率（条件、贝叶斯、分布、大数/CLT）、估计、假设检验、相关系数、分位数；与量化对应 |
| **Phase 1** | [回归与时间序列要点.md](回归与时间序列要点.md) | OLS、R²/F/t、共线性与异方差、Ridge/Lasso；平稳、AR/MA/ARMA、ADF、协整；最优化与 ML 要点 |
| **Phase 2** | [多因子与回测要点.md](多因子与回测要点.md) | CAPM/三因子、因子构造与 IC/IR、中性化、分组回测；收益与净值、成本、前视/幸存者、OOS；波动率、夏普、回撤、VaR、组合优化；市场与价量；**扩展**：期权/固收/行为/资产配置、风险平价/BL、执行模型（见第六节） |
| **Phase 3** | [策略与实证要点.md](策略与实证要点.md) | 策略类型（选股/择时、多因子、动量/反转、配对、择时）；过拟合控制（OOS、CV、正则、检查清单）；实盘与系统（信号、仓位、风控、回测差异）；论文与研报 |

## 二、按主题快速定位

| 主题 | 主要文件 | 可补充 |
|------|----------|--------|
| 概率、分布、假设检验、相关系数 | 概率与统计速查 | phase0 概率论、数理统计 |
| 回归、时间序列、优化、ML | 回归与时间序列要点 | phase1 回归、时间序列、最优化、机器学习入门 |
| 多因子、回测、风险与组合、期权/固收/行为/配置 | 多因子与回测要点 | phase2 多因子、回测、风险与组合、期权与波动率基础、固定收益基础、行为金融要点、资产配置、价量（含执行模型） |
| 策略设计、过拟合、实盘、文献 | 策略与实证要点 | phase3 策略类型、过拟合控制、实盘与系统、论文与研报 |
| 书籍、网课、数据与平台、Python 库 | 推荐书籍与资源 | LEARNING_PLAN.md |
