## 一、回归

### OLS
- β̂ = (XᵀX)⁻¹XᵀY；Ŷ = Xβ̂；e = Y−Ŷ — 回归 第2章
- R² = 1−RSS/TSS；R̄² = 1−(RSS/(n−p−1))/(TSS/(n−1)) — 回归 第3章
- F 检验（整体）：F = (ESS/p)/(RSS/(n−p−1)) ∼ F(p,n−p−1) under H₀: 所有斜率=0 — 回归 第3章 §3.3
- t 检验（单系数）：t_j = β̂_j/SE(β̂_j) ∼ t(n−p−1) under H₀: β_j=0 — 回归 第4章
- SE(β̂_j) = √(s²·[(XᵀX)⁻¹]_{jj})，s² = RSS/(n−p−1)

### 多重共线性
- X 列近似线性相关 → (XᵀX) **病态**（条件数大）→ β̂ 方差大、t 小 — 回归 第5章、线性代数应用 第4章
- **条件数** κ = λ_max/λ_min（或范数形式）；κ 大则数值不稳定
- 处理：增样本、删/并变量、Ridge/Lasso、**PCA 降维**

### 异方差
- Var(εᵢ) 随 i 变 → OLS 仍无偏一致，但标准误错误 → 用稳健标准误（White 等）— 回归 第6章

### Ridge / Lasso
- Ridge：min ‖Y−Xβ‖² + λ‖β‖²₂ → β̂ = (XᵀX+λI)⁻¹XᵀY — 回归、机器学习入门 第2章
- Lasso：min ‖Y−Xβ‖² + λ‖β‖₁ → 稀疏解，变量选择

## 二、时间序列

### 平稳与白噪声
- 弱平稳：E(X_t)=μ，Var(X_t)=σ²，Cov(X_t,X_{t−k})=γ_k 与 t 无关 — 时间序列 第1章 定义1.1
- 白噪声：E(ε_t)=0，Var(ε_t)=σ²，Cov(ε_t,ε_s)=0（t≠s）— 时间序列 第1章 §1.3

### AR/MA/ARMA
- AR(p)：X_t = c + φ₁X_{t−1}+…+φ_p X_{t−p} + ε_t；平稳 ⟺ φ(L)=0 的根在单位圆外 — 时间序列 第3章 §3.1
- MA(q)：X_t = μ + ε_t + θ₁ε_{t−1}+…+θ_q ε_{t−q}；总平稳 — 时间序列 第3章 §3.2
- ARMA(p,q)：φ(L)X_t = c + θ(L)ε_t
- AR：ACF 拖尾，PACF p 阶截尾；MA：ACF q 阶截尾，PACF 拖尾 — 时间序列 第2章 §2.3

### 单位根与 ADF
- 单位根 ⟹ 非平稳；ADF 检验 H₀: 有单位根；p 小 ⟹ 拒绝 ⟹ 平稳 — 时间序列 第4章

### 协整
- 两序列都 I(1)，存在线性组合为 I(0) ⟹ 协整；Engle-Granger：回归得残差，对残差 ADF；残差平稳 ⟹ 协整 — 时间序列 第5章

## 三、最优化（要点）

### 无约束
- 一阶必要：∇f(x*)=0
- 凸 f：局部极小 = 全局极小；严格凸则唯一

### 等式约束与拉格朗日
- min f(x) s.t. h_j(x)=0 → L = f(x)+Σ λ_j h_j(x)
- 必要条件：∇_x L=0，h_j=0

### 均值-方差组合（两约束）
- min (1/2)wᵀΣw s.t. 1ᵀw=1，μᵀw=r_target
- 解：w* = Σ⁻¹M(MᵀΣ⁻¹M)⁻¹b，M=[μ,1]，b=[r_target,1]ᵀ

### 数值
- scipy.optimize.minimize；cvxpy 做凸规划（QP、LP）

### 线性代数与 PCA（要点）
- **特征值/特征向量**：A v = λ v；对称阵正交特征向量、谱分解 — 线性代数应用
- **PCA**：协方差矩阵特征值从大到小，前 k 个特征向量为主成分；解释方差比 = λ_i/Σλ_j — 线性代数应用、因子降维

### 机器学习（要点）
- **线性/岭/Lasso**：同回归；Lasso 稀疏 — 机器学习入门
- **树模型**：决策树、随机森林、GBDT；**过拟合**控制：深度、叶节点数、最小样本数、早停 — 机器学习入门
- **交叉验证**：K 折；**时间序列**必须按时间划分、不打乱（TimeSeriesSplit）— 过拟合控制

## 四、与量化的对应

- 横截面回归 → 因子收益、预测收益；t/F → 因子显著性
- 时间序列 → 收益/波动率建模、预测；ADF → 平稳性；协整 → 配对
- Ridge/Lasso → 因子权重、共线性；优化 → 组合权重、风险预算
