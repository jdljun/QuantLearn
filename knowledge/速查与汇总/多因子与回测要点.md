## 一、多因子

### CAPM 与三因子
- CAPM：E(R_i)−R_f = β_i(E(R_m)−R_f)；β_i = Cov(R_i,R_m)/Var(R_m) — 多因子模型 第1章 §1.1
- 三因子：R_i−R_f = α_i + β_MKT·MKT + β_SMB·SMB + β_HML·HML + ε_i — 多因子模型 第2章 §2.1

### 因子构造与暴露
- 原始特征 → 去极值（如 1%–99% 或 MAD）→ 截面标准化或排名 — 多因子模型 第3章
- 暴露 = 标准化后的因子值，或对因子收益回归的 β

### IC / IR
- IC_t = corr(暴露_t, 收益_{t→t+1})（截面 Pearson）— 多因子模型 第5章 §5.1
- mean(IC)、std(IC)；IR = mean(IC)/std(IC) — 多因子模型 第5章 §5.2
- ρ=0 检验：t = r√(n−2)/√(1−r²) ∼ t(n−2) — 数理统计 第7章 §7.2

### 中性化
- 回归法：暴露对「需中性化的变量」（如行业哑变量、ln(市值)）回归，用**残差**作为中性化后暴露 — 多因子模型 第4章 §4.2
- 优化法：约束行业权重=基准、或 wᵀ·(暴露向量)=0 — 多因子模型 第4章 §4.3

### 分组回测
- 按因子五分/十分位；多空 = 高分组收益 − 低分组收益；看 IC、IR、夏普、单调性 — 多因子模型 第5章 §5.4

## 二、回测

### 收益与净值
- **简单收益** R_t = (P_t−P_{t−1})/P_{t−1}；**对数收益** r_t = ln(P_t/P_{t−1})，小 R 时 r≈R — 回测框架 第1章
- 组合收益 R_{p,t} = Σ w_{i,t} R_{i,t}；净值 NAV_t = NAV_{t−1}(1+R_{p,t})；多期累计 (1+R_1)…(1+R_T)−1
- 复权价格；考虑分红与拆股（**后复权**优先，保证收益连续）

### 成本
- 每笔：佣金+印花税+滑点+冲击；从当期收益中扣 — 回测框架 第2章
- 敏感性：多档成本假设看收益与夏普变化

### 前视偏差（禁止）
- 信号与标准化**只用过去数据**；执行在信号**之后**且可执行时点 — 回测框架 第3章
- 不用全样本或未来数据做标准化/参数

### 幸存者偏差
- 用**历史成分**或含退市数据；不用「当前成分回溯」— 回测框架 第4章

### 样本内/外
- IS：训练与调参；OOS：从未参与选择，仅最终评估；严格时间顺序 — 回测框架 第5章

### 检查清单（简）
- 收益与净值计算正确；成本每笔扣；无前视；股票池含退市或历史成分；OOS 严格 — 回测框架 第6章

## 三、风险与组合

### 波动率与夏普
- σ = std(收益)；年化 σ_ann = σ_daily×√252 — 风险与组合
- 夏普 SR = (R−R_f)/σ；年化 SR ≈ SR_daily×√252
- 组合波动 σ_port = √(wᵀΣw)

### 回撤与 MDD
- Drawdown_t = (NAV_t − max_{s≤t} NAV_s) / max_{s≤t} NAV_s — 风险与组合
- MDD = max_t |Drawdown_t|

### VaR / CVaR
- VaR_α：损失的 α 分位数；历史法 = 收益序列的 α 分位数取负 — 概率论 分位数、风险与组合
- CVaR：损失超过 VaR 时的平均损失（尾部期望）

### 组合优化
- 最小方差：min wᵀΣw s.t. 1ᵀw=1，（w≥0）— 最优化基础 第3–4章、风险与组合
- 目标收益：min wᵀΣw s.t. 1ᵀw=1，μᵀw=r
- 最大夏普：max (μᵀw−r_f)/√(wᵀΣw) s.t. 1ᵀw=1
- 实现：cvxpy 或 scipy，注意 Σ 正定与约束

## 六、扩展要点（期权、固收、行为、资产配置、风险平价/BL、执行）

- **期权与波动率**：Call/Put payoff、(S−K)⁺、(K−S)⁺；Put-Call Parity C−P=S−Ke^{-rτ}；Black-Scholes d₁,d₂、C=SΦ(d₁)−Ke^{-rτ}Φ(d₂)；Delta Φ(d₁)、Gamma φ(d₁)/(Sσ√τ)、Vega；隐含波动率、波动率曲面 — 期权与波动率基础
- **固定收益**：债券价格 P=Σ Cᵢe^{-yτᵢ}；YTM；麦考利久期、修正久期 ΔP/P≈−D_mod·Δy；凸性 C；信用利差、期限利差 — 固定收益基础
- **行为金融**：有效市场与异象；认知偏差（代表性、过度自信、锚定、调整不足、损失厌恶、处置效应）；有限套利；情绪与收益；反应不足→动量、过度反应→反转 — 行为金融要点
- **资产配置**：SAA/TAA；定期或阈值再平衡；目标波动率 θ=σ_target/σ̂_t；多资产约束 — 资产配置
- **风险平价**：风险贡献 RC_i=w_i(Σw)_i 相等；对角 Σ 时 w_i∝1/σ_i；一般情形迭代或凸优化 — 风险与组合 §5.5
- **Black-Litterman**：均衡收益 μ_equil + 观点 Pμ=q+ε → 后验 μ_BL → 均值-方差优化 — 风险与组合 §5.7
- **执行模型**：永久/暂时冲击；Almgren-Chriss 最小化期望成本+λ×方差得最优轨迹；TWAP/VWAP — 价量与微观结构 第6章
