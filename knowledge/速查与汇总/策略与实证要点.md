## 一、策略类型

### 选股 vs 择时
- **选股**：截面上区分标的优劣，收益来自截面差异（α、因子溢价）；多因子打分、回归预测、ML — 策略类型 第1章
- **择时**：时间上调整仓位/风险暴露，依赖涨跌或波动率预测；样本外常不稳定，需严格 OOS — 策略类型 第1章
- **组合**：先选股再择时（如波动率择时）；评估时区分选股贡献与择时贡献

### 多因子选股
- **打分法**：去极值 → 截面标准化或排名 → 综合得分 = Σ w_k·s_k → 十分位分组，多空或做多前 20% — 策略类型 第2章
- **回归法**：横截面 收益 ~ 因子暴露，拟合值作预测收益排序；可与组合优化结合（预期收益 + Σ 约束）
- **要点**：标准化用截至当期历史（无前视）；行业/市值中性（回归残差或优化约束）；成本与换手在回测中扣除

### 动量与反转
- **动量**：过去 J 期收益高 → 未来继续好；形成期 3/6/12 月，持有期可同或不同；做多高分做空低分 — 策略类型 第3章
- **反转**：过去短期极端 → 未来短期反向；1 周/1 月收益，做多跌多做空涨多；可与动量组合（短期反转+中长期动量）
- **注意**：动量在危机与风格切换时易回撤；参数与周期需样本外验证

### 统计套利（协整配对）
- **协整**：两非平稳序列的线性组合平稳；价差 z_t = P_A − β·P_B 向均值 μ 回归 — 策略类型 第4章
- **步骤**：选对 → Engle-Granger 检验与 β 估计 → 价差 μ、σ → 开仓 |z−μ|>kσ，回归平仓；仓位 1 单位 A、−β 单位 B
- **风险**：价差发散（止损、重验协整）；双边成本与流动性

### 简单择时
- **趋势**：均线、动量、突破 → 多空信号；仓位二元或连续；需成本与样本外验证 — 策略类型 第5章
- **波动率择时**：目标波动率 σ_target；仓位 θ = σ_target / σ̂_t（设上下限）；风险稳定而非直接预测涨跌

## 二、过拟合控制

### 含义与表现
- **过拟合**：样本内拟合过好、样本外/实盘明显变差；本质方差大、泛化差 — 过拟合控制 第1章
- **表现**：IS 夏普高、OOS 夏普低；参数敏感；逻辑薄弱；多重检验未校正
- **成因**：数据 snooping、模型过复杂、样本不足、市场时变

### 样本与时间划分
- **严格 OOS**：从未参与模型/参数/因子/规则选择；仅用于**最终一次性**评估 — 过拟合控制 第2章
- **训练 / 验证 / 测试**：训练估参数、验证选超参与模型、测试=OOS 只算一次；时间 训练 < 验证 < 测试
- **滚动**：固定长度 T，适应时变；**扩展**：样本递增，估计更稳

### 时间序列交叉验证
- **禁止打乱**：按时间划分，训练早于验证 — 过拟合控制 第3章
- **TimeSeriesSplit**：多折、训练单调扩大或滑动，验证在训练之后；用于选超参/模型
- **滚动预测**：一步前向预测序列，用预测误差的均值与标准差评估 OOS

### 正则化与因子/参数控制
- **Ridge**：β̂ = (XᵀX+λI)⁻¹XᵀY；系数缩小、稳定共线性 — 过拟合控制 第4章
- **Lasso**：‖β‖₁ 惩罚，稀疏、变量选择；λ 用验证或 CV 选
- **因子数量**：先验、样本外筛选、降维（PCA）、正则、多重检验校正 — 过拟合控制 第5章
- **参数与规则**：少而稳、在验证/CV 上搜索、敏感性分析；奥卡姆剃刀、可解释性

### 稳健性检查清单（简）
- [ ] 严格 OOS 且只评估一次；多段 OOS 或多市场
- [ ] 时间序列 CV 选超参/模型，未打乱
- [ ] 正则化或容量控制；因子与参数数量受控；成本与约束在回测中
- [ ] 参数敏感性可接受；逻辑可解释 — 过拟合控制 第6章

## 三、实盘与系统

### 信号
- **研究 vs 生产**：生产用**当期可得数据**、与回测**完全一致**的规则算信号 — 实盘与系统 第1章
- **数据一致**：复权、停牌、发布时间与回测一致；日频明确数据截止、算信号、下单时点
- **缺失/异常**：默认规则（沿用上期或剔除）；逻辑分支与回测一致

### 仓位管理
- **目标权重** w* vs **当前权重** w；调仓 Δw = w*−w；受最小单位、流动性、成本约束 — 实盘与系统 第2章
- **再平衡**：定期（日/周/月）或阈值再平衡；涨跌停不追买/不追卖；大单拆单或算法单、流动性过滤

### 风控
- **止损**：组合止损（回撤/净值阈值）、单标的止损；回测中**同样应用**否则不可比 — 实盘与系统 第3章
- **敞口**：净敞口（市场中性≈0）、行业/因子/单标的上限；实时监控与触发
- **VaR/风险预算**：组合 VaR 不超预算；流动性储备与冲击控制

### 回测与实盘差异
- **成本**：实盘常高于回测；回测用保守假设，实盘记录并修正 — 实盘与系统 第4章
- **数据与时间**：实盘缺失/延迟/错误；可执行时点与回测对齐
- **市场**：流动性、涨跌停、停牌、极端行情；回测逻辑与实盘一致
- **纪律**：系统化执行、少人工干预；实盘与回测对比、归因；新数据评估仍遵守严格 OOS

## 四、论文与研报

### 为何读
- 理解因子**来源、假设与证据**；学习**表述与逻辑**；避免重复与侵权 — 论文与研报 第1章

### 如何读
- **结构**：摘要→引言→数据与方法→实证→稳健性与讨论→结论 — 论文与研报 第2章
- **问题**：因子定义与构造、回测设定与偏差、统计与多重检验、可复现性
- **笔记**：一页纸（问题、方法、发现、局限、可借鉴）；公式与定义；疑问与批判

### 经典方向（速查）
- **因子与定价**：Fama-French 三/五因子、Jegadeesh-Titman 动量、Carhart 四因子、低波/质量 — 论文与研报 第3章
- **统计套利**：Engle-Granger 协整、配对交易
- **回测与过拟合**：White Reality Check、Bailey 等
- **国内与实务**：A 股实证、券商/私募研报；注意版权与数据

### 研报批判
- 数据、回测、结论、利益相关；重点看方法与数据，结论需验证 — 论文与研报 第4章

## 五、与前面阶段的衔接

- **Phase 1**：回归/时间序列 → 横截面回归、协整配对、ARMA 预测
- **Phase 2**：多因子/回测/风险 → 选股信号、回测规范、夏普/回撤/VaR、组合优化
- **Phase 3**：策略类型 → 完整 pipeline；过拟合控制 → 严格 OOS、CV、正则；实盘与系统 → 信号一致、仓位与风控、差异应对；论文与研报 → 文献笔记库与引用
